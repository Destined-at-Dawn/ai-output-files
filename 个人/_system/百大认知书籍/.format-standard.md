# 百大认知书籍 · 统一格式标准 v1.0

> 基于 029《沧浪之水》和 037《结构洞与好主意》的黄金模板
> 所有 62 本书必须统一至此格式

---

## 强制结构（8 个必需章节）

### §一、著作科学定位（必需）
- **长度要求：** 至少 1 段（5-10 行）
- **内容要求：** 说明该书在学科史上的位置、推翻的旧范式、建立的新框架
- **对标：** 037 的 §一

```markdown
## 一、著作科学定位

在[学科]的发展史上，[书名]是一部具有"[xxx]式革命"意义的著作。在此之前，[旧范式]。
[作者]彻底推翻了[旧假设]。他通过[实证方法]，首次用[数学模型/实验证据]证明：[核心发现]。
该著作将[现象]从[玄学/心理学]，降维还原为[可计算/可操作的物理过程]。
```

### §二、核心概念（必需，3-6个概念）
- **每个概念必须包含：**
  1. **原句引用**（原文引用，标注出处）
  2. **深层解读**（机制层面拆解，不只是复述）
  3. **个人应用**（对小黎的具体意义）
- **对标：** 029 的 §一

```markdown
## 二、核心概念

### 概念1：[名称]
**原句：** "[原文引用]"
**解读：** [机制层面的深层拆解，3-8行]
**个人应用：** [1-2句对小黎的具体意义]

### 概念2：[名称]
...
```

### §三、核心金句（必需，8-12条）
- **格式：** 双语引用（如原书为英文）
- **要求：** 每条有穿透力、可独立引用
- **对标：** 037 的 §三

```markdown
## 三、核心金句

1. **"[原文]"** —— [中文翻译/解读]
```

### §四、对现在的我（必需，深度！）
- **必须包含两个子部分：**
  - **当前诊断：** 直呼"小黎"，个性化描述当前困境（至少 5 行）
  - **SOP 行动方案：** 3-5 个命名 SOP，每个包含 痛点→机制→行动
- **对标：** 029 §三 + 037 §四

```markdown
## 四、对现在的我

### 当前诊断
小黎，你当前的状态——[具体描述]——在你身上有一个[极其隐蔽但极度危险的认知盲区]：
**[核心诊断，加粗，一句击中]**
[展开说明，至少5行]

### [SOP名称 1]
**痛点：** [具体痛点描述]
**机制：** [底层原理/学科依据]
**行动：**
- [具体行动 1]
- [具体行动 2]
- [具体行动 3]

### [SOP名称 2]
...
```

### §五、对未来的我（必需，深度！！！）
- **必须包含：道、法、术、器、成长四维表格、终极破局**
- **每个子部分至少 4 行**，禁止只写 2-3 句
- **对标：** 029 §四（最佳范例！）

```markdown
## 五、对未来的我

### 道（底层认知）
[4-8行的深度哲学阐述，含公式/引用/社会规律]
[不是"长期价值：XXX"这种空话，而是像029那样有穿透力的洞察]

### 法（原则策略）
[3-5条可操作的原则，每条有解释]
[像029那样含公式——P=f(C,S,N)——把模糊原则变成可计算的框架]

### 术（方法技巧）
| 领域 | 策略 | 具体执行 |
|------|------|---------|
| [领域1] | [策略] | [具体怎么做] |
| [领域2] | [策略] | [具体怎么做] |
（至少4行，用表格！）

### 器（工具资源）
- **[工具名]**：[用途]，[具体如何用]
- （至少4条，每条有具体工具名称和使用场景）

### 成长四维评估

| 维度 | 短期影响（1-2年） | 长期影响（5-10年） |
|------|-------------------|-------------------|
| 钱 | [具体，不是空话] | [具体，不是空话] |
| 影响力 | [具体] | [具体] |
| 关系 | [具体] | [具体] |
| 技能 | [具体] | [具体] |

### 终极破局
[2-5行的远见结语，像029那样有启发性]
[把本书的洞察投射到小黎的未来——OPC/AI时代/阶层跃迁]
```

### §六、速查索引（必需，表格格式！）
- **必须是表格，8行**
- **禁止**用标题格式
- **对标：** 029 §五 + 037 §六

```markdown
## 六、速查索引

| 场景 | 对应概念 | 立即行动 |
|------|---------|---------|
| [场景1] | [概念名] | [一句话行动] |
| [场景2] | [概念名] | [一句话行动] |
（至少8行）
```

### §七、与百大认知其他书籍的交叉耦合（必需）
- **已在之前的 agent 中添加，保留即可**
- 两个子表格：对现在的我（3-5条）+ 对未来的我（3-5条）

## 禁止模式（❌ 这些就是"水"的特征）

| ❌ 禁止 | ✅ 正确做法 |
|---------|-----------|
| 对未来的我每条只写2-3句 | 每条至少4行，有深度 |
| 道："长期价值：在XX领域保持竞争力" | 道：像029那样写——"世界不是按'谁有理'来运转的，而是按'谁有权'来运转的" |
| 法："1.建立XX框架 2.掌握XX方法" | 法：像029那样含公式 P=f(C,S,N)，每个原则有解释 |
| 术：4个概念名词拼凑 | 术：表格！领域×策略×具体执行 |
| 器：三个场景名词（"学习/竞赛/团队"） | 器：具体工具名+具体用途 |
| 成长四维："短期：提升效率 → 长期：竞争力" | 成长四维：具体数字/场景/路径 |
| 速查索引用标题格式 | 速查索引用表格 |
| 缺少终极破局 | 必须有！ |
