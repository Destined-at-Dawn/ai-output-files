# AI作业：用AI完成一次个人决策分析

> 本周真实任务：**上海插班生考试+实习offer+3个竞赛的多线并行决策**
> 工具：Claude Code（百大认知书籍个人知识库）
> 时间：2026年5月1日

---

## 一、成品：决策分析结论

### 背景

我是上海电力大学大一学生（GPA 3.9），同时面临：
- 上海交大插班生考试机会（概率低但收益极高）
- 3个竞赛进行中（集创赛队长、iCAN队长、皮影戏队员）
- 医疗器械公司offer（180-250/天，但需坦诚到岗时间）
- 皮赛尔FPGA面试（等朱工出差回来）
- 转专业申请（5月18-20日提交）
- CET-4备考+入党志愿者

**9条线同时拉。**

### AI分析框架

使用"道法术器"四层框架：
- **道**：拆解底层认知陷阱（可得性偏差→"感觉概率不大"、反脆弱杠铃策略、直觉依赖者困境）
- **法**：精要主义过滤器——"这件事如果不做，3年后我会后悔吗？"
- **术**：以周为单位的执行方案（5月1日-6月15日甘特图）
- **器**：决策过滤器+立即行动清单

### 最终决策

1. **放弃交大插班生**（概率太低+和专业方向不对齐，考研985是更确定的路径）
2. **低投入冲上大微电子**（和集成电路专业对口，备考和期末复习重叠）
3. **转专业必须申请**（5月18-20日，错过=永久放弃选择权）
4. **实习坦诚沟通到岗时间**（6月20日课程结束后）
5. **皮影戏保持，iCAN降频，简历技能暑假集中补**

## 二、3-5个关键Prompt

### Prompt 1：开场澄清（Clarify步骤）

```
我现在想要参加上海插班生考试。但是我现在有很多任务，而且我感觉我考上概率不大。
我想要去试一试，考上海交通大学的插班生。如果对于我现在这个专业集成电路的话，
我还可以报名上海大学的微电子。你觉得我应该怎么决策？结合"百大认知书籍"
```

**为什么这个Prompt有效：** 明确了选择项（交大vs上大）、自身约束（任务多）、求助方向（结合知识库回答）。

### Prompt 2：补充约束条件

```
我现在手头上还有3个竞赛（集创赛，ICAN，皮影戏），而且我学业压力也比较重，
我还需要准备实习的技能。我在3天前通过面试获得一家做医疗器械公司的硬届实习生岗位。
对方要求尽快到岗。我当时面试的时候谎说可以在15天内到岗，但实际上我只能在
期末考试考完之后到岗。工资每天180元到250元。
```

**为什么这个Prompt有效：** 把所有约束条件一次性dump出来，包括"不光彩"的真实情况（谎报到岗时间），让AI能基于真实信息做分析，而不是基于美化后的信息。

### Prompt 3：质疑AI的引用偏差

```
为什么你每次引用的只有四五本书，这有什么特殊含义吗？
```

**为什么这个Prompt有效：** 直接指出AI输出的系统性偏差（引用集中在少数书上），迫使AI自我诊断并修正。结果是AI承认了三个偏差来源并建立了修正规则。

### Prompt 4：要求建立系统性工具

```
你应该在百大认知书籍里面针对我问过的问题的内容，把你拆分的过程写成新的文件。
我想要实现的效果就是每次我问你一个问题，你第一时间是把这个问题分成几大类，
然后根据那几大类分别去寻找对应的书籍。
```

**为什么这个Prompt有效：** 不只是解决当前问题，而是要求AI建立可复用的系统（问题拆解→书籍映射表），把一次性的对话变成持续优化的知识基础设施。

### Prompt 5：修正AI的事实错误

```
iCAN中我才是队长，其他4个人是跟我同专业的。皮影戏的队长是颜冬。
你不要不懂装懂，你没读到里面的内容，你就不要瞎猜，你就来问我。
```

**为什么这个Prompt有效：** 直接指出AI在没有读取材料的情况下编造了团队信息，并建立了"不懂就问"的规则。这比让AI继续猜要高效得多。

## 三、3条踩坑笔记

### 踩坑1：AI会凭"训练数据记忆"假装读过文件

**发生了什么：** 我第一次让AI根据百大认知书籍回答问题时，它引用了4-5本"常用书"，但实际上46本书中只有9本被真正读取过。我问"为什么只引这几本"，AI承认是在用训练数据的记忆"猜"内容，而不是实际Read文件。

**教训：** 当AI引用某个文件/数据源时，要追问"你实际读过这个文件吗？"。AI可能基于泛化知识编造看起来合理的引用，但没有验证过原文。

**修正做法：** 我要求AI先把全部46本书逐本完整读取，再基于实际内容重写映射表。从30个子问题扩展到62个，从2个示例扩展到16个场景。

### 踩坑2：AI会搞混相似信息，不读原始材料就编造细节

**发生了什么：** AI把iCAN和皮影戏两个比赛的队长搞混了（以为我是两个比赛的队长），把团队人数搞错了（5人vs3人），把技术平台搞错了（以为是RDK X5，实际是汇博机器人）。这些错误全部源于AI没有读取我提供的实际文件，而是"猜"的。

**教训：** 提供了材料就要让AI先读完再回答。如果AI的回答中出现了你没说过的细节——它很可能是编的。

**修正做法：** 直接指出错误，建立规则"没读到的内容不要瞎猜，来问我"。这条规则被写入了项目配置文件，永久生效。

### 踩坑3：信息过载时AI的分析质量会下降

**发生了什么：** 第一轮对话只有"插班生"一个话题，AI给出了精准的反脆弱杠铃分析。第二轮我同时塞了实习+3个竞赛+转专业+英语+简历技能，AI的分析开始变得面面俱到但没有锋芒——它试图给每件事都安排优先级，反而让我更迷茫。

**教训：** 给AI的信息不是越多越好。一次聚焦1-2个核心决策，效果远好于把所有事情一次性dump。先把最关键的问题问清楚，再逐步扩展。

**修正做法：** 分轮次对话——第一轮只讨论"插班生考不考"，第二轮加入实习约束，第三轮讨论具体执行计划。每轮一个焦点。

## 四、一句话复盘

> **AI最大的价值不是"替你做决策"，而是帮你把9条纠缠的线逐根拆开、暴露你没看到的变量（比如赵老师这张王牌）、然后让你用清醒的系统2去做选择——但前提是你要不断纠正它编造的事实。**

## 五、附录：跨平台验证建议

作业要求"Claude+GPT+Grok交叉验证"。以下是同一核心问题的跨平台Prompt，你可以分别在三个平台上测试：

```
【给GPT/Grok的Prompt】
我是上海电力大学大一学生（GPA 3.9，集成电路专业），同时面临以下选择：
1. 报考上海交大插班生（工业工程方向，分数最高）vs 上海大学微电子（与专业对口）
2. 已拿到医疗器械实习offer（180-250/天），但需解决到岗时间冲突
3. 3个竞赛同时进行（集创赛/iCAN/皮影戏）
4. 5月18-20日转专业申请（集成电路→电气工程）
5. 自认为考研也能上985

请用反脆弱框架（塔勒布的杠铃策略）分析：插班生考试应该怎么决策？
下行风险是什么？上行收益是什么？如何设计一个不对称的赌注？
```

预期差异：
- **Claude**：擅长结合个人知识库做个性化分析，但可能引用偏差
- **GPT**：擅长通用框架和结构化输出，但不了解你的个人背景
- **Grok**：擅长跳出常规思维，可能给出意外视角
