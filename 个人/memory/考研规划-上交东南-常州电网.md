> 终极目标：国网常州市局（买得起房+工资高+福利好+稳定）
> 数据来源：上海交通大学研招网、东南大学研招网、国家电网招聘平台、知乎考研社区

## 一、战略定位：为什么考研是必选项

### 1.1 国网常州局学历门槛

| 学历 | 进常州局难度 | 岗位层级 | 薪资预期 |
|------|-------------|---------|---------|
| 本科（上电） | ★★★★☆ 较难 | 基层运维 | 12-15万/年 |
| 硕士（211/双一流） | ★★☆☆☆ 可控 | 技术/管理岗 | 18-25万/年 |
| 硕士（985/电气强校） | ★☆☆☆☆ 把握大 | 技术骨干/管培 | 20-30万/年 |

**结论：硕士学历是进常州局的最优路径。** 国网校招对985/211电气硕士有明显倾斜，尤其东南大学作为江苏省内唯一电气985，在江苏电网系统内有极强校友网络。

### 1.2 两校对比一览

| 维度 | 上海交通大学 | 东南大学 |
|------|------------|---------|
| 学科评估 | B+（全国8-12名） | A-（第五轮晋升） |
| 电气方向强项 | 电力系统、新能源、智能电网 | 电力电子、电机、电力系统 |
| 对常州局优势 | 985光环+上海区位 | **江苏省内985，江苏电网校友网络最强** |
| 考研难度 | ★★★★★（极高） | ★★★★☆（很高） |
| 专业课 | 822基本电路理论（自命题） | 电气科学基础（2026起新科目） |
| 2025/2026复试线 | 学硕300/专硕300 | 学硕365/专硕375 |
| 统考招生人数 | 约66人 | 约70人 |
| 报录比（专硕） | 约5-8:1 | 约4.7:1（2023数据） |
| 录取均分 | 约380分 | 约380-390分 |
| 建议目标分 | ≥370分 | ≥380分 |

### 1.3 择校建议（结论先行）

**主攻：东南大学（专硕085801）**
- 理由：江苏电网系统内校友网络最强、电气学科评估A-高于上交B+、专硕招生人数多（52人）、地理位置（南京）离常州近便于实习/求职
- 「东南电气硕士进江苏电网 = 近水楼台先得月」

**冲刺：上海交通大学（专硕085801）**
- 理由：985招牌全国通用、上交复试线近年持续下降（315→310→300）、若能上岸则进国网任何局都有优势
- 风险：专业课822为自命题，难度高；上海读研成本高

**保底策略：** 两校考试科目高度重合（政治+英语一+数学一+电路），可同时备考，10月报名时根据复习状态决定。

## 三、四年时间线（大一→考研）

### Phase 1：大一下（2026.5 - 2026.8）—— 打地基

> 核心任务：**转专业 + 英语突破 + 数学基础**

| 优先级 | 任务 | 时间 | 验收标准 |
|--------|------|------|---------|
| P0 | 5月18-20日提交转专业申请 | 本周 | 成功转入电气工程 |
| P0 | 高数下册期末冲刺 | 5-6月 | GPA≥3.8，保持前5% |
| P1 | CET-4备考 | 6月考 | 通过（≥425分） |
| P1 | 暑假预习电路原理 | 7-8月 | 读完《电路》邱关源前8章 |
| P2 | ICAN小车竞赛推进 | 全程 | 9月前完成硬件调试 |
| P2 | 皮影戏竞赛初稿 | 6月15日前 | 提交初稿 |

**英语备考（每日30分钟铁律）：**
- 周一~五：逐句听写VOA慢速/四级真题听力
- 周六：Anki间隔复习本周单词
- 周日：做一套四级真题阅读

### Phase 2：大二（2026.9 - 2027.7）—— 积累期

> 核心任务：**课程GPA + 实验室经历 + 竞赛**

| 优先级 | 任务 | 时间 | 验收标准 |
|--------|------|------|---------|
| P0 | 电气核心课冲高分 | 大二全年 | 电路/电机/电力电子≥90分 |
| P0 | GPA维持前5% | 每学期 | 累计GPA≥3.8 |
| P1 | 进赵老师推荐的实验室 | 大二上 | 获得项目经历 |
| P1 | 电子设计大赛 | 大二下/暑假 | 获省级以上奖项 |
| P1 | CET-6备考 | 大二上考 | 通过（≥425分） |
| P2 | 积累科研/项目经历 | 全程 | 发1篇论文或参与1个项目 |
| P2 | 大创红色赛道（高教授） | 大二 | 完成立项或结项 |

**这个阶段为考研铺垫的隐性收益：**
- 电路/电机/电力电子高分 = 专业课基础
- 实验室经历 = 复试加分项
- 竞赛获奖 = 复试加分项 + 综合素质评价

### Phase 3：大三上（2027.9 - 2028.1）—— 考研启动

| 12月 | 数学基础完成 | 一轮结束，开始做660题 |

### Phase 4：大三下（2028.2 - 2028.8）—— 强化期
