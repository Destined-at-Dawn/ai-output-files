## 记忆架构的认知科学原理（百大认知书籍）

> 个人区的记忆系统是四个工作区中最复杂的——因为个人成长本身就是最复杂的认知过程。

| 认知机制 | 来源 | 在个人区记忆系统中的应用 |
|---------|------|------------------------|
| **自传体记忆** | 022-认知神经科学 §概念1 | 每日记忆文件=自传体记忆的外部化——遵循自我定义事件结构 |
| **情境依赖记忆** | 009-认知天性 §概念1 | 日期标签和锚点=情境依赖检索——回忆相似情境即定位记忆 |
| **图式理论** | 025-认知负荷理论 §概念4 | PK速查表=错误图式的索引——10个PK=10个认知图式标签 |
| **认知跃迁** | 023-认知觉醒 §概念2 | 按日期组织反映认知跃迁时间线——每个日期=一次认知版本升级 |
| **间隔效应** | 009-认知天性 §概念3 | 新对话开局先读此文件=间隔效应的逆向应用——主动重建上下文 |
| **组块化** | 022-认知神经科学 §概念1 | 104行记忆被组织为日期块->事件块->链接=三级跳转 |
