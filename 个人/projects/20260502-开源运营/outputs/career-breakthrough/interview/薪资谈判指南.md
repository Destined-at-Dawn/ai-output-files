# 薪资谈判指南

> 实习生也需要谈钱。不是贪婪，是对自己的劳动负责。

---

## 什么时候该谈，什么时候不该谈

| 场景 | 要不要谈 |
|------|---------|
| 结构化校招（大厂统一薪资） | 不用谈，没有谈判空间 |
| 小公司/创业公司实习 | 可以谈，尤其是日薪/月薪 |
| 拿到offer后 | 最佳谈判时机 |
| 初面时 | 不要主动谈钱 |
| HR主动问你期望 | 给区间，不给确切数字 |

## 实习薪资调研

### 怎么查市场价

| 渠道 | 说明 |
|------|------|
| 看准网 | 公司薪资评价，有实习数据 |
| 脉脉 | 匿名薪资分享 |
| offershow.com | 互联网公司offer数据 |
| Boss直聘 | 直接看同类岗位的薪资范围 |
| 学长/校友 | 最真实的一手信息 |

### 参考范围（上海，技术实习）

| 方向 | 日薪范围 |
|------|---------|
| 软件开发 | 150-300元/天 |
| FPGA/嵌入式 | 100-250元/天 |
| 硬件测试 | 100-200元/天 |
| 数据/AI | 200-400元/天 |

> 注意：不同城市差异很大。北京/上海/深圳最高，杭州/广州次之，其他城市较低。

## 谈判话术

### 当HR问"你期望薪资多少"

**初面（不要给确切数字）：**
> "我更看重学习机会和技术成长，薪资在行业合理范围就行。方便了解一下贵公司的实习薪资结构吗？"

**拿到offer后（可以谈）：**
> "感谢offer！薪资方面，我了解到同类岗位的市场价在XX-XX之间，我期望能在XX左右。不知道有没有调整空间？"

### 谈判原则

1. **先让对方出价** — 不要先报底价
2. **给区间不给数字** — "150-200"比"170"有弹性
3. **用市场数据支撑** — "我了解到……"比"我想要……"更有说服力
4. **不要只看日薪** — 考虑整体package

## 日薪之外，还能谈什么

| 维度 | 说明 |
|------|------|
| 住房补贴 | 部分公司提供租房补贴或宿舍 |
| 餐饮 | 包餐/餐补 |
| 交通 | 交通补贴 |
| 转正机会 | 实习转正的条件和概率 |
| 导师制度 | 是否有正式的mentor |
| 学习时间 | 是否允许半天实习/弹性时间 |
| 推荐信 | 实习结束能否拿到推荐信 |

**有时候一个好mentor比多50块日薪更有价值。**

## 中国职场的谈薪文化

和西方不同，中国职场谈薪更含蓄。几个注意点：

- **不要在第一次沟通就谈钱** — 先展示价值，再谈价格
- **语气要诚恳** — 不是对抗，是协商
- **给台阶** — "如果目前不方便调整，我也理解"
- **国企/事业单位基本没谈判空间** — 体制内薪资结构固定

---

## 常见错误

- ❌ 没调研就报价 → 要太高被拒/要太低亏了
- ❌ 初面就谈钱 → HR觉得你只关心钱
- ❌ 只看日薪 → 忽略了学习机会、转正、导师等隐性价值
- ❌ 当场做决定 → "我回去考虑一下，明天给您回复"是完全正常的
