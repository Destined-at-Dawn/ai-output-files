# 贡献指南

感谢你愿意为这个项目做出贡献！

## 你可以做什么

### 📄 提交简历模板
- 新的行业方向（金融/医疗/教育/设计……）
- 新的语言版本（日语/韩语/法语……）
- 新的风格（创意型/学术型/极简型……）
- 放在 `resume/templates/` 对应目录下

### 📝 分享成功案例
- 把你的求职经历匿名化后提交到 `community/success-stories/`
- 参考 `community/success-stories/template.md` 的格式
- **必须脱敏**：去掉真实姓名、电话、邮箱、公司全称

### 🔧 改进方法论
- 补充你的实战经验
- 修正不准确的建议
- 添加新的行业/岗位视角

### 🐛 修复问题
- 文档错别字
- Markdown格式问题
- 链接失效

## 提交流程

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/你的改动`
3. 提交更改
4. 发起 Pull Request

## 注意事项

- **不要提交个人信息**（真实姓名、电话、身份证号等）
- **不要提交未脱敏的公司信息**
- 中文内容用中文写，英文内容用英文写，不要机翻
- 保持和其他文档一致的风格（直接、实用、无废话）
