# 问题拆解 → 书籍映射表（Question Decomposition → Book Mapping）

> **用途：** 每次用户提问，先拆解问题为N个主题维度，再查本表定位对应书籍。
> **设计原理：** 任何复杂问题都不是单维度的。本表将46本书按10大主题维度 + 子问题类型归类，支持快速交叉定位。
> **使用流程：** 拆解问题 → 识别涉及的主题维度 → 查本表锁定书籍编号 → Read原书提取原句 → 回答。
>
> **版本：** v2.2 | **创建日期：** 2026-05-01 | **升级内容：** 核心路由层已扩展至62本全量覆盖，新增047—062跨维度矩阵，同名书区分规则已完成总控层修正
> **当前覆盖范围：** 62本核心路由书（全量）

## 零、快速使用指南

### AI内部操作流程（每次回答前执行）

```
1. 识别用户问题类型 → 看§一的"问题类型快速匹配"
2. 拆解为N个子问题 → 每个子问题对应§二的1个主题维度
3. 查本表§三~§十二 → 每个维度下找到对应的书籍编号
4. 引用追踪去重 → 检查CITATION_TRACKING.md，优先选冷门书（❄️未激活）
5. Read原书 → 提取精确原句（禁止凭记忆复述）
6. 回答 → 标注Source: 百大认知书籍/0XX-书名.md#行号
```
