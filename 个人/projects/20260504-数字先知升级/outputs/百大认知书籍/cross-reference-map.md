# 交叉引用总览（Cross Reference Map）

## 作用

这个文件不是替代图谱，而是给总控层一个可读、可维护、可手工修正的交叉引用骨架。

它主要解决三个问题：

1. 图谱适合看结构，不适合精确维护。
2. 总索引不应该继续承担全部关系细节。
3. 后续做 Graph、Dataview、CSS 优化时，需要一个明确的人工真值层。
