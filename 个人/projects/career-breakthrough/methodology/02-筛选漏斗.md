# 02 | 筛选漏斗：用数据思维过滤机会

> 海投是采集数据，筛选才是做决策。

---

## 为什么需要筛选

海投20家，可能有10家回复你。你不可能面10家——时间和精力都不允许。

而且，不是每个回复都值得跟进。有些公司挂着"实习"的名头，实际上是找免费劳动力；有些公司看着光鲜，进去之后你什么都学不到。

**筛选的目的：把有限的时间投入到最值得的机会上。**

---

## 三层过滤漏斗

### 第一层：基本面过滤（5分钟判断）

这一层筛掉明显不靠谱的：

| 红旗 🚩 | 说明 |
|---------|------|
| 不给工资或给极低工资 | "无薪实习"= 免费劳动力，除非是顶级平台的学习机会 |
| 要求交培训费 | 100%是骗局，没有任何例外 |
| JD写得极其模糊 | "招实习生，要求积极向上" — 连技术栈都不写，说明他们自己都不知道要什么 |
| 面试过程极不专业 | 迟到30分钟不道歉、面试全程看手机、问私人问题 |
| 公司信息查不到 | 企查查/天眼查搜不到，或者注册资金极低、刚注册 |
| 催你当天决定 | 正规公司不会逼你当场签，这是红旗中的红旗 |

**操作：** 打开企查查，输入公司名，看注册时间、注册资本、经营范围、法律诉讼。3分钟就够。

### 第二层：成长性过滤（15分钟判断）

这一层筛掉"能去但不值得去"的：

**关键问题：这份实习能让我学到东西吗？**

| 加分项 ✅ | 减分项 ❌ |
|----------|----------|
| 有明确的技术导师 | 没人带，自己摸索 |
| 能接触真实项目 | 只让你做测试/整理文档 |
| 技术栈和你方向匹配 | 让你做完全不相关的杂活 |
| 公司在上升期 | 公司在裁员/业务萎缩 |
| 团队小但精 | 团队大但你是透明人 |
| 能看到转正机会 | 明确说"没有转正可能" |

**我的实际判断：**

拿我收到的offer举例——A公司（医疗器械，50-100人）：
- ✅ 小团队，能接触完整项目
- ✅ 硬件方向，和我的FPGA技能匹配
- ✅ 日薪180-250，在合理范围
- ⚠️ 通勤2.5小时，需要租房（额外成本）

B公司（FPGA应用工程师）：
- ✅ FPGA方向，完美对口
- ✅ 可转正
- ✅ 有工程师直接带
- ⚠️ 日薪100偏低

### 第三层：综合决策（和自己对话）

通过前两层过滤后，剩下的都是"值得去"的选项。这时候需要做最终决策：

**问自己三个问题：**

1. **这份实习3个月后的简历价值？** — 写在简历上，能让下一份实习/工作更有竞争力吗？
2. **这个团队能给我什么？** — 技术成长？行业认知？人脉？还是只是一份工资？
3. **机会成本？** — 去了这里，我放弃了什么？（暑期课程、竞赛准备、其他更好的机会）

---

## 快速调研公司的方法（5分钟版）

| 步骤 | 操作 | 工具 |
|------|------|------|
| 1. 查公司背景 | 注册时间、注册资本、经营范围 | 企查查/天眼查 |
| 2. 看员工评价 | 工作环境、管理风格、实习体验 | 看准网/脉脉 |
| 3. 搜新闻 | 最近有没有融资/裁员/产品发布 | 百度/搜狗 |
| 4. 看团队 | 技术负责人背景、团队规模 | LinkedIn/公司官网 |
| 5. 问真人 | 有没有认识的人在里面工作 | 微信朋友圈/校友群 |

---

## 什么情况下应该拒绝

即使公司给了offer，以下情况也应该认真考虑拒绝：

- **技术方向完全不对口** — 你做FPGA的，去做销售助理，简历价值为零
- **学不到任何东西** — 纯打杂，没有人带，没有真实项目
- **严重影响学业** — 大一/大二，GPA掉了得不偿失
- **身体/心理健康代价** — 通勤3小时+、PUA式管理、加班文化严重
- **有更好的选择** — 手里有2个offer，选成长性更高的那个

## 决策矩阵模板

当你有几个选项难以抉择时，用这个矩阵打分：

| 维度（权重） | 选项A | 选项B | 选项C |
|-------------|-------|-------|-------|
| 技术匹配度（30%） | ?/10 | ?/10 | ?/10 |
| 学习成长性（25%） | ?/10 | ?/10 | ?/10 |
| 简历价值（20%） | ?/10 | ?/10 | ?/10 |
| 薪资待遇（10%） | ?/10 | ?/10 | ?/10 |
| 通勤/生活成本（10%） | ?/10 | ?/10 | ?/10 |
| 转正可能性（5%） | ?/10 | ?/10 | ?/10 |
| **加权总分** | | | |

> 注意：权重可以根据你的实际情况调整。如果你急需钱，薪资权重可以提高。如果你更看重学习，成长性权重可以提高。
