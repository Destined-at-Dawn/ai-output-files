# 企业追踪模板

> 用表格管理你的投递状态。不追踪 = 混乱。

---

## 使用说明

1. 每投一家公司，加一行
2. 状态变化时立刻更新
3. 每周回顾一次，清理无回复的（超过2周没回复基本凉了）

### 状态说明

| 状态 | 含义 | 下一步行动 |
|------|------|-----------|
| 📝 已投递 | 简历已发送 | 等3-5个工作日，没回复就跟进 |
| 📞 沟通中 | HR已回复/加了微信 | 确认面试时间 |
| 📋 一面 | 第一轮面试 | 面试后做复盘 |
| 📋 二面 | 第二轮面试 | 面试后做复盘 |
| ✅ 已offer | 收到offer | 评估+决定 |
| ❌ 已拒绝 | 被拒/主动放弃 | 记录原因，提炼经验 |
| ⏸️ 无回复 | 超过2周没回复 | 标记为无回复，不再跟进 |
| ⏳ 等待中 | 对方说稍后联系 | 设提醒，1周后跟进 |

## 跟进提醒

- 投递后 **3天** 无回复 → 发一条跟进消息
- 面试后 **24小时内** → 发感谢消息
- HR说"稍后联系"后 **1周** → 礼貌跟进
- 收到offer后 **24小时内** → 不要当天拒/接，给自己思考时间
