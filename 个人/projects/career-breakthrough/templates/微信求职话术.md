# 微信求职话术模板

> 微信是中国职场最重要的沟通工具。怎么发消息直接影响别人对你的第一印象。

---

## 加HR/工程师微信后的第一条消息

### ✅ 好的示范

```
[称呼]好，我是[学校][专业]的[姓名]，在[渠道/场合]了解到贵公司在招[岗位]。
我主要做[方向]，有[核心经历/项目]。非常感兴趣，想了解[具体问题]。
```

### ❌ 差的示范

```
你好
在吗
请问你们还招人吗
```

**差距：** 好的消息在一条之内提供了你的身份、背景、目的，对方可以快速判断你是否值得回复。

---

## 面试后跟进消息

### ✅ 面试后24小时内

```
[称呼]好，感谢今天的面试机会。和您聊了[具体技术话题]之后收获很大，
我对[某个点]特别感兴趣，回去后会深入研究。
期待后续沟通，祝工作顺利！
```

---

## 等待回复时的跟进

### ✅ 发了一条消息3天没回复

```
[称呼]好，上次发的消息不知道您是否看到了？
如果现在不方便回复也没关系，等您有空时再聊。
```

### ❌ 连发3条一样的消息

不要这样做。发一条，等回复。没回复就等几天再跟一次。两次都没回复就放弃。

---

## 微信沟通铁律

| 规则 | 说明 |
|------|------|
| 不发语音 | 对方可能在开会/不方便听 |
| 一条消息说清楚 | 不要分5条发同一个意思 |
| 对方回了"好的""嗯"= 已收到 | 不需要再确认 |
| 改备注名 | 加完立刻改："X工-公司名+方向" |
| 不要深夜发消息 | 急事打电话，不急的事工作时间发 |
| 带实质性内容 | 代码链接/项目文档 > 重复自我介绍 |
