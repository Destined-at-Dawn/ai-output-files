# 中文求职信模板

> 大多数中小企业实习不需要求职信。但如果需要，这里有几个模板。

---

## 模板1：校园实习申请

```
[HR姓名] 您好：

我是[学校名][专业名][年级]的[姓名]，在[渠道]看到贵公司[岗位名]的招聘信息，
对这个岗位非常感兴趣，希望能获得面试机会。

我的核心优势：
- [技术优势1：和JD最匹配的技能/项目]
- [技术优势2：第二个匹配点]
- [软技能/学习能力：一个具体事例]

我对[公司名/产品/方向]的关注已经有一段时间了，[一句话说明你对公司的了解]。
我相信我的[技能/经历]能够为团队带来价值。

简历已附上，期待您的回复。

[姓名]
[电话] | [邮箱]
```

## 模板2：自荐（没有公开招聘信息）

```
[负责人姓名] 您好：

冒昧打扰。我是[学校名][专业名]的[姓名]，关注贵公司[产品/技术方向]已久。

我目前在[方向]上有一定的项目经验：
- [项目1简述 + 核心成果]
- [项目2简述 + 核心成果]

不知道贵公司是否有实习机会？如果有，我非常希望能加入团队学习。
即使暂时没有岗位，也希望能和您交流[技术方向]的见解。

简历已附上，感谢您的时间。

[姓名]
[电话] | [邮箱]
```

---

## 写求职信的注意事项

- **简短** — 200字以内，HR没时间看长文
- **具体** — 不要"我对贵公司充满热情"，要说你了解他们什么
- **匹配** — 重点说你和岗位的匹配点，不是你所有的优点
- **附件** — 简历以PDF格式附件，文件名用"姓名-学校-岗位-简历.pdf"
