# 职场新人手册：入职第一年

> 从学生到职场人，比你想象的变化更大。

---

## 心态转变

| 学生思维 | 职场思维 |
|---------|---------|
| 考试有标准答案 | 工作经常没有"正确"答案 |
| 独立完成作业是美德 | 团队协作是必须 |
| 追求满分 | 追求"够好+按时交付" |
| 学了再做 | 做中学 |
| 失败了扣分 | 失败了复盘 |

---

## 和上级相处

### 理解你的上级

你的上级也是人，也有压力。他的KPI里包含了你团队的产出。

**让上级放心的方式：**
- 主动同步进度（不要等他来问你）
- 遇到风险提前说（不要最后一刻才说"做不了"）
- 给选择题而不是问答题（"我建议A方案，原因是XX，你怎么看？"而不是"我该怎么办？"）

### 1:1 沟通

如果你的上级有定期1:1，珍惜这个时间。准备：
1. 本周进展
2. 遇到的阻碍
3. 需要上级帮忙的事
4. 你的成长困惑

## 技术成长路径

```
初级工程师 → 中级工程师 → 高级工程师 → 技术专家/管理者
   (0-2年)     (2-4年)      (4-7年)        (7年+)
```

### 初级→中级的关键跨越

- 从"能完成任务"到"能独立设计解决方案"
- 从"写代码"到"写可维护、可测试的代码"
- 从"自己干好"到"能带新人"
- 从"只关注技术"到"理解业务需求"

## 软技能清单

| 技能 | 为什么重要 | 怎么练 |
|------|-----------|--------|
| 技术文档 | 你走了别人能接手 | 每个项目写README |
| Code Review | 发现别人的bug也发现自己的 | 主动请求review |
| 汇报能力 | 上级需要知道你的进展 | 练习3句话说清一件事 |
| 时间管理 | 多任务并行是常态 | 用TodoList/日历 |

## 财务基础（中国版）

作为新人，搞清楚这几项：

| 项目 | 说明 |
|------|------|
| 五险一金 | 养老/医疗/失业/工伤/生育保险 + 住房公积金 |
| 个税 | 起征点5000元/月，用个税APP查看 |
| 公积金 | 公司和个人各交一部分，买房时有用 |
| 年终奖 | 不是法定的，看公司和合同 |

**建议：** 入职第一个月搞清楚自己的社保和公积金缴纳基数。

## 什么时候该跳槽

该跳的信号：
- 学不到新东西了（连续6个月没有技术成长）
- 薪资明显低于市场价（差距>30%）
- 管理层/文化有严重问题
- 有更好的机会（明确的、已拿到offer的）

不该跳的信号：
- 刚入职觉得不适应（前3个月是正常的）
- 和某个同事有矛盾（换个团队可能就好了）
- 看到别人跳槽涨薪焦虑了（别人的路不一定适合你）
