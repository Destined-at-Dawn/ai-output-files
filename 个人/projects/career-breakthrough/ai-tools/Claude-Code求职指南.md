# Claude Code 求职指南

> Claude Code不是替代你思考，而是加速你思考。

---

## 适合用Claude Code做的求职任务

| 任务 | 怎么用 | 效果 |
|------|--------|------|
| 简历初稿 | 给素材 → 让它按STAR写 → 自己改 | 省2-3小时 |
| 去AI味 | 把初稿给它 → 让它标记AI味句子 → 自己改 | 比自查更全面 |
| JD分析 | 粘贴JD → 让它提取关键词和匹配度 | 30秒搞定 |
| 面试模拟 | 让它扮演面试官 → 根据你的简历追问 | 随时练习 |
| 公司调研 | 给公司名 → 让它总结信息 | 快速了解 |
| 技术文档 | 给代码 → 让它写README/技术说明 | 文档质量提升 |

## 提示词模板

### 简历生成

```
我需要帮我写简历的项目经历部分。以下是我的素材：

项目名：[名称]
我的角色：[角色]
我做了什么：[粗略描述]
用了什么工具：[工具列表]

请按STAR法则帮我重写，要求：
1. 不要AI套话（赋能、深耕、闭环、综上所述）
2. 每条一个动作，动词开头
3. 加入量化指标（如果我没有具体数字，用"从0到1""100%""首次"等程度词）
4. 技术栈在描述中自然出现
5. 加粗技术名词和核心动作
```

### JD分析

```
以下是[公司名]的[岗位名]JD：

[粘贴JD内容]

请帮我：
1. 提取核心关键词（技术栈、工具、能力要求）
2. 对照我的简历，标注匹配和缺失的点
3. 建议我简历上应该强调哪些经历
4. 这个岗位的隐含要求是什么
```

### 面试模拟

```
你现在是一个[行业]公司的[岗位]面试官。我的简历如下：

[粘贴简历]

请按以下流程面试我：
1. 先问自我介绍
2. 根据简历追问3个技术问题
3. 问2个行为面试问题
4. 最后给我评分和改进建议

每次等我回答后再问下一个。如果我的回答太空泛，追问细节。
```

### 去AI味

```
以下是我的简历内容，请帮我：

1. 逐句检查，标记所有AI味/空泛的表达
2. 对每个标记的地方，给出具体的替换建议
3. 检查是否有重复的句式结构
4. 确保每段经历都有量化结果

[粘贴简历内容]
```

## 使用原则

### ✅ 正确用法

- **AI出初稿 → 人工修改 → AI检查 → 人工确认** — 四步循环
- 给AI真实素材，让它帮你组织语言
- 用AI做信息整理和格式转换
- 用AI模拟面试，练习回答

### ❌ 错误用法

- 直接复制AI生成的内容到简历
- 让AI编造你没有的经历
- 完全依赖AI做决策（AI不知道你的具体情况）
- 不检查就提交

## 其他AI工具对比

| 工具 | 最擅长 | 不擅长 |
|------|--------|--------|
| Claude Code | 长文生成、代码分析、面试模拟 | 实时信息（需要联网） |
| Gemini Pro | 快速问答、多语言、信息汇总 | 深度代码分析 |
| NotebookLM | 文档拆解、论文笔记 | 创意写作 |
| ChatGPT | 通用问答 | 中文表达有时不自然 |

**最佳组合：** Claude Code（简历+面试）+ Gemini（快速调研）+ NotebookLM（技术文档拆解）
