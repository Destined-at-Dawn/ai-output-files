# 面试准备清单

> 面试前30分钟过一遍这个清单，比看10篇面经有用。

---

## 面试前1天

### 公司调研（15分钟）

- [ ] 公司做什么的？主要产品/服务？
- [ ] 公司规模？最近有融资/裁员/新产品发布吗？
- [ ] 这个岗位具体做什么？技术栈是什么？
- [ ] 面试官是谁？（如果知道的话，查一下LinkedIn/脉脉）
- [ ] 准备一个"为什么想来这家公司"的回答（2句话够了）

### 材料准备

- [ ] 简历打印2份（现场面试）/ PDF存手机（线上面试）
- [ ] 项目代码/GitHub链接准备好
- [ ] 波形截图/运行截图存在手机里
- [ ] 笔记本+笔（记面试官的问题和反馈）

## 面试前30分钟

### 自我介绍

**30秒版（电话/初筛）：**
> 我叫XXX，XX大学XX专业大X，GPA X.X。主要做XX方向，做过XX项目（一句话亮点）。对贵公司的XX方向很感兴趣。

**2分钟版（正式面试）：**
> [学校+专业+年级] → [项目1：做了什么+核心成果] → [项目2：做了什么+核心成果] → [AI工具使用] → [为什么对这个岗位感兴趣]

### 核心问题准备

- [ ] "你做过什么项目？" — 准备2个项目，每个3句话
- [ ] "你最大的技术难题是什么？" — 准备1个具体案例
- [ ] "为什么选这个技术方案？" — 说明你考虑过替代方案
- [ ] "你有什么问题想问？" — 准备3个（见下方）

### 你要问面试官的问题

选3个：

**了解岗位：**
- "这个岗位日常的工作内容是什么？"
- "实习生会参与哪个阶段的项目？"

**了解团队：**
- "团队的技术栈是什么？"
- "新人入职会有mentor带吗？"

**了解成长：**
- "您觉得这个岗位最需要什么能力？"
- "实习生有转正机会吗？"

## 面试中注意事项

### 技术面

- 不会的问题不要编 → "这个我还没深入研究，但我知道XX，我会从XX方向去学"
- 回答用具体例子 → "我在做XX项目时遇到过类似问题，当时是这样解决的……"
- 可以画图/展示代码 → 带笔记本电脑，随时准备打开项目

### 行为面

- 用STAR讲故事，但不要像背课文
- 诚实 > 完美。说"我当时犯了一个错"比"一切都很顺利"更有可信度
- 不要说上一家公司/面试的坏话

### HR面

- "你期望薪资多少？" → 如果是初面，可以说"我更看重学习机会，薪资在合理范围就行"
- "你能实习多久？" → 诚实回答，不要过度承诺
- "你有什么问题？" → 不要说"没有了"

## 线上面试额外检查

- [ ] 网络稳定（有线 > WiFi > 手机热点）
- [ ] 光线充足，背景干净
- [ ] 麦克风和摄像头测试过
- [ ] 关闭无关软件通知
- [ ] 提前5分钟进入会议
