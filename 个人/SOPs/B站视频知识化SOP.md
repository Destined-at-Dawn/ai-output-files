# B站视频知识化 SOP

> **核心心法**：收藏 ≠ 学会。视频不文本化 = 看了白看。
> **目标**：把B站视频从「吃灰收藏」变成「可检索、可提问、可复用」的知识资产。

---

## 为什么要做

| 问题 | 后果 | 解法 |
|------|------|------|
| 收藏就等于学会了 | 疯狂吃灰，从不回看 | 强制文本化，进入知识库 |
| 视频内容AI读不了 | 无法检索、无法提问 | 字幕提取 → MD文档 |
| 信息在视频里是线性的 | 找某个知识点要重刷视频 | 文本化后Ctrl+F秒搜 |

---

## 工具链

```
B站视频 → SABATCH插件（字幕提取）→ MD文件 → AI知识库（Obsidian/Copilot）
```

**SABATCH 插件**（完全免费）
- 功能：批量获取B站视频字幕，导出为MD
- 支持：个人主页、收藏夹、合集、单视频
- 多语言：中文/英文/日文字幕可选
- 无字幕视频：调用本地语音识别模型转写（准确率~95%）

---

## 操作流程

### Step 1：安装 SABATCH

1. 下载插件（见视频置顶评论获取安装包）
2. 安装到浏览器

### Step 2：获取 Cookie

1. 打开SABATCH插件
2. 点击「获取Cookie」
3. 确保已登录B站

### Step 3：选择内容来源

根据需求选择获取类型：

| 类型 | 适用场景 |
|------|---------|
| 收藏夹 | 整理某个方向的全部收藏 |
| 合集 | 系统学习某个UP主的系列 |
| 个人主页 | 某个UP主的全部视频 |
| 单个视频 | 临时看到的好视频 |

操作：
1. 选择获取类型
2. 输入对应网址
3. 点击「获取当前网址」
4. 点击「获取视频」

### Step 4：获取字幕

1. 点击「获取字幕」按钮
2. 如果有多语言字幕，选择需要的语言（推荐中文优先，英文辅助）
3. 等待获取完成

**无字幕视频处理**：
1. 插件提示「没有外挂字幕」
2. 点击「转写无字幕」
3. 调用本地语音识别模型转写（时间较长，取决于电脑配置）
4. 转写完成后自动识别语言
5. 正常导出即可

### Step 5：配置导出

推荐配置：

| 设置项 | 推荐值 | 原因 |
|--------|--------|------|
| 导出格式 | **MD** | Obsidian/知识库直接读 |
| 压缩方式 | ZIP压缩 | 批量时方便管理 |
| 文件命名 | 第一个选项（含标题） | 方便识别 |
| 添加元信息 | **全部勾选**（标题/链接/作者/日期） | 追溯来源 |

### Step 6：导入知识库

1. 解压ZIP
2. 将MD文件复制到知识库对应目录
3. 在Obsidian/MOC中创建链接

## 知识库落盘规则

### 分类标准

| 视频内容 | 落盘位置 |
|----------|---------|
| 考研/学习方法 | `知识库/学习方法/` |
| 技术教程（FPGA/AI/编程） | `知识库/01-Concepts/` |
| 职业规划/求职 | `知识库/大学规划/` |
| 电气专业知识 | `知识库/03-Courses/` |
| 创作/自媒体 | `知识库/05-Content/` |

### 元数据模板

每个导入的字幕文件顶部应包含：

```yaml
---
type: video-note
source: "B站"
url: "https://www.bilibili.com/video/BV..."
author: "UP主名"
date: "YYYY-MM-DD"
tags: [b站, 视频笔记, {具体标签}]
status: unprocessed
---
```

### 后处理

字幕是**原始素材**，不是成品。导入后还需：

1. **粗筛**：标记值得深入的段落（用 `==高亮==`）
2. **提炼**：在文件底部加 `## 核心收获` 小节
3. **链接**：与已有的概念笔记、书籍笔记建立双向链接
4. **状态更新**：frontmatter 的 `status` 改为 `processed`

---

## 注意事项

- **字幕质量**：自动生成的字幕有错别字，关键术语需人工校正
- **转写耗时**：无字幕视频的本地转写速度取决于电脑配置，批量时预留时间
- **版权意识**：字幕仅用于个人学习，不公开传播
- **收藏优先级**：先处理「学了但没记笔记」的收藏，再处理「一直想看」的收藏

## 与百大认知书籍的联动

视频笔记可以与书籍概念互相印证：

```
视频：某UP主讲「间隔重复法」
→ 链接到：[[009-认知天性-概念3-间隔效应]]
→ 补充：视频里的实操案例 + 书里的理论支撑
```

这比单独看视频或单独读书都有效——**具身认知 + 编码多样性**。
