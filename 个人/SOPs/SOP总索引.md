## 二、消息路由规则

### 2.1 主题匹配

```
IF 说"学习/搞懂/入门/深度学习" → SOP-01 学习加速
IF 说"费曼/AI拷问/测试/模拟答辩" → SOP-02 费曼检验
IF 说"复盘/回顾/总结/做得怎么样" → SOP-03 复盘分析
IF 说"规划/目标/大学规划/长期计划" → SOP-04 人生规划
IF 说"拆解/道法术器/分析" → 直接调用 dao-fa-shu-qi skill
IF 说"澄清/决策/犹豫" → 直接调用 jc-clarifier skill
IF 说"记忆/自我改进" → 直接调用 li-memory + li-improve skill
IF 说"诊断AI用法" → 直接调用 ai-yuyi skill
IF 检测到个人信息变动信号 → 跨工作区共享SOP（个人信息同步）
ELSE → 回退到 CLAUDE.md 通用规则
```

### 2.2 深度学习链路（个人区独有）

```
用户说"深度学习xxx" →
1. step-search（搜索最新资料）

4. SOP-02 费曼检验（费曼检验）
5. dao-fa-shu-qi skill（深度拆解）
```

## 三、执行后标准收尾（出水管，必做）

1. 更新 `memory/{YYYY-MM-DD}.md` 当日记录
2. 更新 `memory/MEMORY.md` 索引（如有新增）
3. 更新 `memory/long-term.md`（如有重要决策/偏好变化）
4. **回写 SOP 迭代日志**（参照 R09 反馈闭环铁律 F1）
5. **教训文件必须包含「固化规则」段**（目标文件+目标段落+新规则文本）

## 五、自学习框架（R09 反馈闭环）

> 「教训只记不改 = 白记。」—— 复制自创作区 SOP总索引 v1.9

### 5.1 进出水管模型

```
进水管（Step 0）：每次任务开始前 → 读取教训库
  ↓ 执行任务
出水管（收尾）：每次任务结束后 → 回写 SOP + 回写自我进化 + 回写 memory
```

### 5.2 数据源

| 数据源 | 路径 | 回流目标 |
|--------|------|---------|
| 自我进化/做得好的鼓励/ | `自我进化/做得好的鼓励/` | SOP 最佳实践 |
| 自我进化/做得差的避免/ | `自我进化/做得差的避免/` | SOP 避坑清单 → 铁律 |
| memory/lessons-learned.md | `memory/lessons-learned.md` | SOP 迭代日志 |
| memory/long-term.md | `memory/long-term.md` | SOP 决策表 |

### 5.3 升级规则

| 次数 | 动作 |
|------|------|
| 第 1 次 | 记录到自我进化 + SOP 迭代日志 |
| 第 2 次 | SOP 对应步骤旁加 **⚠️ 高频坑** |
| 第 3 次 | 升级为 **铁律**，更新 CLAUDE.md |
