# 认知学习 SOP

> 个人工作区特有的认知学习标准流程。

## 学习流程

1. **选择书籍**：从百大认知书籍中选择
2. **阅读原文**：阅读书籍原文
3. **提取概念**：提取核心概念
4. **个人应用**：结合个人情况应用

## 学习记录

每次学习必须记录：

```markdown
## 认知学习记录

- **时间**：{YYYY-MM-DD}
- **书籍**：{书名}
- **概念**：{概念}
- **个人应用**：{应用}
```

## 学习频率

- **每日**：学习一个概念
- **每周**：学习一本书籍
- **每月**：总结学习成果
