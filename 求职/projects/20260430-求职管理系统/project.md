# 求职简历管理与面试复盘系统

## 项目目标
为李兰源建立一个长期运行的求职管理系统，核心功能包括：
1. 面向不同公司的定制简历生成与版本管理
2. AI模拟HR追问，帮助面试准备
3. 面试后复盘记录与经验沉淀
4. 素材库实时更新与复利积累

## 项目边界
- **项目文件内**：只存对话记忆、问题解决方案、CLAUDE.md约束
- **项目文件外**：简历文件、面试反馈、项目经历详情等实际产出物存放在 `E:\ai产出文件\牛马\求职\求职\` 目录

## 核心工作流
1. 每次投递新公司 → 读取素材库 → 拆JD → 生成定制简历 → 存入简历输出目录
2. 面试前 → AI模拟HR追问 → 准备话术
3. 面试后 → 记录反馈 → 更新素材库 → 沉淀经验

## 约束
- 遵循 CLAUDE.md 中的所有硬规则（v2.0，17章）
- 素材库实时更新能力
- 简历生成必须严格按照SOPs/00_简历生成SOP.md执行
- 自我进化机制：Pattern-Key防重犯 + 做得好/做得差分离
