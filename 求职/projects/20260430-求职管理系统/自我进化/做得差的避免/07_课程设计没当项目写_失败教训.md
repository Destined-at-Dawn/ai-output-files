# 做得差的避免 — 07

## 2026-04-30 课程设计/大作业没按项目经历写

- **Pattern-Key**：PK-求职-007
- **表象**：技术岗简历中，课程设计和大作业被轻描淡写地当作课堂作业
- **根因**：AI区分了"竞赛项目"和"课程作业"，但实际上求职时两者价值相同
- **后果**：少了一段可写经历，简历内容单薄
- **正确做法**：
  - 找技术岗位（特别是大公司），课程设计要说成项目
  - 判断标准：有明确任务 + 技术路线 + 实现过程 + 产出 → 就是项目
  - 不要轻描淡写地当作课堂作业
- **关联**：CLAUDE.md §1.5、参考简历中李证的写法（"找技术岗位需要自学课程设计这类的说成项目"）
- **教训密度**：★★★（内容丰富度）
