# 求职管理 SOP

> 求职工作区特有的求职管理标准流程。

## 求职流程

1. **准备**：准备简历和求职材料
2. **投递**：投递简历
3. **面试**：参加面试
4. **跟进**：跟进面试结果
5. **决策**：做出求职决策

## 求职记录

每次求职必须记录：

```markdown
## 求职记录

- **时间**：{YYYY-MM-DD}
- **公司**：{公司名称}
- **职位**：{职位}
- **状态**：{状态}
- **备注**：{备注}
```

## 求职频率

- **投递**：每周5-10份
- **面试**：按需参加
- **跟进**：面试后24小时内
