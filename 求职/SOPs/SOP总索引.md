## 三、SOP编写规范

> 统一模板：`E:\ai产出文件\牛马\mutual\mutual\sop-template.md`

每个SOP必须包含5个必选元素：
1. **触发条件**：什么情况下执行
2. **执行步骤**：编号步骤清单（可引用CLAUDE.md章节编号和skill名称）
3. **检查清单**：执行完毕后的自检项
4. **关联Pattern-Key**：本SOP涉及的已知错误模式
5. **关联文件**：执行时需要读取的文件路径

---

## 四、执行后标准收尾（出水管，必做）

1. 检查 `memory/MEMORY.md` Pattern-Key表是否有新增
2. 更新 `memory/{YYYY-MM-DD}.md` 当日记录
3. 如有新教训 → 更新 `自我进化/做得差的避免/` + Pattern-Key表
4. 如有素材变化 → 更新 `简历素材/我的经历素材库.md`
5. **回写 SOP 迭代日志**（参照 R09 反馈闭环铁律 F1）
6. **教训文件必须包含「固化规则」段**（目标文件+目标段落+新规则文本）


## 六、与CLAUDE.md的关系

- CLAUDE.md定义**规则**（必须/禁止/不得）
- SOP定义**流程**（先做什么→再做什么→最后检查什么）
- SOP引用CLAUDE.md的规则编号，不重复写规则内容
