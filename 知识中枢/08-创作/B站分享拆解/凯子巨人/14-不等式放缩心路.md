# 凯子巨人：不等式放缩——从「为什么」到「自然想到」
> **来源**：B站 · 分析师凯子巨人 · 2021-08-15
> **分析日期**：2026-05-31

## 一、核心洞察
> 不等式放缩的核心思想只有一个：「放缩是为了得到常数」。通过判断哪个形式能产生常数积，来选择放缩方式——从最简单的例子出发，自然推导到难题。

## 二、方法论拆解

### 道：放缩的目的是「消去变量，得到常数」
- 基本不等式 a+b ≥ 2√(ab) 的使用前提：ab的乘积必须是常数
- 如果ab乘积不是常数，放缩就没有意义——消不掉变量就得不到确定结论
- 「别人给你一个方案让你照做，和你自己主动找到这个方案有着天壤之别」

### 法：从简单例子出发的递进理解
1. **基础版**：X + 1/X → X与1/X乘积为常数1 → 可用基本不等式
2. **变式1**：X + 3/X → X与3/X乘积为常数3 → 同样可用
3. **变式2**：X + 1/(X+3) → 乘积不是常数 → 不能直接用
4. **配凑法**：给X配上一个(X+3) → (X+3) + 1/(X+3) - 3 → 乘积变常数
5. **推广**：任何形式 aX + b/(cX+d) 都可以配凑后使用基本不等式

### 术：核心辨识能力
- **判断标准**：两个量的乘积是否为常数？
- **配凑技巧**：给不匹配的项「穿上衣服」使其乘积为常数，放缩后再「脱去衣服」
- 复杂难题的拆解：看到多字母 → 大概率用不等式 → 检查各项配对后乘积是否常数
- 配凑法可以推广到几乎所有数列不等式问题

## 四、对小黎的价值

- **可迁移性**：电路中的功率优化问题（如最大功率传输定理）本质上也是不等式放缩——在约束条件下求极值。理解「放缩为了得到常数」的思想，在优化问题中同样适用
- **行动建议**：
  1. 遇到不等式/极值问题，先检查各项配对后乘积是否为常数
  2. 配凑法作为通用工具——形式不匹配时「加减同一项」构造匹配
  3. 从最简单的特例出发理解方法，再推广到一般情况

## 五、关键原话

- 「别人给你一个方案让你照做，和你自己主动找到这个方案有着天壤之别」
- 「需要运用到这个不等式的时候，A与B的乘积应该是一个常数，否则我们的放缩就是没有意义的」
- 「给前面的X配上一个(X+3)，问题就解决了。其实本质上并没有变，但是他把衣服脱了，你就认不出来了」
- 「放缩是为了得到常数，所以我们要通过判断哪个形式能够得到常数，来选择不同的不等式」
