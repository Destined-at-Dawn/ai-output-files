# 凯子巨人：不刷题不行，只刷题更不行——新高考的学习方向
> **来源**：B站 · 分析师凯子巨人 · 2022-01-29
> **分析日期**：2026-05-31

## 一、核心洞察
> 新高考的核心转向是：从「用固定的套路解决已知题型」到「用基础知识解决从未见过的问题情境」。刷题是手段，但只刷题而不回归基础知识，等于在沙滩上建楼。

## 二、方法论拆解

### 道：新高考的本质是什么
凯子对教育改革的判断非常清晰：
- 以前：老师修修改改市面上的卷子出题 → 学生觉得新颖就够了
- 现在：套路总结发挥到极致 → 「大招秒杀」泛滥 → 国家必须转方向
- 新高考方向：**选拔真正具备创新能力的人才** = 能用科学知识解决从未出现过的问题情境

「一个套路所针对的题型越是固定，那么这个套路也就越没有价值。」

### 法：回归基础知识
凯子对「基础」的重新定义非常重要：
- 「基础」≠ 简单
- 「基础」= **奠基性和根基性**
- 课本基础知识是「历史伟人用科学精神总结出来的阐述世界的规律」
- 在机械刷题的背景下，「基础」被冠以「简单」的名义被忽略

### 术：深入课本的呼吁
凯子承认现实的困难：
- 大多数同学不擅长自学，更擅长被动接收信息和练习熟练度
- 即使是他的学生和粉丝，对深入课本的呼吁仍然「大部分无动于衷」
- 他的承诺：「如果还有人没有开始深入课本基础，我就会一直发出这种呼喊」

### 凯子对高考趋势的预判
- 趋势：越来越多的题目无法使用固定的题型套路来预测
- 方法论：无法预测具体考什么 → 但可以预测「考基础知识的灵活运用」
- 对策：不是刷更多题，而是更深入地理解基础知识

## 三、百大认知映射

| 概念 | 百大认知 | 引用 | 分析 |
|------|---------|------|------|
| 套路的局限性 | 《刻意练习》 | 「天真练习：反复做同一件事并不能提升能力」 | 刷套路题 = 天真练习，看似努力但能力没有本质提升 |
| 基础知识的深度 | 《认知负荷理论》 | 「图式自动化：基础知识的深度决定了上层建筑的高度」 | 基础不牢 = 图式不完整 → 遇到新情境无法灵活调用 |
| 创新能力的本质 | 《刻意练习》 | 「心理表征：专家能识别新手看不到的模式」 | 「用知识解决从未见过的问题」= 依赖深度心理表征，而非套路记忆 |
| 教育改革的方向 | 《认知天性》 | 「检索式练习：考试本身就是最好的学习方式」 | 新高考反套路 = 强制学生在考场上主动检索知识，而非回忆套路 |
| 自学能力的重要性 | 《心流》 | 「清晰目标：知道自己在做什么、为什么做」 | 被动接收信息 ≠ 学习，主动探索基础知识才是真正的学习 |

## 四、对小黎的价值

- **可迁移性**：考研数学/专业课同样在向「反套路」方向发展——近年真题越来越注重基础知识的灵活运用。凯子的判断对考研备考同样适用。
- **行动建议**：
  1. 考研数学复习时，不要只刷题型总结，要回到教材理解每个定理的推导过程
  2. 对每个重要公式问「为什么它是这样的」而非「它怎么用」
  3. 建立「基础知识 > 套路技巧」的优先级意识
  4. 遇到新题型时，先回归基础知识寻找解题线索，而非回忆见过的类似题

## 五、关键原话

- 「新高考是反套路的，即使是那些并没有并入新高考的省份，也会有样学样响应国家号召」
- 「基础指的是奠基性和根基性，你所鄙视的课本基础知识，是历史伟人用科学精神总结出来的阐述世界的规律」
- 「一个套路所针对的题型越是固定，那么这个套路也就越没有价值」
- 「能够用科学知识去解决从来没有出现过的问题情境，这才真正反映复杂现实的问题情景」
- 「国家已经在卖力地叫停这种无意义的内卷了，所以现在是时候让我们回归基础知识了」
