# 凯子巨人：理科强者的下意识数学思路
> **来源**：B站 · 分析师凯子巨人 · 2023-03-17
> **分析日期**：2026-05-31

## 一、核心洞察
> 学好数学的两个方向——深刻理解基础知识 + 学会拆解问题的万能方法——相互促进、不可分割。大多数学生的问题根源是「把数学学成了死记硬背」，而非缺乏天赋。

## 二、方法论拆解

### 道：表象记忆 vs 底层逻辑
凯子对当前教育问题的诊断极其精准：
- **经验主义泛滥**：老师用题型、技巧、结论、套路、大招来解释题目
- **学生的错觉**：以为成为学霸 = 记住各种解题技巧和题型
- **恶性循环**：记不住 → 刷更多题 → 还是记不住 → 约等于碰运气

「如果你记住的题目太少，考试就不会用；如果你记住的题目太多……」（暗示记忆负担过重，同样无法灵活运用）

### 法：两个学习方向
1. **深刻理解基础知识**：不是背公式，而是理解公式背后的数学对象关系
2. **学会拆解问题的万能方法**：不是记套路，而是学会分析问题的通用原则

这两个方向「相互促进不可分割」——理解基础知识越深，拆解问题越容易；拆解问题越多，对基础知识的理解越深。

### 术：万能解题思路
凯子将其命名为「多路并进分析法」（详见其他视频），核心特征：
- 适用于初中、高中、数学、物理
- 上到送分题，下到竞赛题
- 本质是「分析和拆解问题」的能力
- 不是具体的方法，而是思考的通用原则

### 对教师群体的批判
凯子指出了一个深刻的问题：
- 老师把所有精力放在数学题目上 → 变得对题目过于熟悉
- 用题型经验来解释题目 → 忘记了自己学生时代也做不到
- **专题总结得越细，学生反而越不会做题**
  - 专题训练时：学生提前知道了要用什么知识
  - 考试时：学生要自行分辨该用什么知识 → 分辨成本随题型数量增加而增加

## 三、百大认知映射

| 概念 | 百大认知 | 引用 | 分析 |
|------|---------|------|------|
| 经验主义的危害 | 《刻意练习》 | 「天真练习：反复做同一件事并不能提升能力」 | 刷题型 = 天真练习，看似熟练但能力没有本质提升 |
| 两个学习方向 | 《认知负荷理论》 | 「图式自动化：基础知识的深度 + 问题解决的自动化」 | 理解基础 = 建立完整图式；拆解问题 = 图式的灵活调用 |
| 专题总结的悖论 | 《认知天性》 | 「流畅度错觉：熟悉感不等于掌握」 | 专题越细 → 学生越觉得会了 → 考试越不会 → 因为「会了」是错觉 |
| 底层逻辑 vs 表象记忆 | 《刻意练习》 | 「心理表征：专家看到本质，新手看到表面」 | 底层逻辑 = 深层心理表征；表象记忆 = 浅层心理表征 |
| 解题能力 ≠ 知识能力 | 《认知负荷理论》 | 「专长反转效应：对新手有效的教学方法对专家可能无效」 | 知识能力是基础，解题能力是综合运用——两者需要不同的训练方法 |

## 四、对小黎的价值

- **可迁移性**：电气工程学习同样面临「经验主义 vs 底层逻辑」的问题——很多同学靠记电路模型的套路来解题，而非理解电路的基本原理。考研数学/专业课同样需要「两个方向」并进。
- **行动建议**：
  1. 定期自检：「我是在理解底层逻辑，还是在记忆表象套路？」
  2. 学习新知识时，先理解原理再做题（不要跳过原理直接刷题）
  3. 做完题后回顾：「我是怎么想到这个方法的？」——培养元认知能力
  4. 对老师的「大招秒杀」保持警惕——它可能只是在帮你死记硬背

## 五、关键原话

- 「想要学好数学，主要有两个学习方向：一是深刻地理解基础知识，二是学会拆解问题的万能方法」
- 「你把专题总结得越细，学生反而越不会做题」
- 「很多学生喜欢用经验来替代逻辑推理——既然想不通，那就记住就好了」
- 「你会发现自己连死记硬背可能都做不到，因为学的东西实在是太多太杂了」
- 「国家不想培养一堆对社会没有用的人，一堆人去做没用的事情，那就叫社会内卷」
