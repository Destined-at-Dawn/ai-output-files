# 凯子巨人：学习的盲区——意义、记忆与知识网络
> **来源**：B站 · 分析师凯子巨人 · 2023-03-04
> **分析日期**：2026-05-31

## 一、核心洞察
> 艾宾浩斯记忆曲线针对的是「无意义音节」，而我们学习的是「有意义的信息」。记忆的本质不是重复，而是在知识网络中建立连接——连接越多的节点，越不容易被遗忘。

## 二、方法论拆解

### 道：意义是记忆的前提
凯子提出了一个关键区分：
- 短期记忆（如随机数字102456）≠ 真正的记忆
- 大脑有一个「判定信息价值」的过程——这就是「意义」
- 意义 = 连接：一个信息在你的知识网络中连接越多，就越有意义，越容易被记住

「如果说一个信息他在你的脑海里面的网络结构中非常重要，你就越倾向于去记住它。」

### 法：知识网络而非记忆曲线
凯子对艾宾浩斯记忆曲线的批判非常犀利：
1. 艾宾浩斯实验对象是「无意义音节」，不是有意义的知识
2. 按照记忆曲线反复复习 = 一直在记东西，信息量大了根本不可持续
3. 观察学霸：他们不会说「有些公式时不时忘了，只能回去复习」
4. 网上很多「记忆宫殿」方法是骗人的——随机发明故事去记公式，只是把公式记住了，没有学到知识

真正的「记忆宫殿」是知识本身的结构——数学知识本来就有结构，把它组织成网状结构，自然就记住了。

### 术：如何建立意义
1. 从媒介（声音、视频、交流）获取信息 → 转换成语言符号 → 进入大脑
2. 关键步骤：**思考**——提炼意义，建立连接
3. 信息进入大脑后有一个「价值判定」过程
4. 有连接的信息被保留，孤立的信息被遗忘
5. 遗忘发生在「价值判定」之后，所以要在「判定」这一步下功夫，而非等到忘了再复习

### 凯子的记忆模型
```
信息输入 → 语言符号化 → 进入大脑 → 价值判定（意义=连接）
                                          ↓
                                    有连接 → 保留
                                    无连接 → 遗忘
```

## 四、对小黎的价值

- **可迁移性**：电气工程中的公式（如麦克斯韦方程组、傅里叶变换）同样可以通过「知识网络」来理解而非死记。每个公式的「意义」在于它与其他公式的连接关系。
- **行动建议**：
  1. 学习新公式时，先画出它与其他已知公式的连接图，而非直接背诵
  2. 复习时不是「重新看一遍」，而是「重新建立连接」——问自己「这个公式和哪些知识有关？」
  3. 放弃机械的艾宾浩斯复习法，改用「网络化复习」——沿着知识网络的连接路径复习
  4. 考研政治/英语单词同样适用：孤立的单词难记，放在语境和词根网络中容易记

## 五、关键原话

- 「意义它很多时候是怎么样的？它是连接的，是和其他的信息连接在一起的，就像一张网」
- 「艾宾浩斯记忆曲线——他实验的对象是无意义音节，但我们学习的都是有意义的信息」
- 「记忆不是这样的……你一直要去复习吗？那一开始信息少还可以，后来越来越多你难道一直去复习吗？」
- 「真正的记忆宫殿是知识本身的结构，而不是你随机发明一个什么故事去记」
- 「你要看书看信息去思考，接下来你会遗忘很多无用的信息，而我们要记住它，其实就是在这一步要去做努力」
