# 凯子巨人：超前模拟运算——做难题的思维模型
> **来源**：B站 · 分析师凯子巨人 · 2021-06-13
> **分析日期**：2026-05-31

## 一、核心洞察
> 超前模拟运算 = 不做具体运算，只追踪量与量之间「追根溯源」的关系，以最低成本进行思路探索。核心是「你会用导航」不等于「你会画地图」——看懂答案不等于会解题。

## 二、方法论拆解

### 道：导航 vs 画地图
凯子提出了一个极其精准的类比：
- 看答案觉得简单 = 「你会用导航」
- 自己做不出来 = 「你不会画地图」
- 解析几何「一听就会，一做就废」的根本原因：你只会跟着答案走，不会自己规划路径

「你要说你能看懂，那也只能证明你会用导航，而我们真正需要的却是画地图的能力。」

### 法：超前模拟运算
具体操作：
1. 从问题出发，找到目标量
2. 追踪目标量依赖于哪些量
3. 不做具体运算，只关心「量与量之间的关系」
4. 建立从问题到已知条件的「关系链」
5. 关系链通了 → 思路确定 → 再做具体运算

以一道解析几何为例：
- 目标：求 CD 的长度
- 发现 CD 与 AB 有关系 → AB 是已知的 → 只需求中间量
- 中间量与斜率有关 → 斜率与 M 有关 → M 可以从方程求出
- 所有量都能表示成 M 的函数 → 维度为 1 → 一个方程就够了

### 术：维度概念的引入
「维度」= 包含的未知量个数：
- 维度 0：已知的函数值
- 维度 1：包含一个未知量（如含 M 的表达式）
- 维度 2：包含两个未知量
- 高中数学一般维度不超过 2

维度的核心作用：**简化分析**。不需要做具体运算，只需要关心每个数学概念需要几个字母来表示。

### 术：信息整理法
面对复杂题目时：
1. 把各个几何元素在草稿纸上列出来
2. 用「继承」的方式画出元素之间的产生关系
3. 把条件列到对应元素旁边
4. 列出隐含条件（如韦达定理可用 → 意味着德尔塔>0）

## 四、对小黎的价值

- **可迁移性**：电路分析中的「超前模拟运算」——先追踪电压/电流/阻抗之间的依赖关系（哪些量是独立的，哪些是派生的），再列方程求解。考研数学的证明题同样适用：先追踪结论依赖于哪些条件，再规划证明路径。
- **行动建议**：
  1. 做大题前先花1分钟画「关系链」——目标量依赖于什么？那些量又依赖于什么？
  2. 引入「维度」概念分析问题复杂度：这道题本质上是几维的？
  3. 复杂题目的信息整理：列出所有元素，画出继承关系，标出条件
  4. 训练「画地图」能力：做完题后回顾「我是怎么想到这个方法的」，而非只看答案

## 五、关键原话

- 「超前模拟运算就是不做具体运算，只在意量与量之间追根溯源的关系，从而以最低成本进行思路探索」
- 「你要说你能看懂，那也只能证明你会用导航，而我们真正需要的却是画地图的能力」
- 「很多被认为是天赋卓越的同学，其实都会下意识的如此思考」
- 「这个方法应对越是复杂的题目就越是有效」
