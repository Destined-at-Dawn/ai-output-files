# 凯子巨人：联结性知识 vs 知识碎片——三角恒等式的网络化理解
> **来源**：B站 · 分析师凯子巨人 · 2021-05-15
> **分析日期**：2026-05-31

## 一、核心洞察
> 知识碎片风一吹就散，联结性知识密不透风。所有三角恒等式都可以从一个「始祖公式」出发，通过代数替换逐步推导——形成知识网络后，忘了也能推回来。

## 二、方法论拆解

### 道：知识的本质是连接，不是记忆
- 大脑中的知识如果是一盘散沙，风一吹就散
- 学习数学最重要的是将知识相互连接起来，形成思维组块
- 深入理解的知识结构，即使忘了也有自信推理出来
- 每次推理都是一次有意义的记忆强化训练

### 法：从始祖公式出发的推导网络
1. **始祖公式**：cos(α-β) = cosαcosβ + sinαsinβ（单位圆 + 向量余弦公式推导）
2. **sin展开式**：用诱导公式转换函数名 → 整理为两式相减 → 利用始祖公式展开
3. **tan展开式**：拥有sin和cos展开式后自然得出 → 上下同除cos消元
4. **α+β展开**：将β整体替换为-β（代数替换，换汤不换药）
5. **二倍角**：将β替换为α → 特殊化
6. **半角公式**：利用sin²+cos²=1消元 → 从二倍角公式中提取
7. **辅助角公式**：提取系数构造平方和为1 → 设为同一角的cos和sin → 应用展开式

### 术：代数替换的威力
- 核心技巧：「贝塔只是一个代数称谓，你可以换成任意的形式」
- 从α-β出发 → 替换为α+β → 再替换为2α → 一条推导链串联所有公式
- 知识网络图：始祖公式居中，各公式通过推导箭头连接
- 「老瓶装新酒，换汤不换药」——同一套代数替换技巧反复使用

## 四、对小黎的价值

- **可迁移性**：电路分析中的阻抗公式、功率公式同样可以从几个基本定律（欧姆定律、基尔霍夫定律）推导出来。建立「始祖公式 → 推导网络」的思维模式，比背公式表可靠得多
- **行动建议**：
  1. 对任何需要记忆的公式组，先找到「始祖公式」，再推导整个网络
  2. 推导路径画成关系图（知识可视化），忘了就从图上重新推
  3. 代数替换是通用技巧——在电路变换（Y-Δ变换等）中同样适用
