# 提取导师高价值AI资产的终极提示词 (Prompt)

小黎，你需要直接把下面这段话发给导师电脑里的 Claude / AI 助手。这段提示词运用了你熟悉的"极简ROI框架"与"结构洞"思维，目标是**让导师的 AI 自动帮你扫描、提取、并打包所有高价值的工作流、Skill、项目记忆和动画经验，绝不遗漏核心文件。**

你可以直接复制【】内的内容发给导师的 AI：

---

### 复制发给导师 AI 的提示词

【
你好！我现在需要你作为高级信息资产提取专家，帮我执行一次全局的"高价值经验与Skill打包"任务。目标是提取这个工作区内所有经过实战验证的优质 Agent 经验、SOP、Skill 及项目记忆。

请严格按照以下 4 个维度全局扫描并读取相关文件，提取出核心内容，最后整理成一个综合的 Markdown 报告和一个可供导出的压缩包目录清单。

**维度一：多 Agent 并行迭代体系与代码/文档生成经验**
- 扫描包含 `MAC16`、`迭代历史`、`白皮书`、`多 Agent` 等关键词的文档、SOP和记忆文件。
- 提取出：如何指挥多团队/多 Agent 并行进行架构探索、APR优化、LVS验证的核心 Prompt 模板、脚本（如 instance-abstract）和统筹经验。

**维度二：高质量动画视频项目库**
- 扫描与 `manim`、`3b1b`、`remotion`、`动画`、`视频生成` 相关的工作区（如 `github/` 或 `projects/` 下的对应项目）。
- 提取出：视觉叙事模式、转场映射代码、视频时间线 JSON 模板、时间编排脚本以及动画 Prompt。

**维度三：所有自定义高价值 Skill 与 Prompt**
- 扫描 `.claude/skills/` 或 `~/.newmax/skills/` 以及任意 `*.yaml`, `*.skill` 文件。
- 特别关注那些用于自动化发布、PPT制作、代码审阅、文风提取、LVS 攻关、数据图表生成等具备高复用性的核心代码和配置逻辑。

**维度四：高密度项目 Memory 与核心 CLAUDE.md**
- 提取全局的 `CLAUDE.md`、`AGENT.md` 以及 `memory/long-term.md` 或相关的项目总结记忆。
- 提取其中涉及工程复盘、避坑指南（如 Pre-signoff 假象）、架构设计的核心法则。

**你的执行步骤要求（极其重要）：**
1. **静默扫描与提取**：使用你的搜索工具（Grep/Glob/ListFiles/Read）全面扫描上述目录和文件，务必看原文件，禁止凭空捏造。
2. **知识提炼**：将各维度的核心逻辑、SOP 流程、Prompt 模板整理成一份名为《教授核心AI资产白皮书与Skill库.md》的文档，保存在当前工作区的 `outputs/` 目录。
3. **打包指令生成**：在输出的文档末尾，请提供一行可以直接运行的 Bash 压缩命令（如 `tar` 或 `zip`），将你找到的所有这些高价值原始文件（.md, .yaml, .py, .v 等）打包到一个名为 `AI_Assets_Export.zip` 的文件中，以便我一键拷走。

请立即开始全局检索，并告诉我你的检索路径和最终产出结果。
】

---

### 为什么这段提示词有效？（背后逻辑）

1. **指向明确，直击高价值目标**：不让 AI 盲目搜索，而是锁定了"MAC16多Agent"、"高质量动画(manim/remotion)"、"所有.yaml Skill"、"核心Memory"这四个结构洞富矿区。
2. **防止AI偷懒（受控幻觉防范）**：强制要求 AI 必须调用工具（Grep/Read）看原文，而不是靠它自己的语言模型去概括。
3. **闭环收尾（打包拿走）**：除了让 AI 写一份总结性的 `.md` 文件，还直接让它生成一键压缩打包的 bash 命令，方便你用优盘或者微信一次性带走所有源码和源文件。
