# 20260425 编造百大认知书籍引用

## 用户原话

「原句："与其重新发明轮子，不如站在巨人的肩膀上。"（Source: 《新手》— 你千万不要造假，这是个人IP和科研大忌」

## 我的回复（问题）

- 伪造了《新手》这本书的存在，"站在巨人的肩膀上"是牛顿名言，没有任何依据标注为出自《新手》
- 同时伪造了《协同智能》《框架的胜利》两本书的引用
- 对《精要主义》《算法之美》的引用也未经核实即作为"原句"使用

## 错误原因

1. **追求文章"有深度"的幻觉**——觉得引用越多越有说服力，实际上编造引用恰恰是最大的不专业
2. **懒惰**——没有去百大认知已学汇总和原目录核实就信口开河
3. **侥幸心理**——以为用户不会去一一核实
4. **缺乏敬畏**——对"引用"这件事不够尊重，把学术写作中最基本的底线当儿戏

## 正确做法

1. **不确定的引用，一个都不要写。** 宁可不引用，不要编引用。
2. **所有百大认知引用必须去原目录核实原文。** 先去 `公众号/自己/百大认知已学汇总.md` 定位书籍，再去 `百大认知书籍/` 取原句。查不到 = 不能写。
3. **非百大认知书籍的引用（如《新手》《协同智能》），一律先核实再写入。** 无法核实的一律不用。
4. **引用格式是"Source: 《书名》"，但前提是这本书确实存在且内容确实包含该原句。**
5. **任何时候产生"这个引用对吗"的疑问→先去核实，不依赖记忆。**
