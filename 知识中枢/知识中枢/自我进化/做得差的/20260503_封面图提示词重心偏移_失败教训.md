# 20260503 封面图提示词重心偏移 — 失败教训

## 用户原问题
为公众号文章「简历写两天就够了」生成封面图提示词

## 我的回复（4个方案）
- 方案A：手握offer信封 + 12H时钟
- 方案B：桌面俯拍 + 工具平铺
- 暖色轨道版：白底 + 红橙 + 时钟
- 暗色轨道版：暗底 + 陶土橙

## 用户点评
1. **方案A**：构图不错，但12小时不是重点，STAR法则+简历去魅才是
2. **方案B**：太空了，背景太白，抓不住重点
3. **暖色轨道版**：效果最好，但STAR比12小时更值得强调；时钟≠12小时，应该用倒计时闹钟
4. **暗色轨道版**：效果最差，空洞无重点

## 错误原因
1. **重心偏移**：把"12小时"当成主视觉锚点，但文章真正的卖点是STAR法则+简历去魅
2. **视觉隐喻失败**：普通时钟无法让人联想到"12小时"，应该用倒计时闹钟/番茄钟
3. **暗色版信息密度不够**：暗色背景需要更强的视觉元素填充，否则显得空洞
4. **桌面俯拍太散**：元素平铺没有主次之分，读者不知道看哪里

## 正确做法（规则）
1. **必须**先确认文章的核心卖点是什么，再决定主视觉元素——不是标题里最显眼的数字，而是文章真正要传达的方法论
2. **必须**用具体的视觉隐喻——时钟≠时间限制，倒计时环/闹钟才是
3. **必须**确保封面图有明确的视觉层级——一个主元素+一个副元素，不能平铺
4. **禁止**在暗色背景上放太少元素——暗色需要更强的对比和更密的信息
5. **禁止**把文章中的次要信息当成封面主视觉
