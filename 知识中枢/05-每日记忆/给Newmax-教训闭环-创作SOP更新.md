## 任务 2：多图一致性验证

在创作相关SOP中添加以下规则：

```markdown
### 多图一致性规则

当同一任务需要生成多张图时：

1. **先锁定参数**：视角、透视、消失点、镜头焦距、风格
2. **生成三联母版**：同一构图内三格（如：线稿 + 彩图 + 人物合成）
3. **逐张校验**：窗框位置、仪表台位置、前景元素是否一致
4. **不一致 = 重做**，不能"解释差异"
```

## 任务 3：模板填充边界规则

在创作相关SOP中添加以下规则：

```markdown
### 模板填充规则

收到"基于XX模板填写"类任务时：

1. **母版唯一**：以原始模板文件作为唯一母版
2. **只填内容**：项目名称、编号、正文内容
3. **保留原样**：表格结构、段落样式、字体体系一律不动
4. **对齐规范**：正文左对齐 + 中文首行缩进；只有小标题居中
5. **交付校验**：
   - 正文单元格段落样式仍为 `Table Paragraph`
   - 普通正文没有居中对齐
   - `document.xml` 中正文大栏 `center_count=0`

**禁止**："重做文档"——只能"打开模板 → 填内容 → 局部必要修正 → 校验结构未变"
```
