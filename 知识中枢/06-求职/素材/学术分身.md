# 国家电网求职助手

## 你的目标岗位与地域分析

| 目标 | 匹配度 | 难度 | 关键筹码 |
|------|--------|------|---------|
| 无锡供电公司 | ★★★★★ | ★★★★★ | 学历（华电/河海硕士）+ 本地关系 + 竞赛经历 |
| 常州供电公司 | ★★★★★ | ★★★★☆ | 学历 + 竞赛经历（相对无锡略易） |
| 浙北（杭州/宁波） | ★★★★☆ | ★★★★★ | 学历 + 浙江户籍优势（如果不是浙江人难度极高） |

**结论**：你的地域偏好（无锡/常州）与国网招聘难度匹配——学历达标（华电/河海硕士）就有机会，但需要高笔试分数+面试表现。

## 国网招聘全流程

```
网申（9-10月）
  ↓
笔试（11月底-12月初）
  ↓
面试（12月-次年1月）
  ↓
录用公示（次年1-2月）
  ↓
入职（次年7月）
```

### 1. 网申（关键门槛）

**硬性条件**（不满足直接刷）：
- 学历：硕士（核心生产岗基本要求硕士）
- 学校：原电力部直属高校（华电、河海、三峡等）或985/211
- 专业：电气工程及其自动化、电力系统及其自动化、智能电网等对口专业
- 英语：四级425+（大部分省要求六级，江苏/浙江卡六级）
- 计算机：二级（大部分省要求）

**你的情况**：
- ✅ 学历：目标院校硕士 ✓
- ✅ 专业：集成电路转电气，需确保专业名称符合「电气类」大类
- ⚠️ 英语：四级保底，六级必须过（江苏/浙江硬性要求）
- ⚠️ 计算机二级：大二前必须拿下

**网申技巧**：
- 提前准备材料扫描件（成绩单、证书、照片）
- 志愿填报：无锡/常州可同时报，但只能选一个市局（另一个选区分局）
- 填写时突出：GPA排名（前5%）、竞赛获奖、项目经历

### 2. 笔试（决定性环节）

**笔试结构**（以电工类为例）：
| 模块 | 题量 | 时长 | 分值 | 你的准备重点 |
|------|------|------|------|-------------|
| 行测 | 20-30题 | 60分钟 | 约30分 | 言语理解、数量关系、资料分析（刷题即可） |
| 专业知识 | 100题左右 | 100分钟 | 约100分 | 电路、电分、继保、高电压、电机 |
| 电网知识 | 10-20题 | 20分钟 | 约10分 | 国家电网章程、企业文化、最新政策 |
| 英语 | 20-30题 | 30分钟 | 约10分 | 阅读理解（四级难度） |

**总分计算**：笔试65% + 面试35%（笔试权重更高）

**备考策略**：
- 行测：用《国家电网招聘考试行测》刷题，每天30分钟
- 专业课：用《电路》（邱关源）+《电力系统分析》（何仰赞）+《继电保护》（贺家李）
- 电网知识：关注「国家电网有限公司」官网，背企业文化
- 英语：每天1篇阅读理解（四级难度）

**你的优势**：
- GPA 3.9 → 专业课基础扎实，复习效率高
- ROS竞赛 → 工程思维强，电路/控制类题目上手快

**时间安排**（大三备考）：
- 3-6月：专业课一轮（教材+课后题）
- 7-9月：专业课二轮（考研资料+电网真题）
- 10月：行测+电网知识集中突破
- 11月：全真模拟（每天一套真题）

### 3. 面试（综合考察）

**面试形式**：结构化/半结构化（5-7位考官，10-15分钟）

**常见问题类型**：
1. 自我介绍（1-2分钟，必考）
2. 专业问题（电路/电分概念，如「简述牛顿拉夫逊法」）
3. 项目经历深挖（ROS竞赛的每个细节都要能讲）
4. 求职动机（「为什么选择国网」「为什么选无锡」）
5. 情景题（「如果安排你去偏远地区，你愿意吗」）

**面试准备**：
- 自我介绍：突出「成绩排名+竞赛经历+科研潜力」，控制在1分半
- 项目经历：用STAR法则（情境-任务-行动-结果）准备ROS项目
- 专业问题：复习核心概念，准备10-15个高频问题的答案
- 着装：正装（西装+领带），干净整洁
- 礼仪：进门问好、离开道谢、目光交流

**你的差异化**：
- 「AI+电气」背景（Plan B）→ 展示前瞻思维
- ROS组织协调经验 → 展示团队合作和项目管理能力
- GPA 3.9 → 学习能力强

### 4. 录用与选择

**offer 排序**：国网会按笔试+面试总分排序，从高到低录取。
- 如果笔试分数高（前20%），可以冲无锡/常州主城区
- 如果中等，常州分局或浙北（非杭州宁波）有机会
- 如果偏低，可能被调剂到县级局

**你的策略**：
- 目标：笔试分数 > 75/100（越高越好）
- 志愿：第一志愿无锡/常州，第二志愿留浙江（如果户籍浙江）
- 底线：接受县级局，先进去再说（后期可调动）

## 关键时间节点（你的考研+求职时间线）

| 时间 | 考研节点 | 国网节点 | 你应该做什么 |
|------|---------|---------|-------------|
| 大一（现在） | - | - | 打基础：高数/线代/英语学扎实<br>GPA保持前5%<br>过四级 |
| 大二上 | - | - | 专业课（电路）学透<br>过六级<br>计算机二级 |
| 大二下 | - | - | 进实验室补科研<br>ROS竞赛冲刺<br>开始考研数学一轮 |

# 时间管理与习惯养成助手

## 你的时间现状与痛点

**你的描述**：
- 每天4-5节课，有空闲时间，但空闲时自动刷抖音
- 想做的事很多（ROS、小红书、绩点、四级），但不知道先做什么
- 等转专业结果（4月底），不敢全力投入，怕做「无用功」

**核心问题**：
1. **启动成本太高**：不是「开始学习」，而是「打开PPT第一页」就已经很困难
2. **优先级混乱**：多个任务看起来都重要，结果哪个都没做
3. **习惯替代无效**：知道刷抖音不好，但不知道空闲时该做什么
