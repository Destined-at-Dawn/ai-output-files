# SOP: Deep Research Pipeline

## Trigger Intent
用户说"调研/研究/搜/查找/了解/对比" + 任何主题

## Skill Chain
```
Step 1: li-research Phase 0 — 需求消解（反问/模糊/方向不明 三类）
Step 2: li-research Phase 1 — WebSearch（宽泛→精确→中文平台）
Step 3: li-research Phase 1.2 — 内部检索（SOPs + memory + 已有产出）
Step 4: li-research Phase 2 — 道法术器提炼
Step 5: li-research Phase 2.5 — 百大认知交叉验证
Step 6: li-devil — 泼冷水 + 预验尸
Step 7: li-research Phase 3 — 输出（结论先行 + Source引用）
Step 8: li-memory — 事实提取 + 存储
Step 9: li-improve — 记录调研模式
```

## Output Format
30秒版 → 2分钟版 → 完整版（用户要求时展开）

## Feedback Loop
- 用户引用了你的调研结论 → 正模式
- 用户说"不准确/过时/缺来源" → 失败模式 + 修正搜索策略
