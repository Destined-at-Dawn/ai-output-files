```
IF 说"学习/搞懂/入门/学xx章" → SOP-01 课程学习

```

## 六、自学习框架

### 进出水管模型

```
进水管（Step 0）：每次学习任务开始前 → 读取教训库 + 查 project.md 当前进度
  ↓ 执行学习任务
出水管（收尾）：每次学习结束后 → 回写 SOP + 更新 project.md + 写入 memory
```

### 升级规则

| 次数 | 动作 |
|------|------|
| 第 1 次 | 记录到自我进化 + SOP 迭代日志 |
| 第 2 次 | SOP 对应步骤旁加 **⚠️ 高频坑** |
| 第 3 次 | 升级为 **铁律**，更新 CLAUDE.md |
