# SOP-01 课程学习（新章节/新知识点）

> **触发条件**：用户说"学习/搞懂/入门/学xx章节/新课"
> **认知科学支撑**：检索式练习（009-认知天性）、间隔重复（009-认知天性）、深度工作（005-深度工作）

---

## 流程

### Step 0: 进水管（任务前检查）

1. Read `memory/MEMORY.md` → 查本课程当前进度和已有教训
2. Read 课程 `project.md` → 确认当前在哪个阶段、哪些已学/未学
3. Read `自我进化/做得差的避免/` → 查本课程或类似课程的踩坑记录

### Step 1: 材料预处理

1. 用户提供教材/课件/笔记 → 用 pdf skill 提取内容
2. 结构化整理：章节目录 → 核心概念清单 → 难度预判
3. 输出「章节地图」：本章包含哪些知识点、前置依赖是什么

### Step 2: 概念讲解（道法术器式）

对每个核心概念：
1. **道**：这个概念解决什么问题？为什么需要它？（直觉理解）
2. **法**：核心原理/定理是什么？怎么推导的？（逻辑理解）
3. **术**：怎么用？有哪些典型题型？（操作理解）
4. **器**：有哪些工具/技巧/公式表？（效率工具）

> **⚠️ 关键**：不直接抛出公式，先建立直觉。比如讲导数，先说"瞬时变化率"，再推公式。

### Step 3: 费曼检验

1. 让用户用自己的话解释核心概念
2. AI 追问细节、边界条件、反例
3. 标记用户理解有偏差的地方
4. 针对偏差重新讲解

### Step 4: 练习巩固

1. 出 2-3 道由易到难的练习题
2. **不直接给答案**——先引导思路
3. 用户尝试后，给完整解答 + 易错点标注
4. 标记哪些题需要后续复习（间隔重复标记）

### Step 5: 记录与更新

1. 写入 `memory/{YYYY-MM-DD}.md`：学了什么、掌握程度、薄弱点
2. 更新课程 `project.md`：标记已学章节
3. 如果有踩坑 → 写入 `自我进化/做得差的避免/`

## 间隔重复计划

学习新章节后的复习间隔：

| 复习轮次 | 间隔 | 内容 |
|---------|------|------|
| 第1次 | 学完当天 | 快速回忆核心概念和公式 |
| 第2次 | +1天 | 做 2-3 道练习题（不看笔记） |
| 第3次 | +3天 | 费曼检验（口头解释） |
| 第4次 | +7天 | 综合练习（跨知识点） |
| 第5次 | +14天 | 与新章节关联复习 |

> 认知科学依据：间隔重复效应（009-认知天性 §概念3）——间隔越长，重新激活时的记忆巩固效果越强。

---

## 反模式（避免）

| 错误做法 | 正确做法 |
|---------|---------|
| 一上来就讲公式推导 | 先建立直觉，再推公式 |
| 直接给出作业答案 | 引导思路 → 提示 → 完整解答 |
| 一口气学完整章 | 分块学，每块用费曼检验 |
| 学完不复习 | 严格执行间隔重复计划 |
| 只看不练 | 每个概念至少做 1 道题 |
