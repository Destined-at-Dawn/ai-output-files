## 电路分析核心思维框架

### 两类约束（必须牢记）

```
电路分析 = 拓扑约束 + 元件约束

拓扑约束（KCL + KVL）：
  - KCL：任意节点，流入电流 = 流出电流
  - KVL：任意回路，电压升 = 电压降

元件约束（VCR）：
  - 电阻：u = iR（欧姆定律）
  - 电容：i = C du/dt
  - 电感：u = L di/dt
```

### 解题通用流程

```
Step 1: 标注参考方向（关联/非关联参考方向）
Step 2: 列 KCL/KVL 方程（拓扑约束）
Step 3: 代入 VCR（元件约束）
Step 4: 求解方程组
Step 5: 验证（功率守恒？边界条件？）
```

| 模式 | 典型错误 | 正确做法 |
