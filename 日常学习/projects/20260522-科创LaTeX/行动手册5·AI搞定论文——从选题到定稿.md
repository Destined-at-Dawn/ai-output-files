# 行动手册5·AI搞定论文——从选题到定稿

这节课不教你怎么「用AI代写论文」，教你一套**用AI加速论文全流程的方法论**。方法论不过时，工具会变，但思考方式不会变。



---



## **第一部分：道——论文这件事你要先想清楚的**



### **论文的本质**



论文的本质就三步：**找问题 → 用方法 → 得结论。**



「得结论」是关键——没有结论的论文不算完整研究。

你交上去一篇五千字的东西，如果只是「我看了很多资料然后总结了一下」，那不叫论文，叫读书笔记。

论文必须有一个**你自己的判断**：你发现了什么、你怎么证明的、你的结论是什么。



课程论文3000\-5000字，不是让你写一篇长作文，是让你走一遍「发现问题→解决问题→输出答案」的完整流程。

这个流程，考研复试要用、求职面试要用、创业找投资要用——区别只是问题不同，思维方式完全一样。



一句话记住：**论文训练的不是写作能力，是研究能力。课程论文就是最低成本的练手场。**



### **论文对你到底有没有用**



不是每个人都需要卷论文。先搞清楚你的目标，别在不该花时间的事上死磕：



|你的目标|论文重不重要|为什么|
|---|---|---|
|保研/考研/留学<br>|极重要|南开经管：一篇顶级期刊=25分活动分，一篇就占1/4|
|求职|有印象分|研发岗看重，产品经理/运营岗帮助不大|
|考公|不重要|校园活动、学生干部经历比科研有用得多|
|创业|理论有用，实际不大|除非做科技方向，否则训练的是思维不是商业能力|



如果你走保研/考研/留学路线——论文是硬通货，现在就该开始练。



如果你走其他路线——课程论文交个合格的就行，但这套方法论学了不亏。「找问题→用方法→得结论」这个思维方式到处用得上。



### **学术概念极简科普**



你可能听过这些词但不太懂，6句话讲清楚：



**期刊**：就是学术杂志。Nature、Science是顶级的，知网上的中文期刊是国内的

**SCI**：一个美国机构把它认可的期刊画了个圈，你的论文发在圈里的杂志上 = 发了SCI。蓝哥原话：「甚至你自己都可以画个圈说这些期刊是我认可的，叫做你名字期刊——只不过没人认可而已」

**影响因子**：评价的是期刊（杂志），不是你的论文。被引用次数越多，影响因子越高

**SCI分区**：一区前25%最牛，四区75%\-100%最水。但能发SCI四区已经很强了

**第一作者**：论文作者里最核心的那个，保研加分只认第一作者（或除导师外第一作者）。通讯作者默认是指导老师

**「发表了一篇SCI」**：就是论文发在了被SCI认可的杂志上。为什么不直接说期刊名？因为期刊太多没人认识，说SCI全世界都懂



课程论文不需要发表，但搞懂这些概念，等你大三大四真要发论文时不会懵。



### **AI在论文里的角色**



很多人以为AI在论文里的用法是：你给它一个题目，它帮你写完，你交上去。



不是这样的。



AI在论文里的正确角色是**思考搭档**——你们一起想选题、一起讨论文献、一起打磨论证逻辑。

观点不是AI单方面给你的，是你们聊出来的。但最终选哪个方向、用哪个观点、怎么收尾，你拍板。



> 💬 举个例子：你写市场营销课的论文，对「直播带货」有点感觉但没有具体方向。你跟ChatGPT/Claude说：「我对直播带货感兴趣，但不知道从什么角度写3000字课程论文。帮我发散一下，什么角度适合大二学生写？」
> 
> 
> 
> ChatGPT/Claude可能会给你8个方向。你扫一眼觉得「主播信任机制对复购率的影响」挺有意思。你接着跟它聊：「这个角度会不会太大？我只有3000字。」它帮你缩小范围。你再问：「知网上这个方向大概有多少论文？会不会写烂了？」
> 
> 
> 
> 

这个过程里，AI在帮你**想清楚**，不是在帮你**写完**。最终选哪个题、怎么切入，是你在对话中逐步形成判断的。



一句话记住：**观点是聊出来的，不是AI单方面给的。但最终决策者是你。**



### **学术诚信红线**



**🟢 能做（放心用）：**



- 用AI发散选题思路、评估选题可行性

- 用AI检索和梳理文献

- 用AI帮你理解看不懂的外文文献

- 用AI写代码处理数据、生成图表

- 用AI润色语法和学术表达

- 用AI翻译外文文献辅助理解

- 用AI模拟答辩提问

**🟡 灰色地带（看学校规定）：**



- 用AI生成初稿框架后大幅改写（多数学校允许，但有些要求声明）

- 提交时是否需要声明使用了AI工具（越来越多学校要求）



**🔴 不能做（碰了就是学术不端）：**



- 让AI直接写完整论文，不经改写就提交

- 用AI生成的虚假参考文献（AI会编造不存在的论文！）

- 用AI伪造实验数据或调查结果

- 学校要求声明AI使用情况，你不声明



### **AIGC检测的真相**



2025\-2026年，大部分高校已经上了AIGC检测系统。你需要知道三个事实：



**事实一：检测系统远没有你想的那么准。** 同一篇论文拿去不同平台检测——知网说AI率38%，维普说67%，万方说5%。差距高达62个百分点。你论文「合不合格」，取决于学校用的是哪个平台。



**事实二：检测系统会冤枉人。** 老舍的《林海》被茅茅虫检测系统判为99\.9% AI生成。朱自清的《荷塘月色》被判62\.88% AI疑似。有学生纯手写论文，2月检测AI率0\.84%，3月系统升级后一字没改飙到41\.3%。系统不是神。



**事实三：导师的眼睛比检测系统准。** 在真实翻车案例里，40%是教授肉眼发现的，不是机器抓的。教授看的是「你的写作风格有没有突然变了」「你能不能解释清楚你写的东西」——这些东西AI率数字反映不了。



**所以策略很清楚：不要赌检测系统的漏洞，要让自己真的理解自己写的每一句话。** 这比任何降AI率技巧都管用。



### **真实翻车案例**



**案例一：港大博士论文24篇虚假引用**

2025年，香港大学一篇博士论文被发现61篇参考文献中有24篇是AI虚构的——标题、作者、期刊名格式完美，但在任何数据库里都搜不到。结果：论文撤稿，副院长卸任。



**案例二：论文里出现ChatGPT按钮文字**

2023年，一篇物理学期刊论文中「Regenerate response」——ChatGPT网页版的按钮标签——直接出现在正文第三页。复制粘贴都没检查。结果：论文撤稿。



**案例三：明尼苏达大学博士被开除**

2025年，博士生在课程考试中被4名教授发现使用AI。证据不是AI率数字，而是：论文包含课堂未涉及的概念、使用领域内不常用的缩写、写作风格「没有个性」。结果：开除博士项目，失去学生签证。教训：**教授看的不是AI率，是「你的论文有没有你自己的声音」。**



**案例四：澳门科技大学毕业后撤销学位**

2024年，管理学博士毕业后，发表的期刊论文被查出抄袭。结果：撤销博士学位，毕业证作废。教训：**不是过了答辩就安全了，学位可以追溯撤销。**



**案例五：纯手写论文被误判（冤案型）**

2025年，法制史专业学生纯手写毕业论文，2月定稿AI率0\.84%。3月系统升级后一字没改，AI率飙到41\.3%——摘要345个字里343个被判AI生成。教训：如果你确实是自己写的，不要慌，跟导师沟通说明情况。



### **灰色产业链（你应该知道的真相）**



以下内容不是教你这么做，是让你知道这个世界长什么样：



**买论文**：SCI一区约10万元，四区约2\-3万元。花钱买第一作者，自己什么也不写

**校外专业机构全程辅导**：从选题到发表一条龙，根据需求报价不同

**委托外部实验机构做实验**：自己没有实验条件，花几千到上万找机构代做

**降AI率工具**：付费3\-6元/千字，可以把AI率从90%降到10%以下



这些东西存在，而且很多人在用。



**但你不需要走这条路。** 你有AI当思考搭档——别人花10万买论文，你用AI免费获得同等甚至更好的研究训练。别人花钱买的是一张纸，你练到的是真本事。



蓝哥有句话讲得很好：**和光同尘，而不是同流合污。** 你可以知道这些东西存在，但不需要参与。你有更好的路走。

## **第二部分：法——方法论框架**



### **核心逻辑**



论文是长链条任务。不能一口气丢给AI「帮我写篇论文」——和第3节讲的一样，复杂任务必须拆成小步骤，每步和AI协作思考。



**七步走全流程：**



```Plain Text
选题 → 文献调研 → 搭框架 → 写初稿 → 润色 → 自检 → 答辩准备
```



每一步的产出是下一步的输入。选题定了才能搜文献，文献读了才能搭框架，框架定了才能写初稿。**跳步 = 返工。**



### **工具矩阵**



开始之前先搞清楚——每个环节该打开哪个AI。



|环节|最优工具|一句话理由|
|---|---|---|
|选题头脑风暴|ChatGPT 5\.4thinking、Claude<br>|跨学科联想最强，能帮你从意想不到的角度切入|
|搜学术论文|秘塔AI搜索<br>|能搜知网万方，免费，结果带来源链接|
|读长文献<br>|ChatGPT、Claude、gemini、Kimi<br>|kimi200万字上下文，传PDF直接读，引用溯源跳到原文高亮|
|搭大纲\+写初稿\+润色|ChatGPT、Claude、gemini<br>|中文学术写作质量最高，长文档风格一致，保留你的语感|
|搜学校规定、国内政策|豆包专家模式|中文互联网信息覆盖最全|
|数据处理\+可视化|通义千问 / Claude Code/Gemini|直接跑Python代码画图|

|查重/AIGC检测<br>|学校指定平台<br>|知网/维普/万方，学校用哪个你就用哪个<br>|



一句话记住：**ChatGPT/Claude帮你发散想，Kimi帮你读文献，Claude帮你写论文，秘塔帮你搜论文。各管一摊，别一个AI从头干到尾。**

### **步骤一：选题——选对了搞定一半**



大部分人论文翻车不是写不好，是**题目选错了**。题目太大写不完，太小没东西写，太热门被人写烂了，太冷门找不到文献。选题选对了，后面事半功倍。



**怎么和AI一起选题**



1. 告诉ChatGPT/Claude你的专业、课程、字数要求，以及你大概感兴趣的方向（没有方向也行，说大概领域就行）

2. ChatGPT/Claude帮你发散出一批候选选题

3. 你和它一起讨论——挑2\-3个你觉得有感觉的，问它：这个题目会不会太大？能在3000字内说清楚吗？

4. 去秘塔搜一下你看中的选题——知网上同题论文超过100篇 = 太热门，换一个；不到5篇 = 太冷门，也换

5. 选定1\-2个候选，**带着你的选题分析去找老师确认**



> 💬 具体长什么样：你是大二市场营销专业的，要写「消费者行为」相关的课程论文，3000字。你跟ChatGPT/Claude说：
> 
> 
> 
> 「我是大二市场营销专业，要写消费者行为方向的课程论文，3000字。帮我想8个选题，要求范围小，别给我XX对XX的影响这种大而空的题目。」
> 
> 
> 
> ChatGPT/Claude给你8个。你挑了「小红书种草内容对大学生冲动消费的影响机制」，但你不确定范围够不够小。你接着问：「这个题目3000字写得完吗？如果太大了帮我缩小一下。」
> 
> 
> 
> ChatGPT/Claude建议你改成「以美妆品类为例」。你觉得可以，去秘塔搜了一下——知网上有三四十篇相关论文，不多不少，刚好。
> 
> 
> 
> 你把这个选题和AI帮你做的分析截图发给老师确认。老师说可以，选题就定了。
> 
> 



**启动方式：** 复制下面的提示词发给ChatGPT/Claude，它会先问你几个问题，然后帮你发散选题。不需要你提前填任何东西。



```Plain Text
# 角色
你是一个大学课程论文的选题顾问。你擅长从学生模糊的兴趣方向中，找到范围合适、文献充足、老师大概率会接受的论文选题。

# 任务
通过提问了解用户情况，然后生成一批候选选题供挑选。目标是：帮用户从「我大概想写XX方向」变成「我要写这个具体题目」，然后带着选题分析去找老师确认。

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你是什么专业、几年级？这是哪门课的论文？」
   示例：「大二市场营销专业，消费者行为课程论文」

2.「论文字数要求多少？老师有没有给具体要求或限制？」
   示例：「3000-5000字，老师要求写消费者行为相关的」
   没有具体要求就说「没有」。

3.「你对什么方向感兴趣？哪怕只是一个模糊的感觉也行。」
   示例：「对直播带货/新消费品牌/大学生消费习惯比较感兴趣」
   完全没想法也没关系，说一下你最近关注什么就行。

4.「你有没有看过同学的论文、或者老师给过的选题参考？」
   有的话粘过来，我帮你避开撞车的方向，同时从中找灵感。
   没有也没问题。

# 第二步：拆解参考信息（用户提供了才执行，没提供直接跳到第三步）
1. 分析老师给的选题范围或参考列表，找出哪些方向空间大、哪些已经被同学占了
2. 如果有学长学姐的论文，拆解他们的选题逻辑——什么角度、什么粒度、为什么老师认可
3. 总结可借鉴的选题模式，作为下一步生成选题的参考

# 第三步：生成选题（按顺序执行）
1. 根据用户的专业、课程和兴趣方向，生成8个候选选题
2. 每个选题包含：完整论文标题、一句话研究问题、难度评估、文献量预判
3. 8个选题要有区分度——不要8个都是同一个角度的变体
4. 标出你最推荐的2个，说明为什么

# 输出格式示例
以下是一个选题的写法示例，你的每个选题都要达到这个质量：

选题3：小红书种草评论区从众效应对大学生美妆冲动消费的影响
研究什么：评论区的「跟风」氛围是否会让大学生更容易冲动下单
难度：中等偏易。概念清晰，问卷设计不复杂，3000字能说清楚
文献量：知网上「小红书+冲动消费」约30-40篇，但从评论区角度切入的不到5篇——有空间
⭐ 推荐理由：切口小、有新意、文献够用但不烂大街

# 约束
题目范围必须小。「XX对XX的影响」太大了，「XX背景下XX对XX的影响——以XX为例」才是课程论文的正确粒度
不要推荐需要做实验或获取非公开数据的选题
8个选题里，至少3个是问卷调查能做的，至少2个是纯论证/案例分析能做的
不要推荐知网上已有超过200篇同题论文的选题
每个选题必须能在用户要求的字数内说清楚
```



**这一步做完你手上有什么：** 1个确认的论文题目 \+ 1份选题分析（为什么选这个、研究什么问题、文献量预估）。



**怎么接上下一步：** 选题分析保存下来。下一步文献调研时，直接把选题分析粘给AI：「这是我的选题和研究方向，帮我整理以下文献……」——AI立刻知道你要什么。



**⚠️ 防坑**



- 不要用AI给你的第一个选题，那通常是最平庸的

- 题目范围要小：「XX对XX的影响」太大，「XX背景下XX对XX的影响——以XX为例」才是课程论文的正确粒度



**不要自己闷头选，去找老师聊。** 老师见过几百篇论文，他一句话能帮你避开很多坑



一句话记住：**好选题的标准——一个具体的问题 \+ 一个可操作的方法 \+ 能找到的文献。三个条件缺一个就换。**



**补充：毕业论文还有一个「开题报告」环节**



课程论文选完题直接开始写就行。

但如果是毕业论文，选题确认后还要写一个1500\-3000字的开题报告——说明你**打算研究什么、为什么研究、怎么研究、时间安排。**

很多人把开题报告当走过场，结果后面写论文时发现方向不对，要推翻重来。课程论文现在不需要，但提前知道有这个东西，大四不慌。

### **步骤二：文献调研——搞清楚别人做了什么**



论文的核心价值是「在别人研究的基础上往前走一小步」——微创新。你不知道别人走到哪了，怎么往前走？



这一步有个铁律：**文献必须你自己搜****，AI只负责帮你读和整理。** ——必须自己去验证。去点开那些链接，看是否属实，是否能点。

为什么？因为AI会编文献。你让AI「帮我找几篇论文」，它大概率编造——标题、作者、期刊名格式完美，但论文根本不存在。港大那个24篇虚假引用的案例就是这么翻车的。

1\. **你自己搜文献**：打开秘塔，搜你选题方向的关键词。目标：找到8\-15篇相关论文，下载PDF或复制摘要

### **步骤四：写初稿——一章一章和AI协作**



到这一步，你有了选题、文献、大纲。现在要把骨架填成血肉。



写初稿的核心心法：**先完成再完美。** 先把字数写够、结构填满，再去打磨质量。

完美主义是论文最大的敌人——一段话反复改三天不往下写，最后全文只完成了20%。

先粗糙地把每一章都写出来，后面有专门的润色步骤帮你打磨。



**为什么不能一键生成全文？** 你丢给AI一句「帮我写一篇3000字论文」，它确实能写出来。但你会遇到三个问题：①答辩时老师问你任何一段的逻辑你都说不清楚 ②全文AI味极重，风格统一得不像人写的 ③AIGC检测大概率会标红。



正确做法是：**逐章节协作，一章写完确认了再写下一章。** 每写一章就发一次下面的提示词。



**每一章的协作流程：**



1. 你先和Claude讨论：这一章要回答什么问题？你的核心观点是什么？有什么证据？

2. Claude帮你把讨论中形成的想法组织成学术语言，生成这一章的初稿

3. 你读完初稿——每句话你都能解释吗？有没有看不懂的地方？看不懂就问Claude「这句话什么意思」，直到你真的理解

4. 改掉你觉得不对的、不像你说话方式的地方

5. 确认没问题了，进入下一章

> 💬 你写到第四章「数据分析」。你在问卷星上收了150份问卷，用Excel初步整理了一下数据。你跟Claude说：
> 
> 
>

>

```Plain Text
3.1 研究模型与假设
基于文献综述中的发现，本研究构建了「评论区从众感知→冲动消费倾向」的基本研究模型。从众效应理论认为，个体在面对不确定决策时，倾向于参考他人行为作为判断依据（Cialdini, 2001）。在小红书种草场景中，评论区中「已下单」「回购了」等表述可能构成一种社会证明信号，降低消费者的决策不确定性，从而促进冲动消费。据此，本文提出以下假设：
H1：评论区从众感知对大学生美妆冲动消费倾向有显著正向影响。
```

**颗粒度越细AI输出越好。** 不要「帮我写第三章」，要「帮我描述表3的数据趋势，然后解释为什么评论数量和冲动消费正相关」

**数据分析不要让AI跑数据。** 你自己在SPSS/Excel里跑完，把结果表格给AI帮你描述和解读

**去AI味的根本方法是往里填真东西**：你自己的数据、你做问卷时发现的细节、老师给的建议、具体的案例——这些是AI写不出来的，也是让你的论文有「人味」的关键

### **步骤五：润色——消灭AI「指纹」**



初稿出来后，内容可能是对的，但表达不够「学术」，而且可能有AI留下的指纹。润色就是：内容不变，表达升级，AI痕迹清除。



**怎么做：**



1. 把每章内容分别给Claude，让它做学术语言润色——口语改学术、统一人称、检查逻辑

2. **要求Claude标注它改了哪些地方**，你逐条确认。改得好留下，改歪了改回来

3. 全文搜索以下高频AI词汇，出现超过3次的替换掉：

> 综上所述、由此可见、毋庸置疑、不可否认、至关重要、日益增长、具有重要意义、本质上、值得注意的是、首先……其次……最后……
> 
> 



这些词不是不能用，但满篇都是 = 一看就是AI写的。



**启动方式：** 一章一章粘给Claude，不要一次粘全文。



```Plain Text
# 角色
你是一个学术语言编辑。你擅长在不改变作者原意的前提下，把口语化的表达升级为学术表达，同时消除AI生成痕迹。

# 任务
润色用户提供的论文章节。只改表达，不改内容和观点。

# 第一步：信息收集
1.「把需要润色的章节粘过来。」
2.「这是第几章？前后章节需不需要风格对齐？」
   如果你之前已经润色过其他章节，告诉我，我会保持风格一致。

# 第二步：润色（按顺序执行）
1. 口语→学术：「很多人觉得」→「已有研究表明」、「越来越多」→「呈增长趋势」
2. 统一人称：全文用「本文」「本研究」，不要出现「我」「我们」
3. 检查逻辑：有没有前后矛盾或跳跃的地方
4. 消灭AI指纹：删除「综上所述」「由此可见」「毋庸置疑」「值得注意的是」「首先……其次……最后」
5. 标注修改：每处修改用删除线+新文本标注，让用户能对比

# 输出格式示例
原文：很多大学生觉得小红书上的评论区会影响自己买东西的决定，首先是因为看到别人说好用会放心，其次是因为大家都买了自己不买好像亏了。
润色后：研究显示，小红书评论区对大学生消费决策存在显著影响。一方面，其他用户的正面评价降低了消费者的感知风险；另一方面，「已购买」等从众信号可能引发错失焦虑，进而促进冲动消费。
修改说明：①「很多大学生觉得」→「研究显示」②删除「首先……其次」排列 ③「亏了」→「错失焦虑」

# 约束
不要改变作者的原意。表达模糊的标注「⚠️ 此处原意不明确，请确认」而不是自己猜着改
不要加入新的内容、新的论点、新的数据
不要过度润色。课程论文用词平实就行，不需要堆砌高级术语
润色后的文本不应该比原文长超过20%
```



**⚠️ 防坑**



- 润色 ≠ 重写。如果AI把你的意思改歪了，以你的原意为准

- 不要过度润色。课程论文用词平实就行，堆砌术语显得「高深」反而扣分

### **步骤七：答辩准备——你说得清楚才算你的**



AIGC检测过了、查重过了，但答辩时老师问你「这段什么意思」你说不清楚——照样翻车。

答辩的本质是验证：这篇论文是不是你自己做的。



**答辩现场真实情况：**



大部分评委不会仔细读你的全文。他们翻一遍，然后问你几个问题。

最常问的：为什么选这个题？你用了什么方法？核心结论是什么？有什么局限性？



最容易暴露的：你引用了某个理论但解释不清楚、你的数据分析结论你说不出怎么得到的。



**怎么和AI一起准备：**



1. 让ChatGPT/Claude扮演答辩评委，从5个角度向你提问：选题动机、方法论选择、数据可靠性、结论局限性、一个刁钻追问

2. 你先自己答一遍。ChatGPT/Claude给你反馈：回答得怎么样、哪里可以更好

3\. **重点练你最薄弱的部分。** 如果某段是AI帮你写的，你要确保自己能用自己的话解释那段的逻辑



**启动方式：** 论文定稿后，把摘要和核心章节粘给ChatGPT/Claude。



```Plain Text
# 角色
你是一个大学课程论文的答辩评委。你教了十几年这门课，见过上千篇学生论文，对什么是好论文、什么是凑字数的一眼就能看出来。提问风格：温和但精准——不故意刁难，但会戳到学生最心虚的地方。

# 任务
通过提问了解用户的论文内容，然后以评委身份进行模拟答辩。目标是：帮用户在正式答辩前把所有可能被问到的问题练一遍。

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你的论文题目是什么？把摘要粘过来。」

2.「把论文的核心章节粘过来（至少包含研究方法和主要结论）。」
   越完整我问得越精准。论文不长的话粘全文也行。

3.「答辩有时间限制吗？」
   不知道就说「不确定」。

4.「你自己觉得论文里最薄弱的部分是哪里？」
   知道你心虚什么，我重点问那里。

# 第二步：模拟答辩（按顺序执行）

依次从以下5个角度提问，每个问题后面附参考回答思路：
1. 选题动机：为什么选这个题，不选别的
2. 方法论选择：为什么用这个方法，有没有考虑过其他方法
3. 数据/论据可靠性：你的证据够不够硬，样本量够不够
4. 结论的局限性：你的结论有没有漏洞，有没有过度推广
5. 一个针对你论文最薄弱环节的追问

每个问题的格式：
评委提问：（像真人评委一样提问，不要太学术）
参考回答：（给一个回答思路，不要写成背诵稿）
如果答不上来怎么办：（给一个体面的兜底说法）

# 输出格式示例
问题3：数据可靠性
评委提问：「你的问卷只收了150份，而且全是你自己学校的学生。这个样本能代表大学生这个群体吗？」
参考回答：「150份样本在课程论文层面是可接受的样本量，能够满足基本的统计分析需求。但您指出的问题确实是本研究的主要局限性之一——样本来源单一，结论的外部效度有限。」
答不上来怎么办：坦诚说「这是本研究的局限性，样本的代表性确实不够，这个问题在结论章节已经提到了。」——评委知道你知道自己的不足，反而会加分。

# 约束
不要当好人。评委不会因为你是学生就不问难题
提问要具体，针对论文里的具体内容问，不要泛泛地问「你的创新点是什么」
参考回答是思路，不是背诵模板
如果论文有明显硬伤（逻辑断裂、数据可疑），直接指出来
```



> 💬 你跟ChatGPT/Claude说：「我的论文题目是小红书种草评论区从众效应对大学生美妆冲动消费的影响，摘要如下（粘贴）。请扮演答辩评委向我提问。」
> 
> 
> 
> ChatGPT/Claude问你：「你的问卷只收了150份，样本量够吗？如果扩大到500份，结果会不会不一样？」
> 
> 
> 
> 你发现你确实没想过这个问题。你跟ChatGPT/Claude讨论：「150份够不够？怎么回答这个质疑？」它帮你想了一个回答思路：「150份在课程论文层面是可接受的样本量，但确实是本研究的局限性之一。未来可以扩大样本……」
> 
> 
> 
> 这样答辩时真的被问到，你就不慌了。
> 
> 



**⚠️ 防坑**



答辩不是背稿。老师问的角度可能和你准备的完全不同。你需要的是**真的理解你写的东西**，而不是背一套说辞

### **和导师打交道**



这部分不是教你写论文，是教你**怎么和论文的「甲方」沟通**。很多人论文写得还行，但不懂和老师打交道，踩了不该踩的坑。



**先打一剂预防针：初稿被打回来是正常的。**

90%的论文初稿都会被老师要求修改，一般2\-5轮。

这不是你写得烂，是流程的一部分。

不要等老师催才改——主动推进、每次改完主动发给老师，老师喜欢自驱的学生。



#### **长远视角：找老师做科研项目（大二以后再考虑）**



如果你未来想发论文、保研，早晚要找一位老师带你。以下是蓝哥总结的核心策略：



**找什么级别的老师——能找院长就找院长。**



指导老师的层级极其重要——因为评委就是学院老师，不可能让讲师的项目评分超过院长的。



> 真实案例：有人参加大创做涵道式无人机，准备近一年，是30多个项目中唯一做出实物并成功飞行的。但指导老师只是讲师，仅获校一等奖。室友的项目连实物都没做出来，但指导老师是院长，直接拿到国家二等奖。
> 
> 



**大牛老师为什么愿意带你——因为你免费。**



老师非常需要劳动力。研究生还要发800\-1500/月的补贴，本科生是零成本。只要你态度好、能干活，老师没理由拒绝。

你跟老师说想读他的研究生，相当于在说：「老师，我想长期给你当廉价劳动力。」他凭什么拒绝？



而且大牛老师不屑于抢你的论文一作——人家要冲Nature、Science，你发的小论文在他眼里九牛一毛。反而是你更容易拿到第一作者。



**什么时候联系——大二上学期开学是最佳节点。**



大一全年成绩已出，能证明你优秀。前提是成绩好：「成绩好就是你自律努力的身份标签，老师就愿意信任你。」



**⚠️ 注意**



- 一旦进入老师课题组就很难脱身，暑假想出去实习会很复杂

- 有的老师要求必须考本校研究生，否则「人才两空不划算」——这是PUA，遇到了可以换老师



#### **眼前视角：课程论文怎么和任课老师沟通**



你现在大一大二，面对的是任课老师。以下三个场景的处理方式：



**场景一：选题阶段找老师确认**



> 「老师好，我是XX专业的XXX。课程论文我想写XX方向，之前做了一些初步的选题分析和文献检索（把你的分析展示给老师看）。想请教一下这个方向可行吗？有没有需要调整的地方？」
> 
> 



**关键：带着东西去，不要空手去。** 你带着AI帮你做的选题分析去找老师，老师会觉得你有准备；你什么都没做就去问「老师我写什么好」，老师大概率敷衍你。



**场景二：论文被打回来怎么处理**



不要慌。先仔细看老师的批注，搞清楚**具体哪里不行**——是选题不行？论证不够？还是格式问题？然后把老师的修改意见原文粘给Claude，和它一起讨论怎么改。改完后再找老师确认：



> 「老师，上次您提的XX问题我改了，主要从XX角度做了调整，您看看这个方向对不对？」
> 
> 



**场景三：答辩前和导师对齐**



> 「老师，答辩我准备了这几个方面的回答（列出你准备的要点）。您觉得评委可能还会从什么角度提问吗？」
> 
> 



老师见过无数次答辩，他一句话可能帮你避开一个致命问题。



一句话记住：**导师不是敌人，是资源。带着准备去沟通，大部分老师都愿意帮你。**

## **第三部分：术器——提示词**

### **提示词1：选题发散（发给ChatGPT/Claude）**



对应七步走的第一步「选题」。你对一个方向有感觉但还没有具体题目时，用这个提示词。ChatGPT/Claude帮你发散出一批候选题，你从里面挑，然后去秘塔搜一下文献量验证可行性，最后带着选题分析去找老师确认。



**启动方式：** 复制下面的提示词发给ChatGPT/Claude，它会先问你几个问题，然后帮你发散选题。



```Plain Text
# 角色
你是一个大学课程论文的选题顾问。你擅长从学生模糊的兴趣方向中，找到范围合适、文献充足、老师大概率会接受的论文选题。

# 任务
通过提问了解用户情况，然后生成一批候选选题供挑选。目标是：帮用户从「我大概想写XX方向」变成「我要写这个具体题目」。

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你是什么专业、几年级？这是哪门课的论文？」

   没有具体要求就说「没有」。

3.「你对什么方向感兴趣？哪怕只是一个模糊的感觉也行。」
   示例：「对直播带货/新消费品牌/大学生消费习惯比较感兴趣」

4.「你有没有看过同学的论文、或者老师给过的选题参考？」

   有的话粘过来，我帮你避开撞车的方向，同时从中找灵感。
   没有也没问题。

# 第二步：拆解参考信息（用户提供了才执行，没提供直接跳到第三步）
1. 分析老师给的选题范围或参考列表，找出哪些方向空间大、哪些已经被同学占了
2. 如果有学长学姐的论文，拆解他们的选题逻辑——什么角度、什么粒度、为什么老师认可
3. 总结可借鉴的选题模式，作为下一步生成选题的参考

# 第三步：生成选题（按顺序执行）
1. 根据用户的专业、课程和兴趣方向，生成8个候选选题
```

### **提示词3：研究空间分析（发给Claude）**

```Plain Text

1. 梳理现有研究的主要方向——大家都在研究什么
2. 找出至少3个研究空白——哪些角度还没有人研究过，或研究得不够深
3. 评估每个空白的可行性——作为课程论文来写，难度怎么样、数据能不能拿到

# 输出格式示例

数据获取：问卷星发放，150份以上即可
→ 推荐作为你的切入角度。

# 约束
只基于用户提供的文献分析，不要引入用户没有的文献。你说「已有研究发现XX」，这个XX必须来自用户给你的那些文献

不要推荐听起来很酷但学生没法获取数据的角度（如「可以做眼动追踪实验」）

```

### **提示词4：论文大纲搭建（发给Claude）**

**启动方式：** 每写一章就发一次这个提示词。一次只写一章，写完确认了再写下一章。



```Plain Text

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你现在要写哪一章？把你的论文大纲粘过来。」
   我需要看到完整大纲才能保证这一章和前后文逻辑对得上。

2.「前面的章节写完了吗？写完的粘过来。」
   如果这是第一章就说「这是第一章」。
   有前文的话粘过来，我要确保这一章能自然承接上一章。

3.「这一章你有没有想说的核心观点或者可用的素材？」
   比如：你对这个问题的看法、你调研时发现的数据、文献笔记里某篇论文的观点、老师给过的建议。

   比如：知网上搜到的同类论文，这一章写得特别好，你想参考它的写法。

1. 拆解参考章节的论证结构：怎么开头、怎么推进、怎么收尾
2. 分析表达技巧：数据怎么呈现的、论点怎么展开的、段落怎么衔接的

3. 如果用户说「没想好」，通过2-3个问题引导用户形成观点

1. 回顾大纲中本章的要点和字数要求
2. 检查前序章节的结论，确保本章开头自然承接
3. 按要点逐个展开，每个要点写成完整段落
4. 写完后自查：
   核心问题是否被清晰回答了？

   是否在大纲建议的字数范围内（允许±20%）？

严格遵守大纲建议的字数，允许±20%浮动，不要自行扩展到远超大纲的篇幅
```

```Plain Text
# 角色
你是一个学术语言编辑。你擅长在不改变作者原意的前提下，把口语化的表达升级为学术表达，同时消除AI生成痕迹。

# 任务
润色用户提供的论文章节。只改表达，不改内容和观点。

# 第一步：信息收集

1.「把需要润色的章节粘过来。」

2.「这是第几章？前后章节需不需要风格对齐？」
   如果你之前已经润色过其他章节，告诉我，我会保持风格一致。

# 第二步：润色（按顺序执行）
1. 口语→学术：把口语化的表达改成学术表达
   「很多人觉得」→「已有研究表明」
   「越来越多」→「呈增长趋势」
   「说明了」→「表明」
2. 统一人称：全文用「本文」「本研究」，不要出现「我」「我们」
3. 检查逻辑：有没有前后矛盾或跳跃的地方
4. 消灭AI指纹：删除所有「综上所述」「由此可见」「毋庸置疑」「值得注意的是」「首先……其次……最后」
5. 标注修改：每处修改用删除线+新文本标注，让用户能对比

# 输出格式示例
以下是润色前后对比的示例：

原文：很多大学生觉得小红书上的评论区会影响自己买东西的决定，首先是因为看到别人说好用会放心，其次是因为大家都买了自己不买好像亏了。
润色后：研究显示，小红书评论区对大学生消费决策存在显著影响。一方面，其他用户的正面评价降低了消费者的感知风险；另一方面，「已购买」等从众信号可能引发错失焦虑，进而促进冲动消费。
修改说明：①「很多大学生觉得」→「研究显示」（口语→学术）②删除「首先……其次」排列结构 ③「亏了」→「错失焦虑」（概念化）

# 约束
不要改变作者的原意。如果某处表达模糊，标注「⚠️ 此处原意不明确，请确认你想表达的是……」而不是自己猜着改
不要加入新的内容、新的论点、新的数据
不要过度润色。课程论文用词平实就行，不需要堆砌高级术语
润色后的文本不应该比原文长超过20%
```

### **提示词8：模拟答辩（发给ChatGPT/Claude）**



对应七步走的第七步「答辩准备」。论文定稿、自检通过后，用这个提示词让ChatGPT/Claude扮演评委提问你。重点练你最心虚的部分——如果某段是AI帮你写的，你要确保自己能用自己的话讲清楚。



**启动方式：** 论文定稿后，把摘要和全文核心部分粘给ChatGPT/Claude，用这个提示词练答辩。



```Plain Text
# 角色
你是一个大学课程论文的答辩评委。你教了十几年这门课，见过上千篇学生论文，对什么是好论文、什么是凑字数的一眼就能看出来。你的提问风格是温和但精准——不会故意刁难，但会戳到学生最心虚的地方。

# 任务
通过提问了解用户的论文内容，然后以评委身份进行模拟答辩。目标是：帮用户在正式答辩前把所有可能被问到的问题练一遍。

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你的论文题目是什么？把摘要粘过来。」

2.「把你论文的核心章节粘过来（至少包含研究方法和主要结论）。」
   越完整我问得越精准。如果论文不长，粘全文也行。

3.「答辩有时间限制吗？」
   比如「5分钟陈述+5分钟提问」。
   不知道或者还没确定就说「不确定」。

4.「你自己觉得论文里最薄弱的部分是哪里？」
   知道你心虚什么，我重点问那里。

# 第二步：模拟答辩（按顺序执行）

依次从以下5个角度提问，每个问题后面附一个参考回答思路：

1. 选题动机：为什么选这个题，不选别的
2. 方法论选择：为什么用这个方法，有没有考虑过其他方法
3. 数据/论据可靠性：你的证据够不够硬，样本量够不够
4. 结论的局限性：你的结论有没有漏洞，有没有过度推广
5. 一个针对你论文最薄弱环节的追问

每个问题的格式：
评委提问：（像真人评委一样提问，不要太学术）
参考回答：（给一个回答思路，不要写成背诵稿）
如果答不上来怎么办：（给一个体面的兜底说法）

# 输出格式示例
以下是一个提问的示例：

问题3：数据可靠性
评委提问：「你的问卷只收了150份，而且全是你自己学校的学生。这个样本能代表大学生这个群体吗？如果换一个学校、换一个城市，结论还成立吗？」
参考回答：「150份样本在课程论文层面是可接受的样本量，能够满足基本的统计分析需求。但您指出的问题确实是本研究的主要局限性之一——样本来源单一，结论的外部效度有限。如果扩大样本范围，结果可能会有差异，这也是未来研究可以改进的方向。」
答不上来怎么办：坦诚说「这是本研究的局限性，样本的代表性确实不够，这个问题在结论章节已经提到了。」——评委知道你知道自己的不足，反而会加分。

# 约束
不要当好人。评委不会因为你是学生就不问难题
提问要具体，不要泛泛地问「你的创新点是什么」——要针对论文里的具体内容问
参考回答是思路，不是背诵模板。真正的答辩需要用自己的话说
如果论文质量明显很差（逻辑硬伤、数据编造嫌疑），直接指出来，不要假装没看到
```
