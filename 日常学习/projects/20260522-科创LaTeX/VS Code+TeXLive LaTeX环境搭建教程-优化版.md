# VS Code + TeXLive：LaTeX 本地环境搭建完整教程

> 适用：Windows | 含 AI 编辑、Codex 适配、一键换刊模板 | 更新时间：2026-05-22

---

## 为什么选 VS Code + TeXLive

| 对比维度 | 线上环境（Overleaf 等） | 本方案（VS Code + TeXLive） |
|---------|----------------------|--------------------------|
| 网络依赖 | 必须联网 | 完全离线可用 |
| 编译速度 | 受服务器负载影响 | 本地编译，速度快 |
| AI 集成 | 有限 | VS Code 内置 AI / Codex 无缝接入 |
| 模板切换 | 手动替换文件 | 一条命令更换期刊模板 |
| 界面 | 浏览器操作 | 专业 IDE 体验（多级目录/符号面板/左右对照） |

核心功能一览：
- **正向/反向定位**：代码 ↔ PDF 一键跳转
- **多级目录 + 符号面板**：LaTeX 命令可视化，点击即插入
- **AI 辅助**：报错自动修复、内容修改通过自然语言指令完成
- **一键换刊模板**：拒稿后切换期刊模板只需一条命令

## 第一步：安装 TeXLive

TeXLive 是 LaTeX 的本地编译环境（类比 Python 之于 .py 文件）。

### 1.1 下载

推荐使用**清华大学镜像站**，速度更快：

https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/

下载 `texlive.iso` 文件。

> 如果下载后看不到 `.iso` 扩展名：文件资源管理器 → 查看 → 勾选「文件扩展名」

### 1.2 安装

1. 双击 `.iso` 文件挂载（Win10/11 原生支持，无需虚拟光驱）
2. 右键 `install-tl-windows.bat` → **以管理员身份运行**
3. 等待命令行窗口弹出安装界面（约 1-2 分钟）

### 1.3 关键配置

**① 修改安装路径**（建议改到 D 盘，C 盘空间紧张时必做）：
- 只改盘符（如 `C` → `D`），**后面的路径不要动**
- ⛔ **整个路径中严禁出现中文**，否则必报错

**② 取消「安装 TeXworks 前端」**：我们会用 VS Code 替代，不需要 TeXworks。

**③ 语言包选择**（点击「Advanced」→「定制」）：
- 全选所有语言 → 体积大，安装慢
- **推荐**：只选中文、中日韩（CJK）、英文（US/UK）这三种，日常完全够用

**④ 点击「安装」**，等待 30 分钟 ~ 1 小时（取决于网络和机器性能）。

看到「欢迎进入 TeXLive 的世界」即安装成功。

## 第二步：安装 VS Code

1. 官网下载：https://code.visualstudio.com/
2. 双击安装包，全程默认即可
3. 安装路径建议同样只改盘符不改后续路径
4. 勾选「创建桌面快捷方式」

---

## 第三步：配置 VS Code

### 3.1 安装中文语言包

1. 点击左侧扩展图标（四个方块）
2. 搜索 `Chinese`
3. 安装「Chinese (Simplified) Language Pack」
4. 右下角弹出提示时点击「Change Language and Restart」

### 3.2 安装 LaTeX Workshop 插件

1. 扩展商店搜索 `LaTeX Workshop`
2. 找到作者为 James Yu 的插件（**只装这个，不要装其他 LaTeX 相关插件**）
3. 点击安装

---

## 第四步：基本使用

### 4.1 编译

- 快捷键：`Ctrl + S`（保存即编译）
- 左下角状态栏出现转圈 → 编译中
- 显示 ✓ → 编译成功，点击右侧图标查看 PDF

### 4.2 正向定位（代码 → PDF）

光标放在代码中任意位置，按 `Ctrl + Alt + J`，右侧 PDF 会高亮对应位置。

> 说明：`J` 键指键盘上 `H` 和 `K` 之间的键。

### 4.3 反向定位（PDF → 代码）

在 PDF 中 `Ctrl + 左键` 点击任意位置，左侧代码会跳转到对应行。

### 4.4 多级目录 & 符号面板

- 左侧 `TEX` 面板：显示文档结构大纲
- 数学符号面板：点击即可插入常用 LaTeX 命令

## 第五步：AI 辅助（进阶）

### 5.1 VS Code 内置 AI

右侧 Copilot/Chat 面板，直接输入自然语言指令，例如：
- 「帮我把作者改成四月学长」
- 「把这张表的列宽加大」
- 「修复所有编译报错」

AI 会直接修改 `.tex` 源文件并自动编译验证。

### 5.2 适配 Codex

Codex 环境可直接挂载工作区，用自然语言完成 LaTeX 编辑、调试、编译全流程——报错自修复、格式自调整。

## 常见问题

| 问题 | 解决 |
|------|------|
| 下载的 ISO 没有显示扩展名 | 文件资源管理器 → 查看 → 勾选「文件扩展名」 |
| 安装路径含中文导致报错 | 卸载重装，确保路径全英文 |
| 编译报错不通过 | 检查 LaTeX Workshop 是否正确安装；尝试 AI 自动修复 |
| 安装过程中其他报错 | 联系四月学长协助排查 |
