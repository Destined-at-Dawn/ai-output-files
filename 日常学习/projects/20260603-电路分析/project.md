| 题型 | 解题模式 | 适用章节 |
|------|---------|---------|
| 电阻电路求解 | 串并联等效 → KCL/KVL → 解方程 | 基础直流 |
| 戴维南等效 | 断开负载 → 求Voc → 求Req → 等效电路 | 电路定理 |
| 暂态响应 | 初始条件 → 稳态值 → 时间常数 → 三要素法 | 一阶电路 |
| 正弦稳态 | 时域→相量 → 阻抗 → 解代数方程 → 相量→时域 | 正弦分析 |
| 功率计算 | 求U和I相量 → S=UI* → P/Q/cosφ | 功率分析 |
