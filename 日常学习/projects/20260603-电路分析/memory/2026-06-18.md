## 高要求复习 PDF 生成路线纠偏

- **问题**：此前口径仍保留“Markdown -> HTML -> PDF”的默认渲染路线，不符合电路分析高要求复习资料的版式质量需求。
- **决策**：复习 PDF 成品链路改为“source-first exam mode -> image2 逐页图片生成 -> 用户/目检审查 -> LaTeX 合成 PDF -> PDF 渲染抽检”。Markdown 只作为内容草稿或审稿稿，不作为最终 PDF 默认成品路径。
- **原因**：电路复习资料依赖复杂公式、表格、红色重点、紫色分块和逐页视觉控制；逐页图片能固定版式，LaTeX 适合稳定合成和分页控制。

## 复习 PDF 证据展示优先级纠偏

- **问题**：此前强调 `Source: 文件路径#行号`，但用户实际不一定会看行号；对高要求复习 PDF，更重要的是让用户直接看到原资料对应部分。
- **决策**：重要判断、题型、公式、老师重点和易错点，优先截取原资料 PDF/图片对应区域；最终逐页图片中应嵌入原资料截图或 AI 融合后的原资料局部。`Source` 保留为辅助追溯，不作为主要展示方式。
- **边界**：AI 融合只能裁切、放大、降噪、标注和合成版面，禁止改变原题条件、数值、图形关系、红线标记和老师重点含义。
