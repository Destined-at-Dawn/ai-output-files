# 电路分析项目 Agent 执行手册

## 结论

本项目默认采用“期末原题复现型复习系统”：资料索引优先，原题数据库优先，题型模板优先，最后才生成概念笔记、复习 PDF、A4 速查和模拟卷。高要求复习 PDF 必须采用“image2 逐页图片生成 -> LaTeX 合成 PDF”的路线，不以 Markdown -> HTML -> PDF 作为默认成品路径；关键判断优先提供原资料 PDF/图片对应区域截图，`Source: path#line` 只作为辅助追溯信息。

Source: shared\CODEX_电路期末资料整理指令.md#L164

## 一、启动顺序

1. Read `CLAUDE.md`，加载考试范围、资料优先级和禁止事项。
2. Read `shared\CODEX_电路期末资料整理指令.md` 和 `shared\codex_circuit_exam_task.md`，获取当前期末复习任务细则。
3. 扫描项目文件，生成或更新资料索引。
4. 若用户要求“复习PDF / 1小时吃透版 / 期末冲刺包”，调用 `study-review-pdf`，且必须走 source-first exam mode。
5. 每次完成重要复习产物后，写入 `memory/YYYY-MM-DD.md`。

---

## 二、资料优先级

1. 最终复习题、老师划重点、考试说明。
2. PPT。
3. 习题答案。
4. 习题。
5. 教材例题和章末题。
6. 教材正文。
7. AI 补充解释。

冲突时优先考试侧材料，并保留冲突说明；不要替资料“强行统一”。

## 四、章节策略

- 第 1 章：功率吸收/发出，参考方向先判定。
- 第 3 章：节点电压法、网孔/回路电流法。
- 第 4 章：叠加、戴维南、诺顿、最大功率传输。
- 第 7 章：一阶三要素法，只到 7-4。
- 第 9 章：相量法、正弦稳态功率。
- 第 10 章：理想变压器、空心变压器/互感；10-3 不考。
- 第 11 章：串联 RLC 谐振，11-2、11-3。
- 第 13 章：非正弦周期电路，有效值、平均功率、分谐波计算，只到 13-4。
- 第 16 章：二端口基础、Z/Y 参数，只到 16-2，按选择题处理。

## 六、禁止事项

- 不要先写长篇概念笔记。
- 不要把生成稿当事实来源。
- 不要把 Markdown -> HTML -> PDF 当作高要求复习 PDF 的默认成品链路。
- 不要只给 Source 行号而不给原资料截图证据。
- 不要编造页码、题号、老师重点。
- 不要把划掉题放进主复习题库。
- 不要把第 16 章做成复杂计算题重点。
- 不要把不同频率相量直接合并。
- 不要混用最大值和有效值。
- 不确定处必须写 `⚠️ 待人工核对`。

---

## 七、Skill 使用

- 创建/优化 skill 必须先用 `li-skillcreate` + `li-skills-mgmt` 的质量门禁。
- 生成复习 PDF / 1 小时吃透版 / 期末冲刺包时，用 `study-review-pdf`。
- PDF 或图片资料需要读版式/红线时，用 `pdf` 或 `ppocrv5`，并优先做页面渲染目检。
- 高要求复习 PDF：内容可先用 Markdown 草稿审稿，但最终必须用 image2 逐页生成图片，再用 LaTeX 合成 PDF。
- 高要求复习 PDF：每个重要判断尽量配原资料对应区域截图；最终页优先展示或 AI 融合原资料截图，让用户直接看到原始依据。
- 更新 SOP/CLAUDE/AGENT/memory 时按 `li-infra` 的基础设施规则处理。
