## 5. 第13章已知重点与整理要求

第13章名称：非正弦周期电流电路和信号的频谱。

考试范围：13.1—13.4。

### 5.1 第13章 PPT 重点

PPT 明确本章重点为：

1. 周期函数分解为傅里叶级数
2. 非正弦周期函数的有效值和平均功率
3. 非正弦周期电流电路的计算

### 5.2 第13章必须整理出的核心题型

#### 题型 1：判断非正弦周期量与谐波分量

要点：

- 是否周期
- 是否由共同基频的整数倍频率组成
- 直流、基波、高次谐波识别

#### 题型 2：有效值计算

公式：

```latex
I = \sqrt{I_0^2 + I_1^2 + I_2^2 + \cdots}
```

若题目给的是最大值：

```latex
I = \sqrt{I_0^2 + \sum_{k=1}^{\infty}\frac{I_{km}^2}{2}}
```

易错点：

- 正弦项最大值要除以 `sqrt(2)`
- 直流分量不除以 `sqrt(2)`
- 有效值与初相位无关

#### 题型 3：平均功率计算

公式：

```latex
P = U_0 I_0 + \sum_{k=1}^{\infty} U_k I_k \cos\varphi_k
```

要点：

- 只计算同频电压和同频电流
- 不同频率正交，平均功率贡献为 0
- 直流功率单独算 `U0 I0`

#### 题型 4：非正弦周期电流电路计算

标准流程：

1. 展开 / 识别傅里叶分量
2. 直流分量单独作用：C 开路，L 短路
3. 基波分量单独作用：`Z_L=jωL`，`Z_C=1/(jωC)`
4. k 次谐波单独作用：`Z_L=jkωL`，`Z_C=1/(jkωC)`
5. 用相量法分别求响应
6. 求瞬时值：各瞬时值直接相加
7. 求有效值/仪表读数：各分量有效值平方相加开根号
8. 求平均功率：各同频功率相加

#### 题型 5：电磁式仪表读数

要点：

- 电磁式电压表 / 电流表读有效值
- 非正弦情况下不能把瞬时表达式系数直接相加
- 直流分量和各谐波有效值平方相加

#### 题型 6：三相高次谐波

虽然当前考试范围第13章只到 13.4，但如果资料或老师重点涉及 13.5，可作为选择题结论处理：

```text
6n+1 次谐波：正序，例如 1、7、13 次
6n+3 次谐波：零序，例如 3、9、15 次
6n+5 次谐波：负序，例如 5、11、17 次
```

## 9. 当前第13章速记结论

第13章的核心不是背傅里叶推导，而是掌握：

```text
非正弦周期信号
→ 傅里叶分解
→ 分频率单独相量计算
→ 按瞬时值/有效值/平均功率三种规则合成
```

最重要的三条合成规则：

```latex
x(t)=x_0+x_1(t)+x_3(t)+\cdots
```

```latex
X=\sqrt{X_0^2+X_1^2+X_3^2+\cdots}
```

```latex
P=U_0I_0+U_1I_1\cos\varphi_1+U_3I_3\cos\varphi_3+\cdots
```
