### 任务 E：专门生成“第 11 章 + 第 13 章综合大题专题”

输出文件：

```text
outputs/05_11章13章综合大题专题.md
```

这是最高优先级专题之一。

必须包含：

1. 题型本质：
   - 非正弦周期信号分解
   - 串联 RLC 对不同谐波的频率响应
   - 各谐波有效值平方相加
   - 平均功率分谐波相加
   - 某次谐波谐振判断

2. 核心公式：

第 \(n\) 次谐波角频率：

```math
\omega_n = n\omega_1
```

串联 RLC 第 \(n\) 次谐波阻抗：

```math
Z_n = R + j\left(n\omega_1L - \frac{1}{n\omega_1C}\right)
```

阻抗模平方：

```math
|Z_n|^2 = R^2 + \left(n\omega_1L - \frac{1}{n\omega_1C}\right)^2
```

第 \(n\) 次谐波电流有效值：

```math
I_n = \frac{U_n}{|Z_n|}
```

总电流有效值：

```math
I = \sqrt{I_0^2 + I_1^2 + I_2^2 + \cdots}
```

串联 RLC 中若含串联电容，直流分量通常：

```math
I_0 = 0
```

平均功率：

```math
P = \sum_{n=1}^{\infty} I_n^2R
```

也可写成：

```math
P =
\sum_{n=1}^{\infty}
\frac{U_n^2 R}
{R^2+\left(n\omega_1L-\frac{1}{n\omega_1C}\right)^2}
```

串联谐振条件：

```math
n\omega_1 = \omega_0 = \frac{1}{\sqrt{LC}}
```

3. 必须解释“平均功率公式分母怎么来的”：

分母来自第 \(n\) 次谐波下串联 RLC 阻抗模平方：

```math
|Z_n|^2 = R^2 + X_n^2
```

其中：

```math
X_n = n\omega_1L - \frac{1}{n\omega_1C}
```

所以：

```math
P_n = I_n^2R
= \left(\frac{U_n}{|Z_n|}\right)^2R
= \frac{U_n^2R}{|Z_n|^2}
```

4. 必须生成至少 5 道专题训练题：
   - 2 道基础代公式题
   - 2 道带谐振判断的题
   - 1 道综合大题，包含有效值、平均功率、谐振判断、品质因数或通频带

5. 每道题必须附：
   - 题目
   - 解题入口
   - 标准步骤
   - 最终答案
   - 易错提醒
   - 若来自原资料，标注来源；若为新编，标注“新编题”

## 6. 章节处理规则

### 第 1 章：电路模型和电路定律

- 电压、电流参考方向
- 关联参考方向与功率正负
- 电阻元件伏安关系
- 理想电压源、电流源
- 受控源
- KCL、KVL

常见题型：

- 判断电流/电压实际方向
- 判断元件吸收还是发出功率
- KCL/KVL 基础计算
- 电源与受控源概念判断

---

### 第 2 章：电阻电路等效变换

重点：

- 电阻串并联
- 分压、分流
- Y-Δ 变换
- 电压源与电流源等效变换
- 输入电阻

常见题型：

- 等效电阻计算
- 电源等效变换
- 含受控源输入电阻，用测试源法
- 分压分流

---

### 第 3 章：电阻电路的一般分析

重点：

- 独立 KCL 方程数：\(n-1\)
- 独立 KVL 方程数：\(b-n+1\)
- 支路电流法
- 网孔电流法
- 回路电流法
- 节点电压法
- 超节点/含电压源节点法

常见题型：

- 节点电压法列方程
- 网孔电流法列方程
- 含电流源/电压源特殊处理
- 方程个数判断

---

### 第 4 章：电路定理

重点：

- 叠加定理
- 替代定理
- 戴维宁定理
- 诺顿定理
- 最大功率传输定理

常见题型：

- 用戴维宁求负载电流/电压/功率
- 求 \(U_{oc}\)、\(I_{sc}\)、\(R_{th}\)
- 叠加定理中受控源保留
- 功率不能直接叠加
- 最大功率传输：\(R_L=R_{th}\)

---

### 第 6 章：储能元件

重点：

- 电容：\(i_C=C\frac{du_C}{dt}\)
- 电感：\(u_L=L\frac{di_L}{dt}\)
- 电容电压不能突变
- 电感电流不能突变
- 电容/电感储能
- 直流稳态：电容开路，电感短路

常见题型：

- 初始值判断
- 储能计算
- 串并联等效
- 换路前后状态判断

---

### 第 7 章：一阶电路

只到 7-4。

重点：

- 换路定则
- RC/RL 时间常数
- 零输入响应
- 零状态响应
- 一阶响应通式/三要素法若老师包含则优先保留

核心公式：

```math
x(t)=x(\infty)+[x(0^+)-x(\infty)]e^{-t/\tau}
```

注意：

- RC：\(\tau=R_{eq}C\)
- RL：\(\tau=L/R_{eq}\)

---

### 第 8 章：相量法

重点：

- 正弦量三要素
- 有效值与最大值
- 相量表示
- 微分积分的相量形式
- R、L、C 相量伏安关系

## 7. 已有对话结论：可直接继承的复习判断

### 7.1 第 11 + 13 章综合题

已经确认的解释：

```math
P=\sum_{n=1}^{\infty}I_n^2R
```

也可以写成：

```math
P=
\sum_{n=1}^{\infty}
\frac{U_n^2R}
{R^2+\left(n\omega_1L-\frac{1}{n\omega_1C}\right)^2}
```

最后一个公式分母来自：

```math
|Z_n|^2
=
R^2+\left(n\omega_1L-\frac{1}{n\omega_1C}\right)^2
```

其中：

```math
Z_n=R+j\left(n\omega_1L-\frac{1}{n\omega_1C}\right)
```

推导链：

```math
I_n=\frac{U_n}{|Z_n|}
```

```math
P_n=I_n^2R
```

所以：

```math
P_n=
\left(\frac{U_n}{|Z_n|}\right)^2R
=
\frac{U_n^2R}{|Z_n|^2}
```
