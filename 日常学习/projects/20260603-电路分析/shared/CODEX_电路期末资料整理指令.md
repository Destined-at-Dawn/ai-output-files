## 3. Codex 首先要做的资料索引任务

Codex 进入工作区后，第一步不是直接总结，而是建立资料索引。

### 3.1 扫描文件

递归扫描当前工作区下所有文件，重点识别以下类型：

- `.pdf`
- `.ppt`
- `.pptx`
- `.doc`
- `.docx`
- `.md`
- `.txt`
- `.png`
- `.jpg`
- `.jpeg`
- `.np`

### 3.2 建立资料清单

生成文件：

`00_资料索引.md`

每个文件记录：

```md
## 文件名

- 相对路径：
- 推测类型：教材 / PPT / 习题 / 习题答案 / 对话记录 / 其他
- 对应章节：第几章 / 哪几节
- 是否在考试范围内：是 / 否 / 部分是
- 可用程度：清晰 / 部分可读 / 不可读 / 需要 OCR
- 备注：
```

### 3.3 命名规则推断

不要只依赖文件名完全匹配。很多资料可能命名不规范，例如：

- `第十三章.pptx`
- `ch13.ppt`
- `13章答案.pdf`
- `电路习题答案第11章.pdf`
- `chapter_16_slides.pdf`
- `11-2 11-3.docx`

需要结合文件名、正文标题、页眉页脚、目录、题号来判断对应章节。

## 4. 资料优先级

当不同资料说法冲突时，按以下优先级处理：

1. 老师明确划重点 / 考试说明 / 对话中已确认范围
2. PPT
3. 习题答案
4. 习题题目
5. 教材例题与教材课后题
6. 教材正文
7. AI 根据通用电路知识补充的内容

### 冲突处理规则

如果发现教材、PPT、习题答案之间有不一致：

1. 不要自行假装统一
2. 在对应章节标记：`⚠️ 资料冲突`
3. 写清楚冲突来源
4. 优先采用 PPT 或习题答案中的考试版本
5. 保留教材版本作为补充说明

引用格式统一使用：

```md
Source: 相对路径#页码/幻灯片号/题号
```

如果无法确定页码或题号，写：

```md
Source: 相对路径#位置待确认
```

### 原题复现概率判断

优先标记为“高”的情况：

1. 同一道题同时出现在 PPT 和习题中
2. 习题答案中给出详细步骤
3. 教材例题被 PPT 引用
4. 题型非常适合闭卷计算题
5. 题目覆盖核心公式或核心方法
6. 第 11 章和第 13 章相关综合题

标记为“中”的情况：

1. 只在教材课后题出现
2. 是核心方法的普通练习题
3. 是 PPT 上一带而过的例题

标记为“低”的情况：

1. 只考边缘概念
2. 计算量过大，不适合期末闭卷
3. 超出考试范围

```md
## 二端口网络 / 四端网络 / 四端口网络辨析

- 二端口网络：有两个端口，通常有四个外接端子
- 四端网络：强调四个外接端子，不一定严格构成两个端口
- 四端口网络：有四个端口，通常有八个外接端子
- 易错说法：二端口网络就是四端口网络
- 正确说法：二端口网络常可称为四端网络，但不能称为四端口网络
- 可能考法：选择题或判断题考“端”和“端口”的区别
```
