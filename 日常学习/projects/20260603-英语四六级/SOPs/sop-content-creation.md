# SOP: Content Creation Pipeline

## Trigger Intent
用户说"写/做/创建/发" + 内容类型（公众号/小红书/文章/帖子）

## Skill Chain
```
Step 1: li-analyze Phase 0 — 输入清晰度检查（5维打分, <6分追问）
Step 2: li-research — 调研主题（WebSearch + 已有知识库）
Step 3: li-analyze Phase 2 — 道法术器深度（确保有认知科学支撑）
Step 4: li-transcript — 如果是逐字稿来源，清洗转录
Step 5: li-mindcoach — 如果涉及个人成长/决策，注入心力卡片
Step 6: li-improve — 记录创作模式 + 用户反馈
```

## Platform-Specific
- 小红书: xhs-content-strategist (标题/选题) + 小红书留学求职ip号帖子改编 (如果适用)
- 公众号: li-transcript style-guide + baoyu-post-to-wechat
- PPT: ppt-generator-pro + li-storyboard

## Output Format
结论先行 → 平台适配格式 → 百大认知引用 → CTA

## Feedback Loop
- 用户说"不错/发了" → 记录"此创作模式对小黎有效"+平台+主题
- 用户说"改/重写" → 记录"此模式需修正"+具体哪里+修正方向
