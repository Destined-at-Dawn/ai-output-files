# 🔴 第03章 刚体力学基础 - 高频考点

## 核心知识点提取
1. **转动惯量 $J$ (Rigid Body Inertia)**
   - 物理意义：刚体转动惯性的量度。
   - 常用转动惯量公式（轴过中心）：
     - 细杆：$J = \frac{1}{12}ML^2$ (绕中心)；$J = \frac{1}{3}ML^2$ (绕端点)
     - 圆盘/实心圆柱：$J = \frac{1}{2}MR^2$
     - 圆环/空心圆柱：$J = MR^2$
     - 实心球：$J = \frac{2}{5}MR^2$

2. **转动定律 (Newton's Second Law for Rotation)**
   - 公式：$M = J \alpha$
   - 其中 $M = \vec{r} \times \vec{F}$ (力矩)，$\alpha = \frac{d\omega}{dt}$ (角加速度)。

3. **角动量守恒定律**
   - 若 $\sum M_{ext} = 0$，则 $L = J\omega = \text{const}$。
   - **易错点**：碰撞瞬间，动量不守恒（受轴的作用力），但**对轴的角动量守恒**。

4. **刚体动能定理与机械能守恒**
   - 转动动能：$E_k = \frac{1}{2}J\omega^2$。
   - 重力势能：$E_p = Mgh_c$ (以质心高度为准)。
