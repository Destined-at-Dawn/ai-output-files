# 🔴 第08章 电磁感应 - 高频考点

## 核心知识点提取
1. **法拉第电磁感应定律**
   - $\varepsilon = -\frac{d\Psi}{dt} = -N\frac{d\Phi_e}{dt}$
   - 感应电动势分为：**动生电动势** ($\int (\vec{v} \times \vec{B}) \cdot d\vec{l}$) 与 **感生电动势** ($-\int \frac{\partial \vec{B}}{\partial t} \cdot d\vec{S}$)。

2. **动生电动势计算**
   - 旋转导体棒（绕一端旋转）：$\varepsilon = \frac{1}{2}B\omega L^2$。
   - 旋转圆盘：边缘与圆心电势差 $\varepsilon = \frac{1}{2}B\omega R^2$。

3. **自感与互感**
   - 自感电动势：$\varepsilon_L = -L\frac{di}{dt}$。
   - 磁场能量：$W_m = \frac{1}{2}Li^2$。

4. **位移电流与麦克斯韦方程组**
   - 位移电流 $I_d = \varepsilon_0 \frac{d\Phi_e}{dt}$。
   - 核心思想：变化的电场产生磁场，变化的磁场产生电场。
