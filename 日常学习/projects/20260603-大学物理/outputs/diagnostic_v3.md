# 物理诊断测试 (力学与电磁学)



小黎同学，如果之前的文件还是乱码，请尝试打开这个新版本。

为了确保兼容性，我使用了 Windows 标准换行符和 UTF-8 BOM 编码。



1. 运动学：

   已知加速度 a = 2t i + 3 j (单位：m/s^2)，初始速度 v0 = 0。

   求 t 秒时的速度矢量 v(t)。



2. 刚体：

   一根质量为 M、长度为 L 的均匀细杆，绕其“中点”轴转动。

   请写出其转动惯量 J 的表达式。



3. 转动定律：

   已知合外力矩为 M，转动惯量为 J。

   求角加速度 alpha 的计算公式。



4. 守恒律：

   子弹击中一端固定的细杆并嵌入。

   碰撞瞬间，系统对“转轴”的角动量守恒吗？为什么？



5. 静电场：

   库仑定律中的常数 k，用真空介电常数 epsilon_0 表示的形式是什么？



6. 高斯定理：

   无限大均匀带电平面（面密度为 sigma）产生的场强 E 的公式是什么？



7. 稳恒磁场：

   半径为 R 的圆电流 I 在其“圆心”处产生的磁感应强度 B 是多少？



8. 安培环路定理：

   在真空中，磁感应强度 B 沿闭合回路的线积分等于多少？



9. 动生电动势：

   长度为 L 的棒以速度 v 垂直切割磁感应强度为 B 的匀强磁场。

   感应电动势 epsilon 等于多少？



10. 感生电场：

    变化磁场产生的感生电场，其电场线是“闭合”的还是“不闭合”的？
