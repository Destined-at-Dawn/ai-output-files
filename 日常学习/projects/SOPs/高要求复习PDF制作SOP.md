# SOP：高要求复习 PDF 制作

## 结论

高要求复习 PDF 必须做成“考试型复习包”，不是知识点摘要。它的目标是让大学生在 1 小时内抓住本章主线，并能立刻训练小题陷阱、大题模板和拔高题。唯一成功主路径是：资料索引 -> 原题/题图数据库 -> 概念地图 -> 概念辨析与自测 -> 小题速通 -> 大题模板 -> A4速查 -> 拔高页 -> Codex 内置 `image_gen` 逐页直出完整图片 -> 保存原图副本 -> 只做图片装订 PDF。

本 SOP 的“唯一”含义：页面主体不得再用 Markdown/HTML/PIL/Matplotlib/ReportLab/LaTeX/字体层重排来冒充复习页。它们只允许做两件事：整理源材料、把已经由 `image_gen` 直出的图片逐页装订成 PDF。

## 触发条件

命中以下任一情况时，必须执行本 SOP：

- 用户要求复习 PDF、章节复习稿、期末冲刺包、1小时吃透版、A4速查表。
- 用户要求把教材、PPT、习题册、老师讲课题、真题截图嵌入资料。
- 用户要求题型模板、小题速通、选择判断易错点、计算题套路。
- 用户批评资料“信息密度低、太基础、没有原题融合、公式乱码”。

## 总输出结构

每个高要求章节复习包至少包含：

- `00_资料索引.md`：本章所有教材、PPT、习题册、答案、总复习 PDF、真题的可用性。
- `01_考试范围与复习策略.md`：本章考什么、跳过什么、按题型怎么分配时间。
- `02_原题与题图数据库.md`：每道题的来源、题号、截图、考点、题型、复现概率、答案状态。
- `03_思维导图与概念辨析.md`：先讲概念关系，再讲教材定义和易混点。
- `04_选择判断易错点.md`：每个易混点至少配 1 道判断或选择自测题。
- `05_填空与小题速通.md`：按高频公式、常见陷阱、边界条件训练。
- `06_计算题题型总表.md`：每类大题的识别信号、标准步骤、必背公式、代表原题。
- `07_综合大题专题.md`：跨知识点、跨章节或高风险综合题。
- `08_1小时吃透版.md`：按 10/20/15/15 分钟拆解。
- `09_考前A4速查表.md`：只保留考前 10 分钟值得看的公式、条件、陷阱。
- `10_逐页图片预览/`：每一页都是独立图片，等待用户确认。
- `11_答案附页.md`：所有概念辨析题、小题速练题的答案集中放在附页，不直接贴在题目下。
- `12_拔高页/`：每章至少 1-2 页高区分度题型，覆盖非套公式题、综合题或易错边界题。

## 资料优先级

资料权重按下面顺序执行，越新、越靠近期末、越接近老师要求，权重越高：

1. 老师最终复习题、划重点、总复习 PDF、考试范围说明。
2. 老师 PPT 例题和课堂强调题。
3. 习题册答案和老师讲解答案。
4. 习题册原题。
5. 教材例题和教材课后题。
6. 往届真题。
7. 教材正文概念定义。
8. AI 补充解释，只能标注为“补充理解”，不能冒充资料原文。

冲突时不强行统一，保留冲突说明，并优先考试侧资料。

## 章节编写顺序

### P0：思维导图

第一页必须是本章思维导图，但不能停留在概念列表。它必须展示：

- 本章要解决的核心问题。
- 与前面章节的连接：需要调用哪些已学概念。
- 与后面章节的连接：哪些概念会继续用。
- 本章内部关系：核心量、定律、题型之间如何互相转换。
- 高频等级：P0 必会、P1 高频、P2 辅助。

### P1：重点难点一口气讲解

对思维导图中的 P0/P1 节点逐个展开，每个节点必须包含：

- 直觉图像：先讲“它在物理/数学/工程上解决什么问题”。
- 前后联系：它从哪里来，后面会怎么用。
- 教材定义：给出教材或课件中的严谨定义。
- 使用条件：公式或定理什么时候能用，什么时候不能用。
- 易混辨析：至少列出 1 个最容易混淆的概念。
- 检验题：出 1-2 道判断或选择题，答案放附页。

### P2：按考试频次展开知识点

重难点之后，按考试频次往下讲，而不是照搬教材目录。每个知识点必须绑定：

- 对应原题或题图截图。
- 典型问法。
- 一句话解法。
- 最容易错的边界条件。

### P3：小题速通

小题速通覆盖选择、判断、填空，目标是把易错点练到位：

- 每个易错点必须有“错误说法 -> 正确说法 -> 为什么错 -> 自测题”。
- 判断题必须训练概念边界，不只考公式记忆。
- 填空题必须训练公式条件、单位、正负号、分段边界。
- 答案统一进入附页，不在题目下方直接暴露。

### P4：大题必会模板

每类计算题必须有：

- 识别信号：看到什么条件就判定为该题型。
- 标准步骤：闭卷时可复写的步骤。
- 必背公式：写清适用条件。
- 代表原题：绑定原题 ID 和截图。
- 变式方向：老师可能怎么改。
- 扣分点：哪些步骤漏写会丢分。

### P5：A4 速查

A4 速查只保留考前 10 分钟值得看的内容：

- 高频公式和适用条件。
- 概念辨析一句话。
- 选择判断陷阱。
- 大题步骤口令。
- 禁止写大段解释和低频背景知识。

## 图片与公式制作规则

### image2 优先

- 本 SOP 中的 `image2` 指 Codex 内置 `image_gen` 图片生成能力，不是本地命令行工具。
- 高要求复习页必须由 Codex 内置 `image_gen` 直接生成完整成品图。每页 prompt 必须一次性包含：标题、知识主线、教材定义、网课/课堂精髓、概念辨析、题卡、1分钟抓手、易错点、答案区和视觉风格要求。
- 高要求复习页不能用 PIL、Matplotlib、普通 Python 脚本、HTML 截图、ReportLab 文字层、LaTeX 字体层冒充 image2 成品；这些只能做源材料整理、临时预览、或最终“图片装订 PDF”。
- 每一轮高要求图片页必须保留 image_gen 调用记录、prompt 摘要和保存路径。

- 高要求复习 PDF 的主体页必须先生成逐页图片。
- 图片要让人感觉“原题图、公式、讲解文字是同一套版式”，不能把截图突兀贴上去。
- 原资料截图允许裁切、放大、降噪、标注、重排和视觉融合。
- 禁止改变原题条件、数值、图形关系、老师标注和答案含义。

### 公式控制规则

- 默认完全用 image_gen 生成完整视觉页，包括必要的文字和公式。prompt 必须明确要求“中文清晰可读、不要乱码、不要英文假字、不要空白占位、公式只放少量核心形式”。
- 公式控制以“少而准”为原则：每页只放能直接做题的公式；复杂推导用步骤口令和图像表达，不堆 LaTeX。
- 公式不允许从 OCR 乱码文本直接复制，必须由人工整理后的标准表达进入 prompt。
- 如果 image_gen 某页出现公式/文字错误，优先重写 prompt 重新生成整页；不要默认进入 LaTeX/字体层修补。
- 所有页面必须用总览图或逐页预览目检，确认无明显乱码、空框、重叠、页序错误。

### PDF 合成

- PDF 合成只做“图片装订”：每张 image_gen 直出图占满一页，不添加文字层、不重排公式、不改内容。
- 如果用户要求直接生成最终 PDF，可以在逐页图片生成后直接装订；如果用户要求先确认，则只交付图片预览。
- PDF 生成后必须验证页数，并生成 contact sheet / 总览图检查页序、漏页、重复页和明显排版事故。

## 验收门槛

交付前必须自检：

- 是否有资料索引和原题/题图数据库。
- 是否先做思维导图，再讲概念关系。
- 每个重点概念是否有教材定义、易混辨析、自测题。
- 是否有小题速通和大题模板。
- 是否有 A4 速查。
- 是否有至少 1-2 页拔高题，避免整章停留在过易选择/判断题。
- 原题是否以截图或融合视觉进入页面。
- 答案是否集中放附页。
- 是否实际调用 image_gen 直出完整图片页，而不是用脚本排版冒充。
- 是否保存了 image_gen 原图副本、页图目录、最终 PDF、总览图和调用记录。
- 是否经过总览图/逐页目检；若 image_gen 出错，是否重新生成整页。
- 图片是否已目检。
- 未经用户授权“直接生成最终 PDF”时，是否没有擅自装订最终 PDF。

不满足以上条件的，只能标为“结构草稿/预览草稿”，不能标为最终复习 PDF。

## 禁止事项

- 禁止只做基础知识点讲解就声称完成复习 PDF。
- 禁止没有原题数据库就开始写题型模板。
- 禁止把 Source 行号当作页面主体证据。
- 禁止原题截图突兀堆贴，必须融合进统一版式。
- 禁止答案紧跟题目，破坏主动回忆训练。
- 禁止编造页码、题号、老师重点。
- 禁止把 AI 生成题冒充教材/PPT/真题原题。
- 禁止把 PIL/Matplotlib/HTML 截图产物冒充 image2/image_gen 成品。
- 禁止把 ReportLab/LaTeX/字体层排版页冒充 image_gen 直出页。
- 禁止把 OCR 乱码公式直接喂给 image_gen。
- 禁止只做基础题；每章至少要有拔高页或综合题页。
- 禁止在用户未确认或未授权“直接生成最终 PDF”时生成最终 PDF。
