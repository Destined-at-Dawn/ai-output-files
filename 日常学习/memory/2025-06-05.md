## 用户偏好：物理/电路题解题格式（硬规则）

**触发时间**：2025-06-05
**优先级**：高（每次写物理/电路相关答案必须遵循）

### 格式模板

后续所有物理/电路题答案，严格按以下顺序输出：

1. **答案先行** — 开门见山给出最终结果
2. **列已知物理量** — 从题图/文字中提取所有已知量，标注数值和单位
3. **解释物理量含义** — 说明每个符号代表什么
4. **写基本公式** — 写出适用的物理定律/公式
5. **根据图中参考方向修正符号** — 依据参考方向、同名端、KCL/KVL 修正正负号
6. **整理成符号表达式** — 代数推导，先不代数字
7. **代入数据** — 最后才代入数值
8. **写最终结果** — 含单位，必要时注明方向/极性

### 表达句式

```
已知……
其中……表示……
由……定律得……
又因为……
代入得……
整理得……
所以答案为……
```

### 禁止事项
- 禁止写"因为所以"的大白话连贯段落
- 禁止跳过符号推导直接代数
- 禁止在没有明确参考方向时随意定符号

- `SOPs/物理电路解题格式SOP.md`（SOP-01，完整八步规范 + 参考方向检查清单）
