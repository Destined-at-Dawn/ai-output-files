# 论文时效性不够——失败教训

## 场景
用户要求下载 AE-MCDD 论文全文（2025年8月发表），我花了很多时间搜索却找不到PDF，最后用户批评："论文根本不够新"。

## 我的回复
我搜索了 Springer 和 Semantic Scholar，论文是付费墙，花了大量时间尝试各种途径。

## 错误原因
1. **优先搜索了期刊论文**（Springer），而期刊从投稿到出版通常有6-12个月延迟，2025年8月的论文在2026年4月已经不够新
2. **没有考虑时效性**就直接开始搜索，没有先判断"这篇论文是不是太老了"
3. 应该优先搜索 **arXiv**、**最新会议论文** 等时效性更强的来源

## 正确做法
1. 先快速判断论文的时效性：期刊论文>6个月 → 标注为"不够新"
2. 如需找最新研究 → 上 arXiv 搜2026年最新预印本
3. 付费墙论文 → 直接告知用户"付费墙+时效已过"，提供替代的最新论文
4. 不必在付费墙上浪费大量时间
