# 竞赛管理 SOP

> 竞赛工作区特有的竞赛管理标准流程。

## 竞赛流程

1. **调研**：调研竞赛信息
2. **组队**：组建团队
3. **准备**：准备竞赛内容
4. **参赛**：参加竞赛
5. **总结**：总结竞赛经验

## 竞赛记录

每次竞赛必须记录：

```markdown
## 竞赛记录

- **时间**：{YYYY-MM-DD}
- **竞赛**：{竞赛名称}
- **角色**：{角色}
- **成果**：{成果}
- **经验**：{经验}
```

## 竞赛频率

- **集创赛**：每年一次
- **ICAN**：每年一次
- **皮影戏**：每年一次
