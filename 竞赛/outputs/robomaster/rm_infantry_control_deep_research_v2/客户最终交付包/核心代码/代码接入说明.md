# 核心代码接入说明

## 结论

本目录中的 `Core/Inc` 和 `Core/Src` 是可移植到 STM32 HAL 工程的应用层代码。接入时不要直接覆盖 CubeMX 自动生成的工程，而是把这些文件加入工程，然后在自己的 `main.c` 中调用 `App_Init()` 和 `App_Loop()`。

## 接入步骤

1. 用 STM32CubeMX 新建 HAL 工程，先保证 LED 和串口能正常运行。
2. 把 `Core/Inc` 下的 `.h` 加入工程头文件路径。
3. 把 `Core/Src` 下的 `.c` 加入工程编译列表。
4. 参考 `main_integration_example.c`，在初始化完成后调用：

```c
App_Init();

while (1) {
    App_Loop();
}
```

5. 修改 `bsp_port.c`，把空函数替换成真实硬件实现：
   - `BSP_GetTickMs()`：返回 `HAL_GetTick()`
   - `BSP_ReadKey()`：读取按键 GPIO
   - `BSP_SetMotorOutput()`：输出 PWM 或 CAN 电流
   - `BSP_GetMotorSpeed()`：读取编码器速度
   - `BSP_GetYawAngleDeg()` / `BSP_GetPitchAngleDeg()`：读取 IMU 角度
   - `BSP_SetLedL1()` / `BSP_SetLedL2()`：控制指示灯

## 注意事项

- 默认平移速度为 0，解除急停后不会自动前进。
- 没有接电机前，只观察变量，不要让电机直接上电乱转。
- 没有接 IMU 前，不能说云台角度闭环已经实测。
- 没有接编码器前，不能说底盘速度闭环已经实测。
