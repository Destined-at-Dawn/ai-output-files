**CPLID 数据详情**
- 训练集：720 张图像，1120 个标注框（含 215 个缺陷框）
- 验证集：128 张图像，201 个标注框（含 33 个缺陷框）
- 类别：2 类（绝缘子 + 缺陷）
- 标注格式：VOC XML → YOLO TXT

### 1.2 模型训练

**训练配置**
- 模型：YOLO12n（2.56M 参数）
- 数据集：CPLID（720 train + 128 val）
- 设备：RTX 5070 Ti（12GB VRAM）
- 优化器：AdamW（lr=0.001667）
- 批次大小：16
- 图像尺寸：640×640
- 轮次：100

**当前进度**
- 已完成：21 个 epoch
- mAP50：0.973（从 0.315 提升）
- mAP50-95：0.721（进行中）
- 状态：训练顺利，损失持续下降

## 三、技术发现

### 3.1 环境问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| ultralytics git 检测失败 | 中文路径导致 OSError | 修改 `git.py` 的 `_find_root` 方法 |
| Windows 虚拟内存不足 | PyTorch 多进程共享内存 | 设置 `workers=0` |
| Git LFS 不可用 | 系统未安装 | 已安装 Git LFS v3.6.1 |
