# 2026-06-11 项目记忆

## 10:00 — 指导老师进度汇报

小黎需要向指导老师汇报竞赛进度，要求生成 MD + docx 两个版本。

### 产出
- `outputs/指导老师进度汇报.md` — Markdown 版
- `outputs/指导老师进度汇报.docx` — 中文公文规范格式（黑体标题 + 仿宋正文 + 28pt 行距 + A4 标准页边距）
- `outputs/gen_report_docx.py` — docx 生成脚本（可复用）

### docx 中文规范
- 标题：黑体 22pt 居中
- 一级标题：黑体 16pt
- 二级标题：黑体 14pt
- 正文：仿宋 12pt，首行缩进 2 字符
- 行距：28pt 固定值
- 页面：A4，上下 2.54cm，左右 3.17cm
- 表格：表头黑体加粗，内容仿宋居中

## 10:30 — 中文写作规范建立

小黎指出必须有明确的中文写作规范，不能碰巧没犯错。

### 新建规则
