## 22:30 经验沉淀 Agent（Memory Watcher）

### 用户需求
"只要 memory 文件变化，就能自动帮我们完成经验文件的书写。每次开始任务前读取经验文件，最大程度节约上下文窗口"

### 实现方案
**后台 Python 脚本 + 关键词提取 + 哈希去重**

- 脚本：`Yolo算法比赛/scripts/memory_watcher.py`
- 经验文件：`Yolo算法比赛/memory/lessons-learned.md`
- 启动器：`Yolo算法比赛/scripts/start_memory_watcher.bat`

### 工作原理
1. 每 30 秒轮询监控两个 memory 目录（项目级 + 工作区级）
2. 检测文件修改时间变化
3. 按 ## 分割段落，用关键词匹配筛选教训段落
4. MD5 哈希去重（避免重复提取）
5. 追加到经验文件 lessons-learned.md
6. 启动序列新增 Step 3.5 读取经验文件

### 技术要点
- 纯 Python，无外部依赖（不需要 watchdog）
- GBK 编码安全处理（移除 emoji 等特殊字符）
- 已有教训哈希持久化（从经验文件中加载）
- 支持 --once / --daemon / --interval 参数

### 首次运行结果
- 从 memory/2026-06-13.md 提取 2 条教训
- 经验文件 2051 bytes，格式正确

### 使用方式
```
# 后台启动
双击 Yolo算法比赛/scripts/start_memory_watcher.bat

# 或命令行
python Yolo算法比赛/scripts/memory_watcher.py --daemon
```

### 与启动序列集成
