## 二、软件环境搭建

### 2.1 RDK X5 系统准备

```bash
# 1. 烧录系统（参考地平线官方文档）
# 下载 RDK X5 Ubuntu 22.04 Server 镜像
# 使用 balenaEtcher 或 dd 写入 microSD 卡

# 2. 系统更新
sudo apt update && sudo apt upgrade -y

# 3. 安装基础工具
sudo apt install -y git cmake build-essential python3-pip i2c-tools

# 4. 启用UART（如果使用UART通信）
sudo nano /boot/firmware/config.txt
# 添加:
# enable_uart=1

# 5. 启用I2C（如果直连PCA9685）
sudo nano /boot/firmware/config.txt
# 添加:
# dtparam=i2c_arm=on

# 6. 重启
sudo reboot
```

### 2.2 Python环境

```bash
# 创建虚拟环境（推荐）
python3 -m venv puppet_env
source puppet_env/bin/activate

# 安装核心依赖
pip install --upgrade pip
pip install opencv-python opencv-contrib-python
pip install mediapipe
pip install numpy
pip install pyserial
pip install smbus2  # I2C通信
pip install adafruit-circuitpython-servokit  # PCA9685控制

# 如果使用ROS2
sudo apt install -y ros-humble-desktop python3-colcon-common-extensions
```

## 三、硬件连接步骤

### 3.1 接线图

```
                    ┌───────────────┐
                    │   RDK X5      │
                    │               │
                    │ GPIO14 (TX) ──┼────── ESP32/STM32 RX
                    │ GPIO15 (RX) ──┼────── ESP32/STM32 TX
                    │ GND ──────────┼────── GND (共地)
                    │ USB ──────────┼────── USB摄像头
                    │               │
                    │ I2C1 SDA ─────┼────── PCA9685 SDA (可选直连)
                    │ I2C1 SCL ─────┼────── PCA9685 SCL
                    └───────────────┘
                            │
                            ▼
                    ┌───────────────┐
                    │   ESP32/STM32 │
                    │               │
                    │ I2C SDA ──────┼────── PCA9685 SDA
                    │ I2C SCL ──────┼────── PCA9685 SCL
                    │ 3.3V ────────┼────── PCA9685 VCC
                    │ GND ─────────┼────── PCA9685 GND
                    └───────────────┘
                            │
                            ▼
                    ┌───────────────┐
                    │   PCA9685     │
                    │   PWM驱动板   │
                    │               │
                    │ PWM0 ─────舵机1 (右肩)  │
                    │ PWM1 ─────舵机2 (右肘)  │
                    │ PWM2 ─────舵机3 (左肩)  │
                    │ PWM3 ─────舵机4 (左肘)  │
                    │ PWM4 ─────舵机5 (右髋)  │
                    │ PWM5 ─────舵机6 (右膝)  │
                    │ PWM6 ─────舵机7 (左髋)  │
                    │ PWM7 ─────舵机8 (左膝)  │
                    │ PWM8 ─────舵机9 (头部)  │
                    │ PWM9 ─────舵机10 (腰部) │
                    │ PWM10 ────舵机11 (道具) │
                    │ PWM11 ────舵机12 (道具) │
                    │               │
                    │ V+ ─── 5V/10A电源 (+) │
                    │ GND ─── 5V/10A电源 (-)│
                    └───────────────┘
```

### 3.2 关键注意事项

1. **共地**: RDK X5、下位机、PCA9685、舵机电源的GND必须全部连接在一起
2. **舵机独立供电**: 10个以上舵机同时工作时电流可达5-8A，绝对不能从RDK X5取电
3. **信号线**: UART通信线长建议<20cm，过长可能引入干扰
4. **上拉电阻**: I2C的SDA/SCL需要4.7kΩ上拉电阻（PCA9685模块一般自带）
5. **电平匹配**: RDK X5是3.3V IO，ESP32也是3.3V，直连没问题；如果接5V单片机需要电平转换

### 3.3 I2C地址检查

```bash
# 检测PCA9685是否被识别（I2C地址默认0x40）
sudo i2cdetect -y 0  # RDK X5可能有多个I2C总线，尝试 -y 0, -y 1, -y 2
```

- [ ] RDK X5系统烧录+网络配置

- [ ] I2C检测到PCA9685

### 阶段2：单关节测试（1天）

- [ ] ESP32/STM32串口通信互通
- [ ] PCA9685单路PWM输出
- [ ] 单个舵机角度控制

### 阶段3：全链路打通（2-3天）

- [ ] RDK X5 → UART → ESP32 → PCA9685 → 1个舵机
