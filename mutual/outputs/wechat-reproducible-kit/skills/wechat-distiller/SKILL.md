## Phase 0: 输入确认

**输入要求**：已导出的 Markdown 聊天文件。

**搜索顺序**：
1. 用户指定路径
2. 创作区导出目录：`E:\ai产出文件\牛马\创作\创作\output\`
3. 个人区导出目录：`E:\ai产出文件\牛马\个人\个人\projects\20260425-成长历程\成长历程\交往\`

**验证**：
- 文件存在且可读
- 文件格式符合 `[日期 时间] 发言人：内容`
- 消息数 >= 100（太少无法做统计分析）

## Phase 1: 消息预处理

**目标**：将 MD 文件解析为结构化数据。

```python
import re
from datetime import datetime

def parse_chat_md(filepath):
    """解析 [YYYY-MM-DD HH:MM:SS] 发言人：内容 格式"""
    pattern = r'\[(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2})\] (.+?)：(.+)'
    messages = []
    with open(filepath, 'r', encoding='utf-8') as f:
        for line in f:
            match = re.match(pattern, line.strip())
            if match:
                ts = datetime.strptime(match.group(1), '%Y-%m-%d %H:%M:%S')
                messages.append({
                    'time': ts,
                    'sender': match.group(2),
                    'content': match.group(3)
                })
    return messages
```

**输出**：消息列表 + 基础统计（总条数/时间跨度/发送者分布）

## Phase 2: 五维心理分析

**目标**：从消息模式推断人格特征。

### 维度 1：情绪模式
- 情绪词频统计（积极/消极/中性）
- 情绪波动曲线（按日/周聚合）
- 情绪触发词识别

### 维度 2：社交模式
- 主动发起 vs 被动回复比例
- 消息长度分布（短句 vs 长段）
- 回复速度分析（如果时间戳精确）

### 维度 3：认知模式
- 问句比例（好奇心指标）
- 抽象词 vs 具象词比例
- 逻辑连接词使用频率

### 维度 4：表达 DNA
- 高频词/短语提取
- 句式偏好（陈述/疑问/感叹）
- 标点使用习惯（省略号/感叹号/emoji）

### 维度 5：关系模式
- 对不同人的语气差异
- 亲密称呼使用频率
- 话题偏好（自我表露深度）

详见 `references/analysis-framework.md`

## Phase 3: 蒸馏为 Skill

**目标**：将分析结果转化为可执行的人物 Skill。

**五步蒸馏法**：
1. **提取心智模型**：这个人如何看待世界？
2. **决策启发式**：遇到选择时，他/她怎么决定？
3. **表达 DNA**：说话/写作的独特模式是什么？
4. **反模式**：什么情况下他/她会犯错或失控？
5. **诚实边界**：哪些推断有足够证据，哪些是猜测？

详见 `references/distillation-guide.md`

**输出格式**：
```markdown
# {姓名} 人格 Skill

## 镜片（核心世界观）
- ...

## 直觉规则（决策启发式）
- 当...时，倾向于...

## 表达 DNA
- 高频词：...
- 句式偏好：...
- 语气特征：...

## 反模式（易犯错误）
- ...

## 边界声明
- ✅ 有充分证据的推断：...
- ⚠️ 中等置信度的推断：...
- ❓ 猜测：...
```

## 反模式

1. **过度推断**：从 100 条消息推断"人格"是粗筛，不是诊断
2. **忽略上下文**：同一个人在不同关系中表现不同
3. **确认偏误**：只找支持预设结论的证据
4. **样本偏差**：导出的聊天记录可能不完整（删了的消息看不到）
5. **文化假设**：不要用西方心理学框架硬套中文聊天习惯
6. **隐私越界**：分析结果只用于用户指定用途，不外泄
