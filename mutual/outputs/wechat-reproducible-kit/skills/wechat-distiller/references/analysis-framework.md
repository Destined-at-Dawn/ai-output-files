# 五维心理分析框架

> 理论基础 + 信号到推断映射表 + 分析报告模板

## 理论基础

### 大五人格模型（Big Five）
- 开放性（Openness）：好奇心、想象力、创造力
- 尽责性（Conscientiousness）：自律、组织性、目标导向
- 外向性（Extraversion）：社交能量、活跃度、表达欲
- 宜人性（Agreeableness）：同理心、合作性、信任度
- 神经质（Neuroticism）：情绪波动、焦虑倾向、压力反应

### 依恋理论（Attachment Theory）
- 安全型：稳定、信任、适度依赖
- 焦虑型：渴望亲密、害怕被抛弃、过度确认
- 回避型：强调独立、回避亲密、情感隔离
- 混乱型：矛盾行为、不稳定

### 认知风格
- 分析型 vs 直觉型
- 系统化 vs 共情化
- 具体思维 vs 抽象思维

## 信号到推断映射表

### 情绪模式信号

| 信号 | 可能推断 | 置信度 |
|------|---------|--------|
| 积极词频 > 60% | 情绪稳定偏乐观 | 中 |
| 感叹号频繁 | 情绪表达强烈 | 中 |
| 省略号频繁 | 犹豫或未完成感或留白风格 | 低 |
| emoji 多样且频繁 | 社交能量高 | 中 |
| emoji 极少 | 内敛或正式风格 | 低 |
| 哈哈哈频率高 | 用笑缓解紧张或习惯性附和 | 低 |
| 负面词集中在特定时段 | 有周期性情绪波动 | 中 |

### 社交模式信号

| 信号 | 可能推断 | 置信度 |
|------|---------|--------|
| 主动发起 > 60% | 社交主导型或关系维护意识强 | 中 |
| 被动回复 > 60% | 社交被动型或等待他人带动 | 中 |
| 消息长度均值 > 50 字 | 表达欲强或信息密度偏好 | 中 |
| 消息长度均值 < 10 字 | 简洁风格或手机打字习惯 | 低 |
| 连续发送 3+ 条短消息 | 思维碎片化或即时表达习惯 | 低 |
| 长段独白式消息 | 深度思考或倾诉需求 | 中 |

### 认知模式信号

| 信号 | 可能推断 | 置信度 |
|------|---------|--------|
| 问句比例 > 30% | 好奇心强或探索型 | 中 |
| 为什么和怎么频繁 | 分析型认知 | 中 |
| 我觉得和应该是频繁 | 直觉型认知 | 低 |
| 引用外部来源 | 信息素养高或论证习惯 | 中 |
| 数据和数字出现频繁 | 量化思维 | 中 |
| 类比和比喻频繁 | 形象思维或教学倾向 | 中 |

### 表达 DNA 信号

| 信号 | 可能推断 | 置信度 |
|------|---------|--------|
| 确实和真的和绝对 | 确定性表达偏好 | 中 |
| 可能和也许和大概 | 不确定性表达偏好 | 中 |
| 哈哈和嘿嘿和呜呼 | 口语化或轻松风格 | 高 |
| 标准书面语 | 正式或严谨风格 | 中 |
| 方言或网络用语 | 地域或代际特征 | 中 |
| 缩写和简写频繁 | 效率导向或年轻群体 | 低 |

### 关系模式信号

| 信号 | 可能推断 | 置信度 |
|------|---------|--------|
| 称呼变化（昵称到全名） | 关系距离变化 | 中 |
| 自我表露深度增加 | 信任建立过程 | 中 |
| 话题从浅到深 | 关系发展阶段 | 中 |
| 幽默和调侃频率 | 亲密程度指标 | 中 |
| 正式和礼貌用语 | 关系边界或尊重 | 中 |
| 争吵后快速修复 | 安全型依恋 | 低 |
