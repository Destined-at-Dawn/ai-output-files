# Marvis 知识库导入指南 — 百大认知书籍

> 目标：让 Marvis 的 RAG 引擎索引 64 本书的读书笔记，实现自然语言检索
> 前提：Marvis 已启动（`E:\Marvis\Application\MarvisLauncher.exe`）

---

## 快速操作（3 步，2 分钟）

### Step 1：打开 Marvis 文档识别设置

1. 点击系统托盘的 Marvis 图标（或从桌面快捷方式打开）
2. 点击左下角 **齿轮图标**（设置）
3. 找到 **「文档识别」** 或 **「本地知识库」** 选项卡
4. 点击 **「立即授权」** 或 **「添加文件夹」**

### Step 2：添加知识中枢目录（推荐，一次搞定）

在文件夹选择器中，导航到以下路径并选中：

```
E:\ai产出文件\牛马\知识中枢
```

> **为什么选这个目录？** 百大认知书籍已经通过 Junction 链接到了知识中枢 vault 下的 `11-百大认知` 子目录。索引知识中枢 = 一次性索引所有内容（知识中枢 + 百大认知 + 所有工作区规则）。

**如果只想索引百大认知**，添加这个路径：

```
E:\ai产出文件\牛马\个人\个人\百大认知书籍
```

### Step 3：等待索引完成

- Marvis 会自动扫描目录下所有 .md 文件（共 74 个，1.7MB）
- 首次索引需要几分钟（取决于 CPU 和模型加载速度）
- 索引过程中，Marvis 会：
  - 解析 Markdown 内容
  - 用本地模型（OpenVINO）生成向量嵌入
  - 存入 LanceDB 向量数据库
- 完成后，你可以直接在 Marvis 对话中提问

## 索引完成后的使用方式

### 方式 1：直接对话提问

在 Marvis 对话框中输入：
- 「反脆弱这本书的核心观点是什么？」
- 「哪些书讲了认知负荷理论？」
- 「间隔效应在哪些书里有提到？」
- 「对比丹尼尔·卡尼曼和纳西姆·塔勒布关于风险的观点」

### 方式 2：跨书主题检索

- 「关于习惯养成，有哪些不同作者的观点？」
- 「决策相关的书籍有哪些核心概念？」
- 「搜索所有提到心流状态的段落」

### 方式 3：结合系统能力

- 「帮我把认知觉醒这本书的笔记复制到桌面」（文件操作）
- 「搜索所有提到 OPPT 模型的段落，生成一份对比报告」（RAG + 生成）

## 常见问题

**Q: 索引卡住了怎么办？**
A: 重启 Marvis 后会自动继续未完成的索引。

**Q: 如何更新索引（新增/修改了笔记）？**
A: Marvis 会自动监控授权文件夹的变化，新增/修改的文件会自动重新索引。

**Q: 搜索结果不准确？**
A: 尝试用更具体的关键词提问。Marvis 的 RAG 是语义检索，不是全文匹配——问「反脆弱的核心概念」比问「书里说了什么」效果好。

**Q: 能同时搜多个知识库吗？**
A: 可以。Marvis 会自动从所有已授权的知识库中检索相关内容。
