# Marvis 即用 Prompt · Obsidian 知识库搭建

> 复制以下内容到 Marvis 对话框中执行。
> 前置条件：Obsidian 已安装，知识中枢 vault 已打开过一次。

---

## Prompt 1：桥接百大认知（5 分钟）

```
我需要你帮我把两个知识目录连通。

背景：我有一个 Obsidian vault 在 E:\ai产出文件\牛马\知识中枢\
里面已有求职、竞赛、创作等 15 个目录。
另外我在 E:\ai产出文件\牛马\个人\个人\百大认知书籍\ 有 64 本书的结构化读书笔记（含 172KB 的总索引、7 域色标分类、42 个概念聚类）。

这两个目录目前完全隔离。我需要你：

1. 以管理员身份运行以下命令，创建 Windows Junction：
   mklink /J "E:\ai产出文件\牛马\知识中枢\11-百大认知" "E:\ai产出文件\牛马\个人\个人\百大认知书籍"

2. 验证 Junction 创建成功（列出 11-百大认知 目录内容，确认有 000-总索引-Index.md 等文件）

3. 在 Obsidian 中打开 vault，确认左侧文件树能看到 11-百大认知 目录

注意：Junction 不复制文件，只创建入口。删除 Junction 不会删除原文件。
```

## Prompt 2：安装 Obsidian 插件（10 分钟）

```
帮我配置 Obsidian vault（E:\ai产出文件\牛马\知识中枢）的插件。

请按以下顺序操作：

1. 打开 Obsidian → 设置 → 第三方插件 → 关闭安全模式
2. 浏览社区插件，安装以下 3 个：
   - Dataview（动态查询笔记元数据）
   - Templater（新笔记自动套模板）
   - Local REST API（让外部 AI 工具通过 HTTP 读写 vault）

3. 配置插件：
   - Dataview：设置中开启 "Enable JavaScript Queries"
   - Templater：模板文件夹设为 _templates
   - Local REST API：记下默认端口（应该是 27124）和 API Key

4. 删除 vault 里 .obsidian/plugins/ 下的 local-rest-api.zip（9字节占位文件）

5. 创建模板目录 _templates/，在其中创建以下 4 个模板文件（内容我稍后提供）

完成后告诉我：3 个插件是否全部启用成功，以及 REST API 的端口号和 Key。
```

## Prompt 4：批量添加 frontmatter（20 分钟）

```
帮我为 64 本认知书籍的读书笔记批量添加 Obsidian frontmatter。

目标目录：E:\ai产出文件\牛马\知识中枢\11-百大认知\

规则：
1. 只处理文件名格式为 NNN-书名.md 的文件（如 001-多巴胺国度-Dopamine-Nation.md）
2. 跳过已有 frontmatter（以 --- 开头）的文件
3. 在文件头部追加以下 YAML 块（不改动正文内容）：

---
title: "书名"
book_id: "NNN"
quality: "A/B/C"
tags: [认知书籍]
---

质量分级依据（来自 .format-standard.md）：
- Level A（13本）：029, 037, 041, 044, 045, 046, 047, 050, 051, 052, 057, 060, 062
- Level B（25本）：001-022, 025, 026, 028
- Level C（24本）：其余

4. 操作前先备份：在 E:\ai产出文件\牛马\归档\ 创建 2026-06-10-百大认知-frontmatter备份\ 目录，复制全部 .md 文件
5. 执行完成后，报告：成功注入 X 个文件，跳过 Y 个文件
```

## Prompt 5：更新仪表盘（5 分钟）

```
帮我更新 Obsidian vault 的仪表盘。

文件路径：E:\ai产出文件\牛马\知识中枢\00-主页\00-仪表盘.md

需要修改：在「工作区导航」区域，在"🔴 个人档案"之前，新增一个板块：

### 🧠 [[11-百大认知/000-总索引-Index|百大认知]] — 64本书 · 7域 · 42概念
> 认知科学工具箱：脑科学、学习科学、习惯系统、专注力、系统思维、社会认知、人生哲学

- [[11-百大认知/000-总索引-Index|总索引]] · [[11-百大认知/.cross-domain-index|跨域索引]] · [[11-百大认知/.format-standard|格式标准]]

同时在「全局索引」表格中新增一行：
| 百大认知 | 64本书·7域·42概念聚类 | [[11-百大认知/000-总索引-Index|总索引]] |

修改前先读取当前文件内容，确认不会覆盖已有内容。
```

## 使用顺序

```
Prompt 1（桥接）→ Prompt 2（插件）→ Prompt 3（RAG）→ Prompt 4（frontmatter）→ Prompt 5（仪表盘）
```

每个 Prompt 独立可执行。如果某一步失败，不影响其他步骤。
