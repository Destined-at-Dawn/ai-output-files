## 📚 结构化笔记

### 模块一：极限的概念

- ⭐ **极限的直觉理解**：一个量无限接近某个值，但不一定等于那个值。就像你走向墙壁，每步走剩余距离的一半——你永远在接近，但永远没碰到。
- ⭐ **ε-δ定义**（epsilon-delta）：极限的严格数学定义。「对于任意小的误差ε，我都能找到一个范围δ，让函数值落在误差范围内」——本质是在说「我可以让它想多近就多近」
- ⚠️ **极限存在的条件**：左极限 = 右极限。如果从左边靠近和从右边靠近得到不同的值，极限不存在。这是很多人做题会踩的坑
- 极限的四则运算法则：两个有极限的函数，加减乘除后极限等于极限的加减乘除（除法要求分母极限不为0）
- 夹逼定理：如果一个函数被两个有相同极限的函数夹住，那它的极限也相同。像三明治一样，上下两片面包碰到一起，中间的馅也只能在那个位置

### 模块二：连续性

- ⭐ **连续的三个条件**：①函数在这点有定义 ②极限存在 ③极限值等于函数值。三个缺一不可
- ⭐ **间断点分类**：可去间断点（补个值就连续了）vs 跳跃间断点（左右极限不等）vs 无穷间断点（趋向无穷）
- ⚠️ **连续函数的性质**：介值定理——连续函数如果在两端取了不同符号的值，中间一定过零点。直觉上就是「画一条不断开的线，从正的画到负的，一定会穿过x轴」
- 一致连续 vs 点点连续：一致连续要求整个区间上用同一个δ，点点连续只要求每个点各自有δ。一致连续比点点连续要求更高

### 模块三：导数的引入

- ⭐ **导数的几何意义**：就是切线斜率。函数图像在某一点的「瞬时变化速度」
- ⭐ **导数的物理意义**：位移对时间的导数 = 速度。速度对时间的导数 = 加速度
- 可导一定连续，连续不一定可导：经典反例是|x|在x=0处，连续但不可导（尖角处没有唯一的切线方向）

---

## 🗺️ 学科地图

### 核心概念网络

**极限**（一切的基础）
  ├── 定义了 → **连续性**（极限值 = 函数值）
  ├── 定义了 → **导数**（极限的一种特殊形式：变化率的极限）
  └── 定义了 → **积分**（无穷小量之和的极限，后续章节）

**连续性**
  ├── 是 → 可导的必要条件（可导一定连续）
  └── 保证了 → 介值定理、最值定理等工具可用

**导数**
  ├── 描述了 → 函数的局部变化行为
  └── 连接了 → 几何（切线）和物理（速度、加速度）

一句话总结：极限是地基，连续性是第一层楼，导数是第二层楼。地基不稳，上面全塌。

### 专家思维模型

数学家遇到一个分析问题时，通常会：
1. **先想极限行为**——这个东西在边界/趋向无穷/趋向某点时会怎样？
2. **再看连续性和光滑性**——有没有断裂、尖角、奇点？
3. **然后用局部逼近**——在局部用简单的东西（直线/多项式）近似复杂的东西
4. **最后关注存在性和唯一性**——解存在吗？唯一吗？还是有多个？

这个思维方式贯穿整个高数：从极限到微分到积分到级数，核心都是「用简单逼近复杂」。

### 共识与争议

✅ **共识**：
- 极限的ε-δ定义是严格的、无歧义的（19世纪柯西和魏尔斯特拉斯确立）
- 实数的完备性保证了很多极限的存在性

⚖️ **争议**：
- **无穷小量的地位**：标准分析（用ε-δ）vs 非标准分析（用真正的无穷小量）。两种体系都是严格的，但哲学理解不同。你的教材用的是标准分析。

❓ **开放问题**：
- 这些在本课程范围内不太涉及，但如果你感兴趣：函数空间上的收敛性问题（点态收敛 vs 一致收敛）会在后续课程中出现。

假设学员喂入了宏观经济学第2-3章课件：

---

## 📚 结构化笔记

### 模块一：GDP的衡量

- ⭐ **GDP的定义**：一个国家在一定时期内生产的所有最终商品和服务的市场价值。关键词是「最终」——中间产品不算，否则会重复计算
- ⭐ **三种计算方法**：支出法（C+I+G+NX）= 收入法（工资+利润+租金+利息）= 生产法（增加值加总）。三种方法得到的数字理论上相同
- ⚠️ **名义GDP vs 实际GDP**：名义GDP用当年价格算，会被通胀「虚胖」。实际GDP用基期价格算，剔除了价格变化的影响。比较经济增长必须用实际GDP
- GDP平减指数 = 名义GDP / 实际GDP × 100，衡量整体物价水平变化
- GDP的局限性：不反映收入分配、不包含地下经济、不衡量幸福感。一个国家GDP很高但基尼系数也很高，说明总量大但分配不均

### 模块二：通货膨胀

- ⭐ **CPI（消费者物价指数）**：选一篮子商品，追踪它们的价格变化。反映的是普通消费者感受到的物价变化
- ⭐ **通胀的本质**：太多的钱追逐太少的商品。货币供应增长快于实际产出增长 → 物价上涨
- ⚠️ **需求拉动型 vs 成本推动型**：需求太旺（大家都要买）→ 需求拉动型通胀。成本上升（原材料涨价）→ 成本推动型通胀。两种原因不同，应对策略也不同
- 通胀预期的自我实现：如果所有人都预期明年物价涨10%，工人要求涨薪10%，企业就真的涨价10%。预期本身变成了通胀的原因
- 菲利普斯曲线：失业率和通胀率的短期反向关系——失业率低时通胀倾向于高，反之亦然。但长期这个关系可能不成立

---

## 🗺️ 学科地图

### 核心概念网络

**GDP**（衡量经济总产出）
  ├── 受影响于 → **消费C、投资I、政府支出G、净出口NX**
  ├── 会被 → **通货膨胀** 扭曲（所以要区分名义vs实际）
  └── 反映不了 → 收入分配、幸福感、可持续性

**通货膨胀**（物价总水平的持续上升）
  ├── 由 → 货币供应、需求过旺、成本推动 引起
  ├── 通过 → CPI、GDP平减指数 衡量
  └── 与 → 失业率 存在短期权衡（菲利普斯曲线）

**货币政策**（央行调控）
  └── 通过调节货币供应和利率 → 同时影响GDP增长和通胀

一句话总结：宏观经济学的核心矛盾就是「增长 vs 稳定」——想让GDP涨得快，可能带来通胀；想控制通胀，可能牺牲增长。

### 专家思维模型

经济学家遇到一个宏观问题时，通常会：
1. **先想激励结构**——这个政策改变了谁的激励？谁会多做什么、少做什么？
2. **再看总量均衡**——总供给和总需求在哪里相交？政策是移动了需求曲线还是供给曲线？
3. **然后考虑时间维度**——短期效果和长期效果可能完全相反（短期刺激→长期通胀）
4. **最后关注预期**——人们预期这个政策会怎样？预期本身会改变结果

### 共识与争议

✅ **共识**：
- GDP是目前最广泛使用的经济总量指标（虽然有缺陷）
- 恶性通胀（年通胀率>50%）对经济有严重破坏

⚖️ **争议**：
- **通胀目标应该是多少**：大多数央行锚定2%，但有经济学家认为3-4%更合理（给货币政策更多空间）
- **菲利普斯曲线是否还有效**：2010年代出现了「低失业率+低通胀」的组合，挑战了传统菲利普斯曲线。有人认为全球化和技术进步改变了这个关系

❓ **开放问题**：
- 数字货币（CBDC）会怎样改变货币供应和通胀的关系？这是目前的前沿讨论
