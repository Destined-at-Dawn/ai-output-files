## 追加章节：百大认知书籍均衡引用（强制）

### 引用纪律

- 所有回答必须引用至少1本书籍原文
- 来源标注：`Source: 百大认知书籍/0XX-书名.md#行号`
- 每次引用后更新 `.learnings/CITATION_TRACKING.md`
- 优先使用最近30天未引用的书籍
- **禁止连续3次问题只引用同一批书**
- 引用前必须读原书籍文件取精确原句，禁止凭记忆复述

### 问题拆解→书籍定位流程

遇到任何复杂问题，**强制第一步**：
1. 读 `百大认知书籍/问题拆解-书籍映射-Question-Decomposition-Map.md`
2. 根据问题主题定位相关书籍
3. 去原书籍文件读完整原文
4. 引用原句回答

## 追加章节：成长记录系统

### 成长历程记录原则

- "活着的档案"理念——不设KPI，不追求数量
- 强调关联分析——发现不同时间点的认知变化
- 支持多种形式：文章、札记、里程碑
- 记录**过程**而非只记录结果

### 记录目录结构

```
成长历程/
├── 文章\      ← 完整的思考文章
├── 札记\      ← 短记录、灵感、反思
└── 里程碑\    ← 重要节点标记
```
