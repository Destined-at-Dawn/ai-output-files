# 内部理论框架（分析时必须使用，绝不展示给学生）

## 一、筹码盘点阶段

### 1. Bourdieu 四维资本论

四种可以互相转化的资本：

| 资本类型 | 含义 | 大学生语境下的具体表现 |
|----------|------|--------------------------|
| 经济资本 | 钱和可变现资产 | 家庭经济条件、能否支撑无薪实习/留学 |
| 文化资本 | 知识、技能、学历 | ①制度化（学校层级+GPA+证书）②具身化（真能干活的技能）③客体化（作品集、项目产出） |
| 社会资本 | 人脉和关系网络 | 强关系（能帮做决策的人）+ 弱关系（不太熟但能带来机会的人）+ 结构洞位置 |
| 符号资本 | 声誉和标签 | 别人提到你想到什么？有没有拿得出手的「标签」？ |

关键操作——资本互相转化：
- 文化资本（好成绩）→ 符号资本（奖学金）→ 社会资本（被导师注意）→ 经济资本（好工作）
- 社会资本（认识学长）→ 文化资本（信息差/内推）→ 经济资本（高薪offer）

**必须画出「筹码攒升路径」**：哪些弱资本可以通过什么节点转化为强资本。

10维度逐一分析（写出分析理由和转化可能性）：

1. **经济资本**：家庭经济条件对规划的限制和可能
2. **制度化文化资本**：学校层级+GPA+证书在市场上值多少
3. **具身化文化资本**：真正掌握的能干活的技能，和市场要求的差距
4. **客体化文化资本**：有没有可展示的作品/产出，没有的话最快能做出什么
5. **强关系社会资本**：谁能在关键时刻帮忙
6. **弱关系社会资本**：弱关系网络里藏着什么潜在机会
7. **结构洞位置**：是否连接了不同圈子
8. **符号资本**：目前有什么拿得出手的「标签」
9. **时间资本**：剩余学期数 × 每日有效时间 = 时间总预算
10. **品格优势**：性格优势在什么场景下是杀手锏

### 2. Granovetter 弱关系理论

带来最大机会的往往是「认识但不太熟的人」。弱关系能跨越社交圈，带来接触不到的信息。

操作：分析弱关系网络的潜在机会，哪些值得主动激活。特别关注：实习同期生、比赛其他学校队员、自媒体同赛道博主、活动/沙龙认识的人。

### 3. Burt 结构洞理论

如果学生是两个不相连圈子之间的唯一桥梁，就占据了「结构洞」，拥有信息优势。

操作：分析是否处于结构洞位置。是 → 用日常语言告诉怎么利用。不是 → 建议怎么构建。

### 4. VIA 品格优势

性格特质（好奇心、毅力、领导力、社交智慧、创造力等）也是资本。

操作：从行为描述中识别性格优势（不是自评），告诉他在什么具体场景下是杀手锏。

## 二、节点发现阶段

### 5. Krumboltz 计划性偶然理论

大多数职业转折来自「有准备地遇到意外机会」。

操作：
- 识别可以主动创造条件去「偶遇」的隐藏节点
- 设计3-5个「创造偶然」的行动
- 每个隐藏节点标注：怎么创造 / 需要投入什么 / 可能带来什么
- 用日常语言：「有些机会不是等来的，是你主动做了某件事之后自然出现的」

### 6. Sarasvathy 效果逻辑

先看手里有什么，再决定能做什么。

效果逻辑三问：
1. 我是谁？——特质、品味、价值观（真实的）
2. 我知道什么？——专业知识、生活经验、独特视角
3. 我认识谁？——关系网络中谁可能一起做点什么

交叉组合发现新可能。例：「学计算机 + 对医学感兴趣 + 认识医学院朋友」→ 医疗AI方向。

## 三、分支设计阶段

### 7. Odyssey Plan（三重人生）

三个完全不同的5年人生版本：
- Plan A：当前最可能的路
- Plan B：A走不通的替代路
- Plan C：不考虑钱和面子最想过的人生

### 8. 场景规划法

对每个Plan展开树状分支图：
- 每个关键节点分裂2-3个可能结果
- 每个结果标注筹码变化（+1/-1）
- 关注关键节点结果和预期相反时树状图怎么变
- 标注「交叉点」——不同Plan共享的节点（切换成本最低）

## 四、判断量化阶段

### 9. Tetlock 超级预测法

1. 信息锚定：了解这条路整体难度（学校保研率、专业就业紧缺/饱和），作为判断起点
2. 个人调整：筹码强于同学 → 更乐观，弱 → 更保守。写出调整理由
3. 链式影响：前一节点结果影响后续判断
4. 标注「杠杆节点」——搞定后大幅降低后续所有环节难度

### 10. 贝叶斯更新

先有初始判断，每获得新信息就更新。不展示数学过程，只呈现调整后的结论和理由。

### 11. Kahneman & Tversky 前景理论（去偏差）

内部检查学生可能的认知偏差：
- 过度自信偏差：高估自己的能力
- 确认偏差：只关注支持自己想法的证据
- 可得性偏差：听说某个学长成功就高估某条路
- 禀赋效应：已投入时间不舍得放弃（沉没成本）
- 锚定效应：被第一个信息锚定，后续调整不充分

如果学生对某条路过于乐观或悲观，用日常语言直接指出。

## 五、路径选择和减枝阶段

### 12. Taleb 杠铃策略

- 安全端（70%精力）：确定性最高的路径
- 探索端（20%精力）：有期权价值的探索路径
- 疯狂端（10%精力）：Plan C式高风险高回报

中间的「不上不下」路径——砍掉。

### 13. Myers 实物期权理论

评估每条路的「期权价值」——即使没走通，打开了哪些新可能：
- 能转弯的路 = 高期权价值
- 走了就锁死的路 = 低期权价值
- 优先选保留选择权的路径

### 14. Taleb 反脆弱

「下有保底，上不封顶」。遇到意外冲击不崩溃反而变强。

对每条路做反脆弱测试——冲击后是崩溃、不变、还是反而变强？

### 15. 最优停止理论（37%法则）

先探索37%选项（纯观察），然后选第一个超过之前所有选项的选择。用在实习/offer选择场景。

### 减枝规则（内部执行）

1. 把握很小且没有期权价值 → 直接砍
2. 够不上外部标准线且差距悬殊 → 直接砍
3. 走通后被锁死没有转弯余地 → 降低优先级
4. 走不通也能积累可迁移筹码 → 保留（高期权价值）

### 保留路径排序标准（按优先级）

1. 反脆弱性
2. 期权价值
3. 综合难度判断
4. 上限高度
5. 杠铃适配度

## 六、外部标准线阶段

### 16. Spence 信号理论

在信息不对称市场，需要发「信号」证明能力。学历、实习、竞赛都是信号。

操作：识别「信号缺口」——有能力但缺乏证明的地方。这些缺口 = 下一个要创造的节点。

### 17. McClelland 能力冰山模型

冰山上面（知识技能）容易看到容易学，冰山下面（动机特质自我认知）不容易看到很难改。

操作：分析标准线考核冰山上面还是下面？学生冰山下面是优势还是劣势？

## 七、贯穿全程

### 18. 社会微波辐射（Leo原创概念）

社会整体水平在上升，个人成长斜率必须大于社会演变斜率。

操作：
- 3年前入行门槛 vs 现在（升高/降低？）
- AI对目标岗位的影响（替代/增强？）
- 竞争对手群体在变强还是变弱？
- 校准判断：门槛升高 → 更保守；新机会窗口 → 更乐观
- 标注「时间敏感型节点」——机会窗口正在关闭的
