# wechat-consultant

## 描述
基于同辈互助群数据的自动化咨询agent，提供文风DNA优化、私域运营诊断、社群增长策略等服务。

## 触发词
- 微信咨询 / 群运营咨询 / 社群诊断 / 文风优化 / 私域运营
- 我的群怎么运营 / 如何提高群活跃 / 怎么做社群
- wechat consultant / group consulting / community diagnosis

## 使用方式

### 模式1：快速诊断
用户说"诊断一下我的群"或"我的群运营得怎么样" → 自动执行完整诊断

### 模式2：策略建议
用户说"怎么提高群活跃"或"如何做嘉宾分享" → 给出具体策略

### 模式3：文风优化
用户说"优化我的文风"或"我的发言风格怎么样" → 分析文风DNA并给建议

## 执行协议

### Phase 0：数据加载
1. 读取 `outputs/wechat-groups/group_data.json`（群数据）
2. 读取 `outputs/wechat-groups/voice_dna_analysis.md`（文风DNA）
3. 读取 `outputs/wechat-groups/operation_philosophy.md`（运营心法）

### Phase 1：诊断分析
根据用户需求，执行以下诊断：

#### 1.1 群健康度诊断
- 活跃率：活跃成员 / 总成员
- 中心化指数：群主发言占比
- 内容分布：各话题占比
- 增长趋势：成员增长 vs 活跃增长

#### 1.2 文风DNA诊断
- 句式模式：疑问句/感叹句/省略句占比
- 语气词：常用语气词分析
- 结构标记：列表式/长段落占比
- 回应模式：肯定/否定/补充占比

#### 1.3 运营策略诊断
- 内容策略：AI工具/求职/学习/变现占比
- 资源分享：飞书/GitHub/激活码频率
- 嘉宾运营：分享频率/主题覆盖/效果评估
- 互动设计：提问引导/即时回应/资源倾斜

### Phase 2：策略建议
根据诊断结果，给出具体建议：

#### 2.1 群活跃度提升
- 降低中心化：培养副群主
- 内容沉淀：精华合集
- 新人破冰：入群模板
- 变现测试：付费分享

#### 2.2 文风DNA优化
- 句式多样化：增加疑问句/感叹句
- 语气词平衡：避免过度使用
- 结构优化：列表式 vs 长段落
- 回应模式：肯定/否定/补充平衡

#### 2.3 运营策略优化
- 内容策略：聚焦高价值话题
- 资源分享：提高独家资源占比
- 嘉宾运营：优化邀请流程
- 互动设计：提高互动质量

### Phase 3：输出建议
根据诊断结果，输出：

#### 3.1 诊断报告
- 群健康度评分（1-10）
- 文风DNA评分（1-10）
- 运营策略评分（1-10）
- 关键问题清单

#### 3.2 优化建议
- 短期行动（1-2周）
- 中期规划（1个月）
- 长期目标（3个月）

#### 3.3 可复用模板
- 嘉宾邀请模板
- 分享总结模板
- 新人入群模板

## 输出格式

### 快速诊断（30秒版）
```
📊 群健康度诊断

- 活跃率：32.8%（40/122）
- 中心化指数：41.4%（你发言占比）
- 内容分布：AI工具(14.9%) > 求职(6.4%) > 学习(3.7%)

🎯 关键问题：
1. 中心化过高（41.4% → 目标20-30%）
2. 内容沉淀不足（缺少精华合集）
3. 新人留存待观察（需要破冰机制）

💡 优化建议：
1. 培养2-3个副群主
2. 每周分享后整理精华合集
3. 设计新人入群模板
```
