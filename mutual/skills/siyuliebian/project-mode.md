## 请求处理协议

### 写朋友圈文案

1. 确认这条朋友圈的目的（拉人进群 / 种草 / 悬念 / 案例展示）
2. 从 `项目全景档案.md` 读取产品档案 + IP信息
3. 基于知识库「点线面体」框架 + 「稳赚靠你」心法生成
4. 输出 2-3 个版本，标明类型（悬念型 / 价值型 / 故事型 / 互动型）
5. 铁律：朋友圈只拉人进群，不出价格，不直接成交

### 写私戳话术

1. 确认私戳目的（进群 / 招种子 / 成交）+ 对象类型（陌生人 / 熟人 / 已入群）
2. 基于知识库「顾问式销售七步法」+「上推下切」模型生成
3. 输出：群发初稿 + 回马枪话术 + 常见回复应对
4. 风格：松弛、笃定、霸道，有私聊感，不像群发通知

### 写社群话术

1. 确认话术类型（欢迎语 / 炒群 / 答疑 / 逼单）
2. 从 `项目全景档案.md` 读取产品档案 + 当前阶段
3. 基于知识库「社群销讲SOP」章节生成

### 话术诊断与优化

1. 用户提供聊天记录或话术文本
2. 用知识库诊断以下问题：
   - 有没有「上推」聊死（用陈述句终结对话）？
   - 有没有「看见回应」（接住对方情绪）？
   - 有没有明确的行动指令（CTA）？
   - 有没有「回马枪」（跟进没回复的人）？
3. 输出改写版本 + 逐条说明改了什么、为什么改

### 发售阶段建议

1. 读取 `项目全景档案.md` 的「发售时间线」+「10步进度」
2. 结合知识库给出当前阶段 2-3 个优先执行动作
3. 如有关键前置步骤未完成，提醒一次
