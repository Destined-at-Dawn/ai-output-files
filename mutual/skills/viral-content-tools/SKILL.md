# 模式一：爆款开头

基于叙事模型生成爆款短视频开头文案。

## 模式选择

| 信号词 | 模式 | 方案数 |
|--------|------|--------|
| 默认 / 「快速」 | **快速版** | 3个方案 |
| 「完整版」「全部」「8个」「--full」 | **完整版** | 8个方案 |

### 核心人格

唯一身份是"战友"，不是老师。所有文案必须是第一人称。

### 快速版：三大叙事模型

**模型一：【亲历者顿悟】**
公式：[惨痛过去] + [戏剧性顿悟瞬间] + [惊人新结果]

**模型二：【秘密观察者】**
公式：[巨大研究努力] + [惊人规律/共同点] + [颠覆性价值]

**模型三：【灵魂拷问者】**
公式：[直击灵魂反问] + [也曾痛苦不堪] + [已找到答案]

### 完整版：八大叙事模型

**基础攻击组：**
1. 【亲历者顿悟】
2. 【秘密观察者】
3. 【灵魂拷问者】
4. 【跨界类比大师】— 巧比喻解释复杂问题

**进阶心理组：**
5. 【成果前置展示家】— 直接展示最终结果
6. 【隐形陷阱揭露者】— 警告致命错误（损失厌恶）
7. 【单点爆破手】— 聚焦极小细节（以小博大）
8. 【反常识玩家】— 挑战行业公认方法

### 注意事项

- 所有文案必须第一人称
- 严格遵守禁用词汇和句式黑名单
- 每个方案生成2个开头文案
- 快速版3个方案，完整版8个方案

# 模式二：爆款标题

基于八大故事模型生成多角度爆款标题。

## 使用方式

```
生成标题：[主题]              → 默认陈浪浪犀利风格
生成标题：[主题]，温柔风格     → 温柔扎心风格
生成标题：[主题]，理性风格     → 冷静分析风格
生成标题：[主题]，闺蜜风格     → 毒舌闺蜜风格
```

## 执行流程

### Step 1：八大故事模型生成8个角度

| 模型 | 核心逻辑 | 公式 |
|------|---------|------|
| 1. 极端经历 | 反差制造戏剧性 | 极端数字/状态 + 普通结果 |
| 2. 价值交换 | 强调经验珍贵 | 巨大成本 + 微小智慧 |
| 3. 皇帝新衣 | 说出行业真相 | 挑战主流观点 |
| 4. 身份对立 | 突出转变 | 过去的我 vs 现在的我 |
| 5. 认知颠覆 | 反常识 | 与大众认知完全相反 |
| 6. 方法论祛魅 | 否定复杂，推崇简单 | 否定复杂 + 简单技巧 |
| 7. 错误归因 | 指出真正原因 | 你以为是A，其实是B |
| 8. 笨蛋方法论 | 自嘲反衬智慧 | 我曾经很笨 + 现在明白 |

### Step 2：风格精选改写3个

从8个方案中选最有潜力的3个，用选定风格极致化改写。

## 风格库

### 默认：陈浪浪犀利风格
粗暴、直给。封面极简口语化，标题≤20字符。
高刺激词汇：搞砸、完蛋、骗、毁了、爽、发疯、月入X万

### 温柔扎心风格
不骂但说到心坎里。共情、理解、被看见的感觉。
关键词：没关系、慢慢来、你已经很努力了

### 理性分析风格
纯逻辑碾压。数据、事实、结论、逻辑推导。
关键词：我算了一笔账、数据告诉你、真相是

### 毒舌闺蜜风格
损你但爱你。吐槽、调侃、但有爱。
关键词：就你这样、醒醒吧姐妹、我跟你说实话

## 绝对原则

1. **第一人称叙事**：所有标题围绕"我"
2. **必须有细节**：数字、场景、对话、感官体验
3. **封面标题**：每行≤7个汉字，换行分隔
4. **文字标题**：≤20字符（含标点）
5. **独立完整**：封面和文字标题是两个独立句子
6. **风格一致**：精选版必须极致贯彻选定风格
