# 科学理论词典

用于识别逐字稿中提到的科学理论，包括脑科学、心理学、经济学、管理学等领域的经典理论。

## 心理学

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **荷塘效应** | Lotus Effect | 指数增长的滞后性：荷叶每天翻倍，第30天铺满，第29天才50% | 指数增长、滞后性、最后几天爆发 |
| **羊群效应** | Herd Behavior | 个体盲目跟随群体，缺乏独立判断 | 从众、跟风、失去独立思考 |
| **集体沉默** | Group Silence | 群体决策中，有不同意见的人选择沉默，导致群体极化 | 沉默螺旋、群体极化、虚假共识 |
| **幸存者偏差** | Survivorship Bias | 只看到成功者，忽略失败者，导致错误认知 | 成功者、失败者被忽略、错误归因 |
| **锚定效应** | Anchoring Effect | 第一印象或初始信息对后续判断产生强烈影响 | 首因、初始锚点、难以调整 |
| **确认偏误** | Confirmation Bias | 倾向于寻找支持自己已有观点的信息 | 选择性注意、回声室、强化已有观点 |
| **沉没成本谬误** | Sunk Cost Fallacy | 因为已经投入而继续错误决策 | 已投入、难以放弃、理性决策 |
| **达克效应** | Dunning-Kruger Effect | 能力低的人高估自己，能力高的人低估自己 | 无知者自信、专家谨慎、认知盲区 |
| **损失厌恶** | Loss Aversion | 失去的痛苦大于获得的快乐 | 亏损、恐惧、保守决策 |
| **峰终定律** | Peak-End Rule | 对体验的记忆主要取决于高峰和结束时的感受 | 峰值体验、结尾影响、记忆偏差 |
| **稀缺原理** | Scarcity Principle | 稀缺的东西更有价值 | 有限数量、时间限制、紧迫感 |
| **社会认同** | Social Proof | 看到别人做，自己也会做 | 从众、他人行为、社会压力 |
| **权威偏误** | Authority Bias | 过度信任权威人物的判断 | 权威、盲从、质疑不足 |
| **旁观者效应** | Bystander Effect | 旁观者越多，个体越不会出手帮助 | 责任分散、旁观、不作为 |
| **登门槛效应** | Foot-in-the-Door | 先提出小要求，再提出大要求，更容易被接受 | 渐进式、承诺一致性 |
| **互惠原理** | Reciprocity | 给别人东西，他们会回报 | 交换、回报、人情 |
| **承诺一致性** | Commitment & Consistency | 公开承诺后会保持一致 | 承诺、一致性、自我监控 |
| **心流状态** | Flow | 完全沉浸于活动中的心理状态 | 沉浸、专注、忘我、时间扭曲 |
| **延迟满足** | Delayed Gratification | 放弃即时满足，追求长期更大的回报 | 自控、长期主义、即时满足 |
| **自我效能感** | Self-Efficacy | 对自己完成任务能力的信念 | 能力信念、信心、自我评价 |
| **成长型思维** | Growth Mindset | 能力可以通过努力发展，而非固定 | 可发展、努力、学习导向 |
| **固定型思维** | Fixed Mindset | 能力是固定的，无法改变 | 固定、天赋决定、害怕失败 |
| **元认知** | Metacognition | 对自己认知过程的认识和调控 | 思考思考、自我监控、认知调节 |
| **认知失调** | Cognitive Dissonance | 行为与信念冲突时的心理不适 | 不一致、心理紧张、态度改变 |
| **破窗理论** | Broken Windows Theory | 小问题不解决会导致大问题 | 小问题、蔓延、环境效应 |

## 经济学/金融

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **帕累托法则** | Pareto Principle | 80%的结果来自20%的原因 | 80/20法则、少数关键、不平等分布 |
| **机会成本** | Opportunity Cost | 选择一个选项时放弃的其他选项的价值 | 放弃的价值、选择的代价、资源配置 |
| **边际效用递减** | Diminishing Marginal Utility | 每增加一单位消费，效用递减 | 边际、递减、饱和 |
| **复利效应** | Compound Interest | 利息产生利息的指数增长 | 复利、时间、指数增长 |
| **边际收益** | Marginal Benefit | 每增加一单位投入带来的额外收益 | 边际、增量、收益递减 |
| **沉没成本** | Sunk Cost | 已经发生且无法回收的成本 | 已投入、无法回收、不应影响决策 |
| **边际成本** | Marginal Cost | 每增加一单位产量增加的成本 | 边际、增量、成本变化 |
| **规模经济** | Economies of Scale | 产量增加，单位成本下降 | 大规模、成本降低、效率提升 |
| **范围经济** | Economies of Scope | 同时生产多种产品更经济 | 多产品、资源共享、成本优势 |
| **网络效应** | Network Effect | 用户越多，产品价值越高 | 用户、价值递增、赢家通吃 |
| **长尾理论** | Long Tail | 少量冷门产品合计可超越热门产品 | 冷门、长尾、个性化 |

## 管理学/商业

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **飞轮效应** | Flywheel Effect | 持续努力逐渐加速，最终形成动力循环 | 飞轮、持续、加速、循环 |
| **第一性原理** | First Principles | 从最基本的事实出发，而非类比推理 | 基本事实、去类比、本质思考 |
| **逆向思维** | Inversion | 从反面思考问题 | 反向、倒推、避免错误 |
| **最小可行产品** | MVP | 用最小成本验证市场 | 最小、验证、快速迭代 |
| **精益创业** | Lean Startup | 快速验证、快速迭代、降低浪费 | 快速、迭代、精益 |
| **蓝海战略** | Blue Ocean Strategy | 开创无人竞争的新市场 | 新市场、无竞争、价值创新 |
| **产品市场契合** | PMF | 产品与市场需求的匹配程度 | 匹配、市场需求、产品定位 |
| **用户增长黑客** | Growth Hacking | 快速增长的策略和技巧 | 增长、黑客、快速扩大 |
| **平台经济** | Platform Economy | 连接供需双方的双边市场 | 平台、网络效应、生态 |

## 脑科学/认知

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **系统1 vs 系统2** | System 1 vs System 2 | 系统1快速直觉，系统2缓慢理性 | 快速思考、慢速思考、双系统 |
| **认知负荷** | Cognitive Load | 工作记忆容量有限，过度负荷影响学习 | 工作记忆、容量、信息过载 |
| **多巴胺奖励** | Dopamine Reward | 大脑通过多巴胺奖励学习行为 | 奖励、强化、成瘾 |
| **镜像神经元** | Mirror Neuron | 看到他人行为时大脑模拟该行为 | 模仿、共情、学习 |
| **记忆宫殿** | Memory Palace | 通过空间联想增强记忆 | 空间、联想、记忆技巧 |
| **睡眠巩固** | Memory Consolidation | 睡眠时大脑巩固记忆 | 睡眠、记忆、强化 |
| **注意力分散** | Attention Fragmentation | 多任务处理降低注意力质量 | 多任务、分散、注意力下降 |

## 社会学/群体行为

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **邓巴数** | Dunbar's Number | 人类能维持的稳定社交关系约为150人 | 社交关系、150人、认知限制 |
| **弱连接理论** | Weak Ties | 弱连接比强连接更能带来新信息 | 弱关系、新信息、强连接同质化 |
| **社会网络理论** | Social Network Theory | 人际关系网络影响行为和结果 | 网络、结构、影响力 |
| **群体极化** | Group Polarization | 群体讨论后观点会变得更加极端 | 极端、强化、同质化 |
| **群体迷思** | Groupthink | 群体为了和谐而忽视异议 | 和谐、忽视异议、错误决策 |

## 策略/博弈

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **囚徒困境** | Prisoner's Dilemma | 个人理性导致集体非理性 | 困境、合作、背叛 |
| **纳什均衡** | Nash Equilibrium | 没有人有动力改变策略的状态 | 均衡、稳定、策略 |
| **零和博弈** | Zero-Sum Game | 一方收益等于另一方损失 | 输赢、竞争、零和 |
| **正和博弈** | Positive-Sum Game | 双方都能获益 | 合作、双赢、增值 |
| **先发优势** | First Mover Advantage | 先进入市场的优势 | 先发、优势、占据心智 |

## 其他经典理论

| 理论名称 | 英文 | 核心内容 | 关键词/特征 |
|---------|------|---------|-------------|
| **奥卡姆剃刀** | Occam's Razor | 简单的解释往往更好 | 简单、经济、剃刀 |
| **彼得原理** | Peter Principle | 人会被提升到他不胜任的位置 | 晋升、不胜任、组织 |
| **墨菲定律** | Murphy's Law | 凡事可能出错，就一定会出错 | 出错、准备、悲观 |
| **帕金森定律** | Parkinson's Law | 工作会自动膨胀，占满所有可用时间 | 时间膨胀、效率、官僚 |
| **蝴蝶效应** | Butterfly Effect | 初始条件的微小变化导致巨大差异 | 初始条件、敏感、混沌 |
| **黑天鹅事件** | Black Swan | 极端罕见但有重大影响的事件 | 罕见、重大影响、不可预测 |
| **灰犀牛事件** | Grey Rhino | 明显存在但被忽视的威胁 | 明显、忽视、高风险 |
| **二八定律** | 80/20 Rule | 80%的结果来自20%的原因 | 不平等分布、关键少数 |
| **最小阻力路径** | Path of Least Resistance | 人们倾向于选择阻力最小的路径 | 阻力、习惯、惰性 |
