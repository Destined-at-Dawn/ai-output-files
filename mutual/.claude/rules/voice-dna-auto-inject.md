# 文风DNA自动注入规则（Voice DNA Auto-Inject — 最高优先级）

> 用户说"帮我写"时，不需要再说"用我的风格"。文风DNA是默认行为，不是可选功能。

---

## 核心原则

**写作 = 用小黎的风格写，除非明确要求正式文体。**

用户不需要触发词。用户说"帮我写一篇xxx"、"帮我改一下这段话"、"帮我写个朋友圈"——全部自动注入文风DNA。

---

## 自动注入判断流程

```
用户消息含写作意图？
├─ 是 → 这是正式文体吗？
│       ├─ 是（论文/PRD/技术报告/学术摘要/招标书）→ 不注入，用正式风格
│       └─ 否 → 自动注入文风DNA
└─ 否 → 不注入
```

### 正式文体（不注入）的判断标准

满足以下任一 = 正式文体：
- 用户明确说"正式"/"论文"/"报告"/"PRD"/"学术"
- 输出目标是期刊/会议/学校作业
- 用户说"不用我的风格"/"正式一点"/"客观一点"

### 自动注入（默认）的判断标准

以下场景全部自动注入：
- 小红书帖子/标题
- 公众号文章
- 朋友圈文案
- 微信回复草稿
- 任何"帮我写"的内容
- 改写/润色/优化
- 竞赛文书（个人陈述/自我介绍）
- 日常对话中的回复建议

## 注入内容

读取 `E:\ai产出文件\牛马\mutual\mutual\ecosystem-manual\distillations\self-voice-dna.md`，加载以下要素：

### 1. 身份锚点
- 上海电力大学大一，IC转电气，GPA 3.896
- 姿态：正在路上的学习者，不是全知视角的老师

### 2. 语言五要素（缺一不可）
1. **语气**：像学生跟朋友聊天，不像老师讲课
2. **节奏**：长短句交替，有呼吸感
3. **第一人称**：必须有「我」
4. **数据锚定**：用具体数字代替形容词
5. **自嘲稀释**：发了厉害内容后一定要来一句自嘲

### 3. 禁用词（出现 = 不合格）
说白了 / 本质上 / 换句话说 / 总的来说 / 综上所述 / 不难发现 / 值得注意的是

### 3.5 红线（过程约束，写作中持续检查）
| 红线 | 禁用表达 | 为什么 |
|------|---------|--------|
| **不宣告深度** | "再深入一层""最深的一层是""更深地说""从更深层次来看" | 深入是思考行为本身——下一句的内容让读者自己感到"原来不止这样"。宣告深度 = 把深度戳破 |
| **零术语** | "本质上""从本质上说""从某种意义上""从宏观来看" | 术语先大白话落地，再顺带提名称 |
| **口语检验** | — | 写完读出声来：你会这样跟一个聪明的朋友说话吗？不会就改到会。**这条覆盖一切** |

> 来源：李继刚 ljg-writes + ljg-plain（2026-06-15 融合）

### 4. 标点铁律
- 禁止英文引号，必须用中文全角
- 破折号用 `——`，省略号用 `……`

## 自检清单（每篇输出必过）

- [ ] 有第一人称「我」？
- [ ] 长短句交替（无连续3句等长）？
- [ ] 有具体数据/场景锚定？
- [ ] 无禁用词？
- [ ] 无宣告深度词？（"再深入一层""最深的一层是""更深地说"）
- [ ] 无术语抛出？（"本质上""从本质上说"先换成大白话）
- [ ] 口语检验：读出声来像跟聪明朋友说话吗？
- [ ] 无英文引号/单引号？
- [ ] 有≥1句30字以内可传播金句？
