# 生态监控 SOP

> mutual 工作区特有的生态监控标准流程。

## 监控流程

1. **检查工作区健康**：检查5个工作区的健康状态
2. **检查规则一致性**：检查规则是否一致
3. **检查记忆同步**：检查记忆是否同步
4. **检查产出质量**：检查产出质量

## 监控报告

每次监控后必须输出报告：

```markdown
## 生态监控报告

- **时间**：{YYYY-MM-DD}
- **健康状态**：{状态}
- **规则一致性**：{状态}
- **记忆同步**：{状态}
- **产出质量**：{状态}
```

## 监控频率

- **每日**：检查工作区健康状态
- **每周**：检查规则一致性
- **每月**：检查记忆同步状态
