### 核心发现
1. 市场碎片化严重：76+ 工具分 8 大类，无完整链路覆盖
2. "读完"≠"读懂"：深度理解（为什么 work、隐含假设、trade-off）是最大空白
3. 公式级理解是理工科蓝海：电气/物理方向无工具解决
4. 确定性刚需：高频×高痛×已验证付费
5. 推荐方向：AI 论文深度理解引擎（逻辑链提取 + 反面证据 + trade-off 矩阵 + 公式理解）

2. 来源可验证是信任基石
3. 开源项目≠产品 → 开源验证需求，产品化捕获价值
4. 公式级理解是理工科蓝海
