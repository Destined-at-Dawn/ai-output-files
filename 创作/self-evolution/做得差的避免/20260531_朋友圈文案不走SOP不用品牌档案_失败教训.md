# 失败教训：朋友圈文案不走SOP不用品牌档案

## 日期
2026-05-31

## 场景
用户要求写朋友圈引流文案（社群推广、嘉宾招募、个人故事版）。AI直接基于群公告风格和营销号模板写了5条文案（v2），没有：
1. 读 `微信朋友圈/CLAUDE-朋友圈.md`（朋友圈影响力系统的5个Workflow）
2. 读 `微信朋友圈/自己/品牌档案.md`（品牌定位、人设、风格DNA、禁忌清单）
3. 读 `微信朋友圈/自己/朋友圈skill.yaml`（完整的朋友圈影响力工程Skill，含模板A/B/C/D+自审流程+认知科学支撑）
4. 走Workflow A的5个Phase（需求对齐→品牌校准→内容生产→自审→存档）
5. 用品牌档案中的禁忌清单自审

## 用户批评原话
- "你的朋友圈文案还是没有运用我自己的文风DNA"
- "我之前让你写的那些朋友圈，你为什么一开始没有激发？"
- "你应该每次都是先查找我们有没有过去的经验可不可以复用"

## 问题
1. **不走路由**：CLAUDE.md路由表明确写了「朋友圈/发圈/影响力」→读`微信朋友圈/CLAUDE-朋友圈.md`，AI完全跳过
2. **不读SOP**：已有的朋友圈影响力系统有完整的Workflow A（5个Phase）+ 4种文案模板 + 品牌校准 + 自审checklist，AI一个都没用
3. **不读品牌档案**：`品牌档案.md`有明确的风格DNA编码（第一人称/语气缓冲/不端不装不说教）、禁忌清单（禁"在当今社会"/禁"首先其次最后"/禁排比句堆砌）、人设光谱（60%价值+30%日常+10%互动），AI写出来的文案全部踩了禁忌
4. **不读已产出**：`微信朋友圈/产出/`里有之前成功的朋友圈案例，AI没有参考
5. **用营销号模板代替小黎风格**：AI用了"你是否知道""区别在哪""不是你不够努力"的营销号框架，而小黎真实风格是kaomoji穿插+#话题标签+真实经历+自嘲+口语化

## 根因
跟教训05（YouTube分析套错模板）、10（图片提示词未读库未调技能）、15（图片提示词质量太差）、18（公众号读取敷衍）、24（有约束文件不用英文标点）**完全同一类错误**——
**有约束文件不用，凭自己的"感觉"直接写。**

同类教训已出现6次。这不是"下次注意"就能解决的，必须升级为硬规则。

## 正确做法
1. **先走路由**：收到任何"写朋友圈"需求 → 先读 `微信朋友圈/CLAUDE-朋友圈.md`
2. **读品牌档案**：写之前读 `品牌档案.md`，校准人设和风格DNA
3. **走Workflow A**：需求对齐（目的/受众/入口三问）→品牌校准（对照禁忌清单）→内容生产（选模板A/B/C/D）→发布前自审（7项checklist）→存档到 `产出/`
4. **读已产出**：先看之前成功的朋友圈风格，不能凭空编
5. **自审去AI味**：对照禁忌清单逐项检查，特别是开头方式和结尾方式

## 量化影响
- v2文案用户不满意，需要全部重写为v3（浪费2轮对话）
- 用户信任度下降："你让我很失望"
- 同类教训第6次触发，应升级为铁律

## 关联工程法则
- 法则8（门禁文化）：没过品牌校准就产出 = 跳过门禁
- 法则10（规则结晶）：同类教训6次 = 必须升级

- 010-思考快与慢：图式匹配——读者大脑自动匹配"这是谁发的"，不一致=信任崩塌
