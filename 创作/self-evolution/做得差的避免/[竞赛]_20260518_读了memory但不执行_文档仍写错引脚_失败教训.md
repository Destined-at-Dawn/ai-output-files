## 错误内容

实验文档中 5 处错误描写：
- 硬件连接表：RST/BL 列为"输出"引脚
- 原理描述："背光由 FPGA 的 lcd_bl 引脚控制"
- ST7796S 特性表："背光控制 | 独立 BL 引脚"
- 引脚分配表：lcd_rst/lcd_bl 占用 IO
- 常见问题："用万用表测量 BL 引脚电压"

## 根因分析

| 层 | 分析 |
|----|------|
| 直接原因 | 生成文档时没有读取 memory 文件 |
| 深层原因 | SOP-11 实验文档编写流程中缺少"对照 memory 检查"这一步 |
| 系统原因 | memory 记录和 SOP 执行之间没有强制捆绑机制 |

## 正确做法

- RST：硬件已做固定连接，由系统复位统一控制
- BL：硬件默认常亮，FPGA 内部仅软件置高
- 学生只需连接 VCC/GND/SCLK/MOSI/CS 五根线
