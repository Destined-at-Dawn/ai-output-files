# 群公告 v1→v5 迭代全记录（做得好的）

> 日期：2026-06-01

> 可复用场景：任何社群公告的写作和迭代

---

## 做了什么

5 轮迭代，从「菜单式公告」到「行动指南式公告」，同时暗转战略方向。

## 每版核心变化

| 版本 | 核心改动 | 学到什么 |
|------|---------|---------|
| v1 | 菜单（「我有这些」） | 信息罗列 ≠ 价值传递 |
| v2 | 从向阳集模板改造 | 模板只是起点，要注入自己的 DNA |
| v3 | 注入飞书文档语气 | **「不画饼，只做事」+「#怎么参与」4 步行动指南** |
| v4 | 飞书链接前置 + 文风注入 | 知识库要放最前面，新人进群第一眼看到 |
| v5 | 暗转战略 | 嘉宾本周特辑，下周起重心回自己。群规①改为「问之前先自己试」 |

## 可复用模式

### 公告结构模板
```
1. 欢迎语 + 一句话定位
2. 知识库链接（必看）
3. 信任声明（不推销/不割韭菜/不打卡）
4. 这个群在做什么（3 句话）
5. 本周分享日程（嘉宾+方向+日期）
6. 开源项目/福利
7. 项目群引流
8. 怎么参与（4 步行动指南）
9. 群规（3 条）
10. 收尾金句
```

### 关键教训
- 群公告是**行动指南**，不是菜单
- 字数控制在 **2000 字以内**（WeChat 限制）
- 先打消顾虑（信任声明），再展示价值
- 收尾干脆，不强推

## 适用场景
- 新建社群的公告
- 社群公告的版本迭代
- 任何需要「从菜单位转向行动指南」的文档
