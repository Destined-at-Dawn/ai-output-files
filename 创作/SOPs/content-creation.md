# 内容创作 SOP

> 创作工作区特有的内容创作标准流程。

## 创作流程

1. **选题**：选择有价值的主题
2. **调研**：调研相关资料
3. **创作**：创作内容
4. **审核**：审核内容质量
5. **发布**：发布内容

## 创作记录

每次创作必须记录：

```markdown
## 内容创作记录

- **时间**：{YYYY-MM-DD}
- **主题**：{主题}
- **平台**：{平台}
- **状态**：{状态}
```

## 创作频率

- **小红书**：每周2-3篇
- **公众号**：每周1篇
- **PPT**：按需制作
