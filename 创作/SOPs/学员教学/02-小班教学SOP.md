## 二、1v1 私教执行步骤

### Step 1: 开营画像（首次30分钟）

- **操作**：完成学员画像采集（总览 § 二.1 的 3W1H 模板）
- **额外采集**（1v1专属）：
  - 学员的学习历史：哪些学过/哪些没学过/哪些学了但没学会
  - 学员的认知风格：喜欢听讲解还是自己看？喜欢具体例子还是抽象原理？
  - 学员的情绪状态：焦虑程度、自信程度、对这门课的态度
- **输出**：学员画像卡 + 初步教学计划

### Step 2: 定制化教学计划

- **操作**：基于画像制定个性化方案

```
教学计划 = 目标拆解 × 时间分配 × 方法匹配 × 检验节点

目标拆解：大目标 → 阶段目标 → 每节课目标
时间分配：总课时 → 各模块占比（重点多给，简单少给）
方法匹配：视觉型→图解/思维导图；听觉型→讲解/讨论；动手型→练习/实操
检验节点：每节课结束前10分钟 = 检验时间
```

- **核心原则**：布鲁姆定律——超过98%同龄人只有两条路：①定制化辅导（私教），②掌握学习方法（每个知识点学透再进下一个）。1v1 = 第①条路的极致形态

### Step 3: 单节课执行流程

```
┌─────────────────────────────────────────────┐
│ 单节课 60-90 分钟                              │
│                                               │
│  ① 回顾（5min）：上节课的核心是什么？口述一遍      │
│  ② 引入（10min）：今天学什么？为什么重要？         │
│  ③ 讲解（20min）：核心概念 + 类比 + 例子          │
│  ④ 练习（15min）：学员自己做，教师观察            │
│  ⑤ 检验（10min）：出3道题，限时做                │
│  ⑥ 总结（5min）：今天最重要的1个收获是什么？       │
│  ⑦ 作业（5min）：布置课后任务（量力而行）          │
└─────────────────────────────────────────────┘
```

- **铁律**：①回顾环节不可省——激活上节课的记忆，否则等于白学
- **铁律**：⑤检验环节必须做——口头问"懂了吗"不算检验，要实际做题

### Step 4: 课后跟进

- **操作**：
  - 24小时内：学员口述本节课核心内容（微信语音/文字均可）
  - 48小时内：完成课后练习并提交
  - 下节课开始前：教师快速检查作业，标记共性问题
- **认知科学依据**：间隔重复——24小时内的回顾将记忆保持率从20%提升到80%

## 三、1v15 小班执行步骤

### Step 1: 开营画像（集体+个体）

- **操作**：
  - **集体画像**：全班平均水平、共同目标、可用时间
  - **个体画像**：每位学员的3W1H卡片（简版，5分钟/人）
  - **分层**：按水平分为A（领先）/B（跟上）/C（需要帮助）三层
- **输出**：班级画像 + 分层名单 + 整体教学计划

### Step 2: 分层教学设计

- **操作**：同一节课，三个层次不同深度

| 层次 | 目标 | 教学内容 | 作业难度 |
|------|------|----------|----------|
| **A层**（领先） | 能教别人 | 深度拓展 + 挑战题 | 进阶题 + 思考题 |
| **B层**（跟上） | 稳固掌握 | 标准教学 + 适量练习 | 标准题 + 变式题 |
| **C层**（需要帮助） | 不掉队 | 基础巩固 + 个别辅导 | 基础题 + 重复练习 |

- **认知科学依据**：心流理论（012-心流）——挑战略高于技能时学习效率最高。A层学员需要更高的挑战，C层学员需要更低的门槛，都在各自的心流区间

### Step 3: 课堂互动设计

- **操作**：小班教学的核心优势是互动，必须设计互动环节

```
互动金字塔（每节课至少3种）：

  👆 小组讨论（2-3人一组，讨论1个问题）
  👆 课堂投票/抢答（全员参与，即时反馈）
  👆 学员讲给学员听（A层教C层 = A层深度加工，C层获得低门槛讲解）
  👆 教师提问（苏格拉底式追问，暴露思维过程）
```

- **铁律**：禁止满堂灌——小班教学如果只是教师讲学生听，跟录播课没区别
- **认知科学依据**：生成效应（009-认知天性）——自己组织语言讲给别人听，比被动听讲的记忆保持率高3倍

### Step 4: 进度追踪与反馈

- **操作**：

| 追踪维度 | 频率 | 方式 |
|----------|------|------|
| 出勤率 | 每次 | 签到表 |
| 课堂参与度 | 每次 | 观察记录 |
| 作业完成率 | 每次 | 作业提交记录 |
| 测验成绩 | 每周 | 小测验 |
| 层次调整 | 每2周 | 根据表现升降层 |

- **反馈模板**（给学员的周反馈）：

```markdown
## 本周学习反馈

### 做得好的
- [具体指出1-2个亮点]

### 需要加强的
- [具体指出1-2个问题 + 改进方法]

### 下周目标
- [1个可量化的小目标]

### AI辅助建议
- [推荐用AI做什么练习/复习]
```

## 四、决策表

| 教学场景 | 核心策略 | 关键方法 | 交付物 |
|----------|----------|----------|--------|
| 1v1基础辅导 | 查漏补缺 | 四步学习闭环 | 个性化计划+错题本 |
| 1v1深度提升 | 刻意练习 | 挑战题+即时反馈 | 进阶练习+能力雷达图 |
| 1v15标准班 | 分层教学 | 互动金字塔+分层作业 | 教学计划+周反馈 |
| 1v15冲刺班 | 考试驱动 | 真题+模拟+错题 | 模拟测试+薄弱点报告 |
| 线上1v1 | 异步+同步结合 | 录播讲解+实时答疑 | 学习记录+答疑汇总 |

## 七、认知科学支撑

| 认知机制 | 来源 | 在小班教学中的应用 |
|---------|------|-------------------|
| **布鲁姆定律** | 锦成AI课 §道 | 超过98%同龄人只有两条路：①定制化辅导 ②掌握学习方法。1v1 = ①的极致形态 |
| **心流理论** | 012-心流 §概念1 | 分层教学的底层逻辑——每个层次的学员都在各自的心流区间（挑战略高于技能） |
| **生成效应** | 009-认知天性 §概念1 | 学员讲给学员听 = 深度加工。A层教C层不是浪费A层时间，而是A层最好的学习方式 |
| **社会建构主义** | 028-我们如何学习 §概念1 | 小组讨论 = 社会建构——知识不是教师灌进去的，是学员在互动中自己建构的 |
| **最近发展区** | 028-我们如何学习 §概念1 | 教学内容应在学员"跳一跳能够到"的区间——太简单=无聊，太难=放弃 |
| **元认知发展** | 009-认知天性 §概念2 | 让学员口述"今天学了什么" = 元认知训练——学会监控自己的学习过程 |
| **即时反馈** | 017-刻意练习 §概念1 | 刻意练习四要素之一。课堂检验→即时反馈→修正理解。延迟反馈的效果衰减50% |
| **认知学徒制** | 028-我们如何学习 §概念1 | 1v1教学的本质是"认知学徒制"——教师示范→学员模仿→逐步放手→学员独立 |
